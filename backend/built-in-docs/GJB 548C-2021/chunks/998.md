# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
### A.3 生产控制

---

为了使金属化系统能达到稳定的、良好的质量和可靠性，并确保产品是根据设计中规定的要求进行生产的，承制方应对生产进行控制。承制方应确定哪些参数对于金属化层完整性是关键的，并根据GJB3014的规定控制这些参数。承制方应能证明至少对金属化层厚度、台阶覆盖和横截面积、金属条线宽、接触区域和通孔尺寸、接触区域和通孔电阻、方块电阻等进行了控制，并能显示它们已把金属化层控制到与设计规定的极限值相一致。对于这些参数，应建立规范极限值。此外，危及金属化层完整性的缺陷应根据方法5004附录A中的替代目标程序(替代程序2)相一致的方法进行控制。
