# 中华人民共和国国家军用标准
## 方法2011.2 键合强度(破坏性键合拉力)
### 3.2.5 试验条件G-推开试验(梁式引线)

---

本试验一般用来进行工艺控制，并采用键合到一特别准备的基板上的半导体芯片样品进行试验，因此不能用于产品或检验批的随机抽样试验。试验时应采用带有一个开孔的金属化基板，这个接近中心位置的小孔应足够大，从而为推压工具提供间隙，但是又不能大到影响键合区。推压工具应足够大，以使在试验期间的器件断裂减到最小，但又不能大到与固定键合区的梁式引线相碰。应按下述方式进行推开试验：牢固固定基板并通过开孔插入推开工具，以小于 \(0.254\mathrm{mm / min}\) 的速率实现推开工具与硅器件的接触(这样不会产生明显冲击),并以恒定速率对键合器件下侧加力。当出现失效时，记录失效时力的大小和分离模式。
