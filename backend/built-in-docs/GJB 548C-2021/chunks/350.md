# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.11.3 补充室温退火试验程序

---

如果器件在标准条件规定的辐射试验中失效，则可能需要按如下步骤附加室温退火试验:  
a) 在完成标准条件的辐射和测试后，使试验样品在最坏情况静态偏置条件下进行室温退火试验。关于最坏情况偏置的要求见4.9。  
b) 试验进行的方式应使试验样品始终处于 \(24^{\circ}\mathrm{C} \pm 6^{\circ}\mathrm{C}\) 的温度范围内。  
c) 如果可能，室温退火试验时间应尽量长，以使超出辐射前器件规范值的参数返回至规范值或承制方给出的辐射后参数极限值(PIPL)以内。注意，室温退火试验的时间不应超过 \(t_{\mathrm{max}}\) ，按公式(1)计算:式中:  
\(D_{\mathrm{spec}}\) ——器件规范规定的总剂量值；  
\(R_{\mathrm{max}}\) ——预定应用时的最大辐射剂量率。  
d) 按4.8的规定测试器件的电性能。如果试验样品在进行补充室温退火后通过电测试，尽管器件在标准条件辐射试验过程中发生电参数失效，但对极低剂量率环境应认为可接受该批次的器件。
