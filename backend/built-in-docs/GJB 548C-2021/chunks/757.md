# 中华人民共和国国家军用标准
## 方法2010.2 内部目检(单片电路)
### 4.2.2.6 金属化层的跨接

---

呈现下列情况的器件不得接收：a）条件A(S级)：金属化层的跨接使金属条之间的距离减小于原始设计间距的 \(50\%\) 。条件B(B级)：金属化层的跨接使金属条之间看不到明显的间隔线。  
注：仅对条件B(B级)，GaAs微波器件中横跨间距小于1um的金属化层跨接只需根据4.2和表1的要求，在400倍～1000倍下进行检查。如果不能着明显间隔线，只需在晶圆级对器件进行功能测试。  
b）条件A(S级)：由于键合使原来在键合区的金属发生移位(挤出金属)超过 \(25\mu \mathrm{m}\) ，或者使未经玻璃钝化的工作金属化层或划片槽与键合区之间的间距减少到小于原设计间距的 \(50\%\) 或 \(6\mu \mathrm{m}\) (取较小值)。  
由于键合使原来在键合区的金属发生移位(挤出金属)，使未经玻璃钝化的工作金属化层或划片槽与键合区之间的间距减小，不能着明显间隔线。  
注：对SOS器件，不包括绝缘划片槽。
