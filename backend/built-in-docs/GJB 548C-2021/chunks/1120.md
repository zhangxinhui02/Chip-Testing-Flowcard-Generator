# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2030.1 芯片粘接的超声检测
### 3.1 超声检测设备

---

该设备的试验频率应能足以穿透芯片粘接界面。不推荐将密封的器件，或者非密封但有空腔的器件打开，超声设备应有能力检测进入封装的顶部、底部或背面并反射或穿透所期望材料界面的声波信号。试验频率和对焦距离应适合于检测直径小到 \(25.4\mu \mathrm{m}\) 的空洞，检测时穿透芯片是所期望的，但由于器件的结构有可能不易实现。在这种情况下，试验频率和对焦距离应选择确保穿透期望的材料界面，所能达到的分辨率作为第二位的考虑因素。
