# 中华人民共和国国家军用标准
## 方法1015.1 老炼
### 3.3.1 一般要求

---

试验条件A～试验条件E的老炼试验环境温度应至少为 \(125^{\circ}C\) (混合集成电路见表1)。承制方可以提高试验条件A～试验条件E的试验温度，并按表1减少相应的试验时间。由于正常情况下壳温和结温将明显地高于环境温度，所以采用的电路应设计成使试验和工作时的最高额定结温不超过规定值(B级不超过 \(200^{\circ}C\) ，对S级不超过 \(175^{\circ}C\) )（见3.3.2)。具有内部过热保护线路的器件应符合方法1005中3.3.4的规定。规定的试验温度是在试验箱中工作区域内所有器件受到的最低环境温度。要做到这一点，可对试验箱的内部结构、负荷、控制或监测仪器的放置位置以及空气或其他合适气体或液体介质的流动进行必要的调整来保证。在校准时，应使试验箱处于满负载(试验电路板上不必放入器件)但不加功率的状态，调节指示器的传感器探头位置，使其位于试验箱内工作区域的最低温度处。
