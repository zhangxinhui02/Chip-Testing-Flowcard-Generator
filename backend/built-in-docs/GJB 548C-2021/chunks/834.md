# 中华人民共和国国家军用标准
## 方法2012.2 X射线检查
### 3.1.3.1 对扁平封装、双列直插封装以及单端圆柱形器件的观察

---

3.1.3.1 对扁平封装、双列直插封装以及单端圆柱形器件的观察除另有规定外， 对扁平封装、双列直插封装以及单端圆柱形器件， 在本标准第 4 章的图 1\\~图 6 所规定的 Y 方向上， 用 X 射线透视观察一次。在需要一个以上的视图时(适用时), 用 X 射线分别在 Z 方向和 X 方向进行第二次、第三次(适用时)透视观察。对于扁平封装， Z 方向或者 X 方向均可。实时成像 X 射线技术不像胶片技术那样受限制， 因此应对每只器件 X、Y 和 Z 方向进行观察。也应包括承制方推荐的检查方向。
