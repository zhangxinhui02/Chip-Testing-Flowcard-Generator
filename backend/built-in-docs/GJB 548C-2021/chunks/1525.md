# 附录A（方法5010）（规范性附录）计算机辅助设计(CAD)的认证要求
### A.1.5 单元设计和仿真的鉴定

---

应按下述程序完成单元设计和仿真的鉴定。  
a) 参数验证/仿真验证；  
b) 功能验证。  
应设计一个芯片或芯片组(称之为单元测试芯片组),能测试一组单元的特性。单元测试芯片组的设计在使用前应提交鉴定机构审查并应得到批准。单元测试芯片至少应包括下述单元:a) 反相器；  
b) 输入与非门；  
c) 2-3输入端与或非门(输入端处为2输入与门，输出端处为3输入或非门);  
d) 带有低电平复位的D锁存器；  
e) 带有低电平复位的JK触发器；  
f) TTL输入缓冲器；  
g) CMOS输入缓冲器；  
h) 输出缓冲器；  
i) 带有上拉的三态I/O缓冲器。  
其目的是包括各类单元的典型代表(即组合电路、时序电路、输入、输出)。应形成测试链(需要避免上升及下降时间的测量问题时，应使用测试链),并且应得到整个工作范围内的实际特性数据(应包括有提取多路转换器及I/O缓冲器延迟的方法)。应确定上述各单元的延迟与金属走线长度以及扇出之间的关系。应将实际性能数据连同计算机程序仿真结果一起提交给鉴定机构。实际性能数据应在仿真结果预计的极限值范围之内。如采用乘法器来外推极限温度下的性能，该乘法器也应加以验证。  
此外，在测试芯片上应该为上述各单元提供一组引出端，以便测试验证这些单元的功能特性。  
注：输入和输出可以多路复用。
