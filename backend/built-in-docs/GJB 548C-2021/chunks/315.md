# 中华人民共和国国家军用标准
## 方法1018.2 内部气体成分分析
### 3.7 执行规定

---

由鉴定机构对检测实验室能够实施本方法的能力(特定极限值及体积)进行认定。检测实验室的能力确认原则被设计为：在确定一个样品是否通过规定的极限值时，一个实验室与另一个实验室间的差别不超过 \(\pm 20\%\) 。高于或低于适用范围的水汽含量不得作为可依据的数据。超出范围的数据只表示一个相对的概念，仅在实验室内部的结果比较时才有用。  
除另有规定外， \(0.5\%\) (5000ppm)的规范极限值可用于所有腔体体积。承制方可允许使用下面的修正因子，前提是这些修正因子的适用性应得到证明，并形成文件。

\(T_{\mathrm{s}}\) ——密封温度， \(^\circ \mathrm{C}\) 。
