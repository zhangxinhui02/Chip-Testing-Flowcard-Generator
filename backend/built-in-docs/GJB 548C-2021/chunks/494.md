# 中华人民共和国国家军用标准
## 方法1031 薄膜腐蚀
### 3.2.2 短期试验

---

对于短期质量保证试验，应按图1中给出的时间- 温度偏置顺序作16次循环，每次3h，共48h。短期试验采用的顺序如下：  
a) AB段 \(125^{\circ}C\) ，使管壳内壁上的表面水分汽化。b) BC段使环境水汽在管壳内壁和管芯上冻结(降温速率为 \(100\mathrm{K / min}\) )。c) CD段使封壳内部的表面温度达到平衡(也就是使水汽重新分布)。d) DE段将水汽从器件壁转移至管芯(冷表面泵)。从E点开始，在管芯还是湿的时候，施加偏置。e) EF段调整升温速率，以便通过减少热耗散和调整升温速率(最大为 \(100\mathrm{K / min}\) )，使管芯总是处于最低的温度。f) FG段保持环境温度为 \(2^{\circ}C\) ，在依然施加偏置的同时使管芯附近相对湿度最高。g) GH段迅速升温(最大升温速率为 \(100\mathrm{K / min}\) )阶段，使水汽从壁上蒸发，可能同时伴有一种剧烈的驱动力，增强污染物从壁到管芯间的传递。在施加偏置且温度为 \(25^{\circ}C\) 时结束循环。将器件从夹具中取出。
