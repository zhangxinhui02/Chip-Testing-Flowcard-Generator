# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.13.1 概述

---

很多双极线性器件都表现出ELDRS现象， 该现象不能像MOS器件那样采用 \(0.5\mathrm{Gy(Si)} / \mathrm{s}\sim 3\mathrm{Gy(Si)} / \mathrm{s}\) 的室温辐射配合加速度退火试验来模拟。对表现出ELDRS现象器件的辐射试验方法为： 使用预定应用环境的辐射剂量率试验、按规定的低剂量率过辐射水平试验， 或加速试验(比如高温辐射试验， 需要规定参数 \(\Delta\) 设计裕量(见4.13.3, ELDRS器件表征试验))。如果不能确定器件是否会表现出ELDRS现象， 应采用判定试验来确定(见4.13.2)。  
对以下情况可不进行4.13的ELDRS试验:a) 已知器件设计不包含双极晶体管；b) 已知器件设计不包含任何线性电路功能；c) 业已证明，该器件类型或 IC 工艺，并未表现出器件参数 ELDRS，而且也已证明影响 ELDRS 响应的变量对于特定的生产方工艺过程已处于受控状态。
