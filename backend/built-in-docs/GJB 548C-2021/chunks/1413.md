# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.10 应对意外失效的分析程序

---

A.10.1 承制方应建立现场失效分析程序，用以确定失效的根本原因。并采取措施用于：失效的确认、对失效进行控制、处理、通报和纠正措施。  
A.10.2 承制方应建立一个程序，用来发现晶圆生产期间未被发现，但在以后工序(如分选、组装、测试等)发现的致命或关键性缺陷，并且采取相应的纠正措施。  
A.10.3 承制方应通过材料审查委员会或经认可的其他管理机构，对意外失效进行评审，召集专家鉴别和确定不合格产品(致命或关键缺陷)，必要时可通报给内外用户，并采取相应的纠正措施。此外，应明确什么情况下需要召集材料审查委员会。
