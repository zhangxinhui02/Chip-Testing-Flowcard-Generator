# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.18 工作寿命

---

制备10个具有电气功能的气密封装器件样品，采用的工艺步骤应与用户装配图规定的正常合格产品相同。如果用户和鉴定机构同意，可以用标准评价电路制备样品。所有样品应满足方法2020试验条件A或试验条件B的PIND试验要求。按方法1005试验条件A要求，在 \(125^{\circ}\mathrm{C}\) 下对样品进行 \(1000\mathrm{h}\) 的寿命试验。测量并记录初始电参数和寿命试验结束后的电参数。分析样品试验数据，确认是否由于存在吸附剂材料而引起器件特性的退化。  
注：若鉴定机构同意，也可以采用鉴定试验数据确认吸附剂材料是否引起了器件特性的退化。
