# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.4.2.2 漏率

---

光学检漏试验的试验压力 \((P)\) 和时间 \((t)\) 的确定应能保证所要求的漏率灵敏度。漏率由公式(5)给出：式中：  
\(O L_{\mathrm{He}}\) ——光学检漏得到的等效标准氦漏率， \((\mathrm{Pa}\bullet \mathrm{cm}^{3}) / \mathrm{s}\) (氦)；  
\(V\) ——封装的内空腔容积， \(\mathrm{cm}^3\)  
\(t\) ——试验时间，s；  
\(P_{0}\) ——绝对大气压力，Pa；  
\(D\) ——盖板的形变变化量， \(\mu \mathrm{m}\)  
\(P\) ——压力室的压力(表压)，Pa；  
\(S\) ——盖板刚度值， \(\mu \mathrm{m / Pa}\) 。
