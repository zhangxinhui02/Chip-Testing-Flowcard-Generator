# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5010.3 复杂单片微电路检验程序
### 3.2.3 外壳评价

---

每一种外壳在使用前应按表2要求进行评价及表征。可采用有限元分析法。用于评价的外壳应遵循同样的规范，采用相同的材料和表面镀涂，并经受同样的组装工艺。用于器件并按本方法要求加工的外壳应按下述要求进行试验：  
a）应从首次提供检验的外壳检验批中随机抽取样品进行外壳评价(见表2)。此外进行3分组试验应使用已密封的外壳，外壳内可以焊接有芯片。  
b）对随后订购的每一批外壳均应进行表2中的1、2和3分组试验。  
c）对表2中规定的可焊性(2分组)、引线牢固性(3分组)及金属外壳的绝缘(4分组)试验，应对引线或外引线按样本大小(接收数)为15(0)的抽样方案进行检验。外引线应从3个外壳中随机地抽取。
