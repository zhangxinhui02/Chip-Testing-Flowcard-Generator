# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 7.3.5 半易弯曲引线封装的试验程序

---

半易弯曲引线的试验力应为 \(2.22\mathrm{N}\pm 0.14\mathrm{N}\) 。除另外规定外，每根引线应进行三次 \(90^{\circ}\pm 5^{\circ}\) 的弯曲循环。  
7.3.6 双列封装和针栅阵列封装引线的试验程序  
7.3.6.1 双列直插封装(扁平、侧边钎焊、浴盆形引线)  
双列直插封装(扁平、侧边钎焊、浴盆形)插装时其引线通常平行排列并与封装底部成 \(90^{\circ}\) 。双列直插封装引线应向内弯曲一个角度三次，该角度应足以使引线保持 \(15^{\circ}\) 的永久弯曲(去掉应力后)。对于结构1和结构2,应从引线末端到第一弯曲处测量弯曲角度(见图1);对于结构3,应从引线末端到安装平面测量弯曲角度(见图1)。
