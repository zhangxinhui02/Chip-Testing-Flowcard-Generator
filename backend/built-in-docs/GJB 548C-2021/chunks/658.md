# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 7.3.9 小外形封装(SOP)和四边扁平封装(QFP)的翼形引线试验程序

---

SOP和QFP封装应按弯曲应力试验(试验条件 \(\mathrm{B}_{1}\) )的6.3.7进行三次弯曲循环，试验力由引线截面模量决定。  
矩形引线的截面模量大于横截面为 \(0.15 \mathrm{~mm} \times 0.51 \mathrm{~mm}\) 的等效矩形引线截面模量，或圆柱形引线的直径大于 \(0.51 \mathrm{~mm}\) ，则试验力为 \((2.22 \pm 0.14) \mathrm{~N}\) 。  
矩形引线的截面模量大于 \(0.10 \mathrm{~mm} \times 0.25 \mathrm{~mm}\) ，且小于或等于 \(0.15 \mathrm{~mm} \times 0.51 \mathrm{~mm}\) 的等效矩形引线截面模量，或圆柱形引线直径大于 \(0.25 \mathrm{~mm}\) ，且小于或等于 \(0.51 \mathrm{~mm}\) ，则试验力为 \((0.83 \pm 0.08) \mathrm{~N}\) 。  
矩形引线的截面模量小于或等于 \(0.10 \mathrm{~mm} \times 0.25 \mathrm{~mm}\) 的等效矩形引线截面模量，或圆柱形引线的直径小于或等于 \(0.25 \mathrm{~mm}\) ，则试验力为 \((0.42 \pm 0.03) \mathrm{~N}\) 。
