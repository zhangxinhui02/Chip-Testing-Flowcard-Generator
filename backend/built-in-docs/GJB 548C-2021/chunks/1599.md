# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.7.3 水汽和其他气体成分的长期放气试验

---

如果按5.8.7.2要求试验的5个封装样品已满足5.3.3的水汽要求，再对按5.8.7.1要求制备的封装样品中剩余的5个器件按方法1008要求，在 \(150^{\circ}\mathrm{C}\) 下加热 \(1000\mathrm{h}\) ，然后立即(间隔时间不得超过5min)置入内部气体分析仪器中，按方法1018进行内部气体分析。除水汽外，含量不小于 \(0.01\%\) (100ppm)的其他气体成分也应以百分数(V/V或ppm)为单位予以报告。
