# 中华人民共和国国家军用标准
## 微电子器件试验方法和程序
### 4.5.9.5 稳定功率温度条件

---

当规定时，器件应在规定的稳态加电条件下，在规定的测试温度( \(T_{\mathrm{A}}\) 、 \(T_{\mathrm{C}}\) 或 \(T_{\mathrm{J}}\) ，适用时)等于、高于或低于 \(25^{\circ}C\) 下至少稳定5min或规定的时间。电参数测量应在温度/功率稳定后尽可能快地或在规定时间内完成。作为替代方法，当有规定时器件温度 \(T_{\mathrm{C}}\) 或 \(T_{\mathrm{A}}\) 可稳定在规定稳定功率状态预计的典型结温的 \(\pm 3^{\circ}C\) 之内至少5min,采用低占空比脉冲技术进行测试。
