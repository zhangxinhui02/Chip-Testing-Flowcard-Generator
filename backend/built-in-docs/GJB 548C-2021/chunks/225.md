# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.3.1 设备

---

本试验所需设备为：a）真空/压力室：用于抽真空及随后加压，能使器件受到 \(724\mathrm{kPa}\) 的压力(绝对压力)作用达 \(23.5\mathrm{h}\) 。  
b）能保持指示用的液体温度在 \(125^{\circ}\mathrm{C}\) 并适于观察的容器，以及一套能将尺寸大于 \(1\mu \mathrm{m}\) 的粒子从液体中除去的过滤系统。  
c）1.5倍 \(\sim 30\) 倍的放大镜：将器件浸入指示用的液体中时，能用该放大镜观察到从器件中冒出的气泡。  
d）表3中所列的检测用I型液体和指示用II型液体。  
e）光源：它在空气中能在距离等于容器中离光源最远的器件与光源之间距离的位置处产生大于或等于 \(4572\mathrm{m}\) 烛光的亮度。光源不需要校准。但光源在观察位置上(即在观察气泡时放置被试器件的位置上)应产生足够的亮度。  
f) 已校准过的合适仪器：指示试验温度、压力和时间是否符合规定。  
g) 使器件浸入指示液中的合适夹具。
