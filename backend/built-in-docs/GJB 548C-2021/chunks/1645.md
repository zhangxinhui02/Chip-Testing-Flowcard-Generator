# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5012 数字微电路的故障覆盖率测量
### 3.1 逻辑仿真器

---

应用本试验方法需要采用一种能够仿真无故障数字逻辑电路所响应的测试向量序列工具，此工具就是逻辑仿真器。这种仿真能力称为逻辑仿真。  
为了仿真时序数字逻辑电路，仿真器应至少支持下述四种逻辑状态的仿真：低电平态(0)、高电平态(1)、高阻态(Z)及不确定态(X)。若仅仿真组合数字逻辑电路，仿真器至少应能仿真两种逻辑状态：低电平态(0)和高电平态(1)。  
对包括有时序逻辑的器件逻辑模型进行逻辑仿真时，每一条逻辑线和每一个含有存储功能单元的初始状态应都是X;如果采用其他初始条件，包括对任何逻辑线或存储器件采用确定的0或1作为初始条件，都应在故障仿真报告中予以说明，并证明其合理性。  
为了仿真线连结构或总线结构，仿真器应能处理由这类结构带来的信号竞争问题。否则，应允许在建立逻辑模型时去除这种结构(见4.1.2)。  
为了仿真时序数字逻辑电路，仿真器应支持事件驱动仿真，至少应支持单位延迟逻辑部件。  
只包含组合逻辑电路的仿真，或特殊情况下的时序逻辑电路的仿真(例如从可扫描时序逻辑模型中提取的组合逻辑)都可以采用非事件驱动仿真，例如拉平法、零时延法或编译码方法。故障仿真报告应阐述所选用的方法等同于更通用的事件驱动法的理由。
