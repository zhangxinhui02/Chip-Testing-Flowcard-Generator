# 中华人民共和国国家军用标准
## 方法1005.1 稳态寿命
### 3.6.6 试验条件F(仅适用于B级器件)(温度加速试验)

---

按此试验条件，器件在远超过其最高额定温度的环境温度 \((175^{\circ}\mathrm{C} \sim 300^{\circ}\mathrm{C})\) 下进行偏置。在较高温度下，器件通常不能正常工作，因此必须特别注意选择偏置线路和条件以保证重要的电路部位得到合适的偏置，且对电路的其他部位不会产生破坏性的过应力。为了适当地选用实际偏置条件，建议把合适的器件样品放在预定的高温环境中，在每个器件引出端上测量电压和电流，以保证规定的线路和外加的电位力不会诱导出破坏性的过应力。承制方可以从表1中自行选择其替代的时间和温度，可以选用时间界限在 \(30\mathrm{h} \sim 1000\mathrm{h}\) 范围内的任何时间- 温度组合。对于采用试验条件F进行的寿命试验，应采用方法1016中3.6寿命试验基本准则，并且得到鉴定机构的特殊批准。当证实在规定电压下工作会导致过量电流通过或过量功耗时，任一或全部引出端上的外加电压可以降低到不低于规定电压的 \(50\%\) 。如果降低电压，则应按公式(1)确定寿命试验时间:式中:  
\[t_{\\mathrm{a}} = \\frac{t_{0}(100\\%)}{100\\% - V\\%} \\quad (1)\]  
\(t_{\mathrm{a}}\) ——调整后的总试验时间， \(\mathrm{h}\) ;  
\(t_{0}\) ——原来的试验时间， \(\mathrm{h}\) ;  
\(V\%\) ——最大的电压降低量在规定电压中所占的百分比。  
对于具有内部过热保护的器件，在超过保护温度的温度下，即使延长试验时间也不可能得出长期工作可靠性的真实结果。对于这种器件，应在规定偏压下的几个不同环境温度中测量外壳温度，并根据这些测量数据来计算结温。工作寿命试验的环境温度应按以下原则确定，即在规定偏压下该环境温度使最坏情况下的结温比进入热保护的最低结温低 \(5^{\circ}\mathrm{C} \sim 10^{\circ}\mathrm{C}\) 。不同级别器件的试验时间由表1规定。
