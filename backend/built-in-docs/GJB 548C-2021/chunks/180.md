# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 4.2.3.1 一般考虑

---

本试验的目的是用芯片上特定半导体单元给出器件的结温来确定集成电路的热阻。为了获得实际平均工作结温 \(T_{\mathrm{J}(\mathrm{平均})}\) ，应给封装壳体内的整个芯片或每个芯片通电，以提供合适的内部温度分布。测量4.2.1所述的其他参数，仅需对用于测温的结通电。在测量结温时，应切断芯片的加热电流而保持结的校正电流稳定。如果假定校正电流不会影响电路工作，那么当切断电源时，校正电流就必须接通。  
为了计算热阻，可用温敏参数来指示半导体器件的平均结温。所测结温仅仅指出紧靠测热单元附近的温度。这样，对于加热电流是恒定的，如果用于测温的结也有功耗，则对于特定的结， \(T_{\mathrm{J}(\mathrm{平均})}\) 近似等于 \(T_{\mathrm{J}(\mathrm{峰值})}\) 。如果加热电流不是恒定的，则测得的将是 \(T_{\mathrm{J}(\mathrm{平均})}\) 。如果用于测温的结是紧靠功耗单元位置，则测得的将是 \(T_{\mathrm{J}(\mathrm{区域})}\) 。当测量 \(T_{\mathrm{J}(\mathrm{区域})}\) 时，加热电源不必关闭。  
一般用于间接测量结温的温敏电参数是二极管正向电压、双极晶体管的发射极一基极正向电压和集电极一基极正向电压。当因为制造工艺原因，而不适宜测量有源结电压时，可用其他合适的温敏参数来间接测量结温。例如，P- N 结隔离的单片集成电路中的衬底二极管(参数)可以用作温敏参数去测量 \(T_{\mathrm{J}(\mathrm{区域})}\) 。在这种特殊的情况下，在衬底二极管上加正向偏置的同时，加热电源必须切断。
