# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 4.4.2 计算封装热响应时间

---

当按4.3的具体步骤分别测量了随时间变化的结温的峰值 \(T_{\mathrm{J}(\mathrm{峰值})}\) 、平均值 \(T_{\mathrm{J}(\mathrm{平均})}\) 或区域值 \(T_{J}(\mathrm{区域})\) 之后，可计算得到器件的热响应时间。若要估计整个封装的热响应时间，应对被试器件施加额定功率 \(P_{\mathrm{D}(\mathrm{封装})}\) 。若用作为质量控制，可用单个被测结的功耗 \(P_{\mathrm{DD}}\) (单结)来计算热响应时间。  
当得到上述各项测量的数据之后，热响应时间可从结温与时间的关系曲线上确定。测量从施加或去掉加热功率的时刻开始到结温达到稳定值为止的时间，该时间的0.9倍就是热响应时间。
