# 中华人民共和国国家军用标准
## 方法2018.2 扫描电子显微镜(SEM)检查
### 4.1.2.3.3 玻璃钝化前的弓形抽样

---

只有当以后的晶圆制造工艺温度低于 \(450^{\circ}C\) (723K),且互连金属条的宽度不小于 \(3\mu \mathrm{m}\) 时才能用本取样方式。如果向鉴定机构提供的相关性数据表明，本程序与采用常规腐蚀的程序之间没有区别，并已得到鉴定机构的批准，本取样方法也可用于较高温度或较细线宽的情况。  
在金属化和腐蚀后，但在玻璃钝化前，从待检查的每个晶圆上相对的两边切下两个弓形。组成弓形的弦边应位于沿着半径方向距晶圆边缘1/3半径的位置。然后在每个弓形的弦边上距两端约 \(1.5\mathrm{cm}\) 处各取一个芯片(共有四个芯片)作SEM检查。
