# 中华人民共和国国家军用标准
## 方法1017.1 中子辐射
### 4.4 辐射

---

4.4 辐射样品和剂量计应接受规定的中子注量辐射，若要求多次辐射，在每次辐射后应进行辐射后电测试（见4.5.1）。若要求使用剂量计，每个辐射等级使用一套新的剂量计。因为中子效应是累计的，所以必须确定每次附加的辐射量，以满足规定的总累计注量。全部辐射应在 \(24^{\circ}\mathrm{C}\pm 6^{\circ}\mathrm{C}\) 下进行，并推算出1MeV等效中子注量时的情况。为避免电离总剂量损伤对NIEL效应的干扰，样品接受的中子注量不应使其承受高于额定值 \(10\%\) 的电离总剂量。若有必要，允许采用屏蔽措施以降低伴生的 \(\gamma\) 总剂量辐射。
