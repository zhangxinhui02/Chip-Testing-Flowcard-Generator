# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.12.1.3 电阻率测量

---

采用带有专用四探针测试夹具的毫欧表测量电阻。四探针夹具可以由一块聚丙烯材料和四根弹压接触探针组成。这四根接触探针必须固定插入在聚丙烯材料中， 使两根电流接触探针之间的间距为 \(5.08\mathrm{cm}\) ， 两根电压接触探针位于两根电流接触探针之间， 且电压接触探针与相近的电流接触探针之间距离应为 \(1.27\mathrm{cm}\) 。将四探针夹具放置在导电性聚合物样品片上， 确保每个探针与材料之间电接触。以欧姆为单位记录测量的电阻值， 采用公式(1)计算电阻率:式中:  
\[\\rho = \\frac{R(W\\times t)}{L} \\quad (1)\]  
\(\rho\) ——电阻率， \(\Omega \cdot \mathrm{m}\) ;  
\(R\) ——电阻测量值， \(\Omega\) ;  
\(W\) ——样品宽度， \(\mathrm{mm}\) ;  
\(t\) ——样品厚度(等于材料加载波片的总厚度测量值减去载玻片的厚度), \(\mu \mathrm{m}\) ;  
\(L\) ——中间两根探针(即测量电压的两根探针)之间的距离， \(\mathrm{mm}\) 。  
应该在 \(25^{\circ}\mathrm{C}\) 、 \(60^{\circ}\mathrm{C}\) 、 \(150^{\circ}\mathrm{C}\) ,以及在 \(150^{\circ}\mathrm{C}\) 空气或氮气环境中存放 \(1000\mathrm{h}\) 之后的 \(25^{\circ}\mathrm{C}\) 这四种条件下，至少分别测量三个样品的电阻率。同一个样品可以用于这四个条件下的测量。
