# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.12 质量分析的实例

---

对LPCVD工艺，已完成的缺陷表征工作表明，主要的缺陷机理是存在颗粒。使用激光表面扫描仪可以定量确定这些颗粒。检验结果表明，按尺寸大小，这些颗粒可分为三类：a) 小于 \(0.3\mu \mathrm{m}\) ，这种颗粒在晶圆内和晶圆间随机分布；  
b) \(1.0\mu \mathrm{m}\) 左右，这种颗粒在淀积炉管末端靠近系的地方密度大；  
c) 大于 \(6.0\mu \mathrm{m}\) ，这些颗粒分布不均匀，集中出现在某些晶圆上，但在其他晶圆上根本没有。  
当利用光学显微镜观察时， \(1.0\mu \mathrm{m}\) 或更小颗粒看来相当小且发暗；而大颗粒(大于 \(6\mu \mathrm{m}\) )看来既大，且是黑色的，分布在晶圆表面。对3种颗粒的成分进行分析后，发现前两种(小于 \(0.3\mu \mathrm{m}\) 和约为 \(1.0\mu \mathrm{m}\) )颗粒是由硅和氧构成，基本上与淀积膜成分相同；大颗粒则主要是由铁(Fe)和镍(Ni)组成。  
包含较小尺寸颗粒的晶圆，接下来将进行光刻，当 \(1.0\mu \mathrm{m}\) 颗粒靠近图形线条时，这些颗粒将对光刻图形产生影响。而小于 \(0.3\mu \mathrm{m}\) 的颗粒，没有观察到它对图形有影响。这两种缺陷可以通过光学显微镜和自动图形检测系统来检测。去掉光刻胶后，虽然 \(1.0\mu \mathrm{m}\) 的颗粒也随之而去，但它对形成图形的影响是明显的。小于 \(0.3\mu \mathrm{m}\) 的颗粒在去胶后仍可观察到。在后来的低温氧化物(LTO)淀积工艺中，由于低温氧化物的形貌保持特性，小于 \(0.3\mu \mathrm{m}\) 的颗粒似乎是放大到尺寸约为 \(1\mu \mathrm{m}\) 。这种“延伸”了的颗粒将对下一步光刻产生显著影响。利用扫描电子显微镜(SEM)对这两种颗粒进行观察，可以看到 \(1\mu \mathrm{m}\) 的颗粒嵌入膜中，而小于 \(0.3\mu \mathrm{m}\) 的颗粒则位于膜下方。这与随后的工艺加工中观察到的缺陷行为相一致。  
大颗粒可能来自不锈钢污染。利用扫描电子显微镜对缺陷进行观察表明，这些颗粒都位于淀积膜的上面。还发现这些颗粒来自LPCVD系统的卸载臂，由于卸载臂偶尔会撞击装载/卸载架的其他部件，每次撞击都会产生一些金属颗粒。  
根据对LPCVD工艺缺陷的表征结果，可以采用下列方法监测：a）对装载/卸载架每周进行一次预防性维护工作，会明显影响卸载臂的对位，因此在每次预防性维护之后，晶圆加工之前先使用颗粒监测仪检测 \(6\mu \mathrm{m}\) 或更大的颗粒，以确保在卸载臂正常工作时，不会再产生这类颗粒。  
b） \(1.0\mu \mathrm{m}\) 颗粒的来源不清，只知道这些颗粒通常在淀积炉管末端靠近的泵的附近很多。因此，对于这种颗粒，应在淀积炉管末段的泵附近实施颗粒监测，同时对淀积炉管进口处同步进行监测。对每个监测点，设置不同的行动极限。  
c）LPCVD工艺中，小颗粒很难监测，这是因为它们的尺寸属于颗粒检测仪的“测量噪声”范围。然而，在下一个光刻后，采用自动图形检测系统进行监测(用“追溯”的方式)就会比较容易。因此，这种缺陷检测在图形形成后的检验步骤进行，并及时反馈给LPCVD工序。
