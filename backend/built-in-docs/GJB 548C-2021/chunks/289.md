# 中华人民共和国国家军用标准
## 方法1017.1 中子辐射
### 3.2 辐射源

---

试验采用的中子辐射源应经过充分评估，可以产生宽中子能谱(例如TRIGA反应堆或快中子反应堆)，或单能中子谱(例如氘( \(\mathrm{DT}\) )或氘( \(\mathrm{DD}\) )加速器)，并经过比对能将输出注量转化为1MeV等效中子注量。中子谱的不确定度采取按能量相关的协方差矩阵形式描述。中子源在使用前，应确定每单位中子注量的伴生电离辐射量。若每 \(1\times 10^{12} / \mathrm{cm}^2\) (1MeV等效)中子注量对应的电离总剂量(TID)超过5Gy(Si)时，则不应采用此类辐射源。可采用订购方接受的一种辐射源。
