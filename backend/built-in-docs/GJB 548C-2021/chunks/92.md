# 中华人民共和国国家军用标准
## 方法1005.1 稳态寿命
### 3.6.5 试验条件E(环形振荡器试验)

---

如图3所示，最后一个电路的输出端通常连接到第一个电路的输入端。这种串联会产生自由振荡，其频率由各个电路和有关布线的传输延迟确定，该频率应不低于 \(50\mathrm{Hz}\) 。在电路会引起倒相的情况下应采用奇数个电路。环路中的每个电路应加上额定的最大负载。虽然该条件提供了连续监视致命失效（即环路阻塞)的机会，但不得用它来代替中间测试(见3.4)。
