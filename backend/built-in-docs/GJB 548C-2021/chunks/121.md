# 中华人民共和国国家军用标准
## 方法1009.2 盐雾
### 4.4 试验样品的安置

---

样品的安置，应使它们彼此不接触，彼此不遮挡，能自由地接受盐雾作用，腐蚀生成物和凝聚物不会从一个样品落到另一个样品上。样品应按下述方位安置在固定的夹具上(有机玻璃棒、尼龙或玻璃纤维筛、尼龙绳等)。  
a) 引线固定于封装侧面或引线从封装侧面引出的双列封装(如侧面钎焊双列封装和陶瓷双列封装):盖板向上，偏离垂直方向 \(15^{\circ} \sim 45^{\circ}\) 。将有引线的一个封装侧面向上，偏离垂直方向大于或等于 \(15^{\circ}\) (见图1a))。  
b) 引线固定于封装底部(与盖板相对的那一面)或引线从底部引出的封装(如金属圆形封装):盖板偏离垂直方向 \(15^{\circ} \sim 45^{\circ}\) 。试验时一半样品应盖板向上；剩下一半，引线向上(见图1b))。对于在盖板同一侧固定或引出引线的封装，只需按一个方向放置(盖板和引线向上)。  
c) 引线固定于封装某一面或从某一面引出，且与盖板平行的封装(如扁平封装):盖板偏离垂直方向 \(15^{\circ} \sim 45^{\circ}\) 。将有引线的封装面向上，且偏离垂直方向大于或等于 \(15^{\circ}\) 。对金属壳封装，试验时一半样品盖板向上。剩下的样品底座向上。其他封装都应盖板向上做试验(见图1c))。  
d) 无引线和有引线片式载体：盖板偏离垂直方向 \(15^{\circ} \sim 45^{\circ}\) ，试验时一半样品盖板向上，剩余的样品盖板向下(见图1d))。  
e) 扁平样品(如仅对盖板和仅对引线框架):偏离垂直方向 \(15^{\circ} \sim 45^{\circ}\) 。  
对于要求进行两种取向的试验，应把规定的样品数分成两等份(或尽可能接近一半)。在所有的情况下，对所有的封装表面按4.8进行试验后的检查。  
对开有窗口的紫外线可擦器件进行试验时，需采取保护措施，以防止光感应电动势产生电解作用。
