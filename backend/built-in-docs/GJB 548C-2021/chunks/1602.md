# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.8.3 萃取程序

---

在由耐热玻璃(或等效的其他物质)制成的光洁的 \(250\mathrm{mL}\) 分析烧瓶中，加入净重量为 \(3\mathrm{g}\) 的经研磨或切割的等效聚合物材料。烧瓶中固化材料的重量记录应精确到毫克。给烧瓶加入电导率测量值不大于 \(0.1\mathrm{mS / m}\) (电阻率不小于 \(1.0\mathrm{M}\Omega \cdot \mathrm{cm}\) )的去离子水 \(150.0\mathrm{g}\) 。再准备一个作对比用的 \(250\mathrm{mL}\) 烧瓶，加入 \(150.0\mathrm{g}\) 去离子水(和极热的片状样品)。两个烧瓶应回流 \(20\mathrm{h}\) 。  
注： \(1.0 / \Omega = 1.0\mathrm{S}\) ； \(1.0 / \Omega \cdot \mathrm{cm} = 100.0\mathrm{S / m}\) 。
