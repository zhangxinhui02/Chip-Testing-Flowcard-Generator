# 中华人民共和国国家军用标准
## 方法1021.1 数字微电路的剂量率翻转
### 5.13 报告

---

报告中至少应包括下述内容:a) 器件识别；  
b) 试验日期和试验操作人员；  
c) 试验设施辐射源规范以及辐射脉冲宽度；  
d) 偏置条件， 输出负载以及测试线路；  
e) 试验中选择状态向量的方法描述；  
f) 用于辐射试验的状态向量和每种状态向量的功能测试条件；  
g) 瞬态输出翻转的判据；  
h) 对每种状态向量的翻转阈值和电源电流记录；  
i) 设备清单；  
j) 噪声试验结果；  
k) 温度(见5.10)。
