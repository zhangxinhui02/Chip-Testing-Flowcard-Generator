# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5005.3 鉴定和质量一致性检验程序
### 3.1 通则

---

针对不同器件类型和等级的3.2、3.3或3.4所包含的程序，应适用于全部的鉴定和质量一致性检验要求，一个检验组内各分组可按任何顺序进行。但一个分组内的各项试验(S级B2分组除外),应按B组、C组、D组和E组检验中的指定顺序进行。当B组、C组、D组和E组检验的分组中要求进行终点电测试时，应按相关的详细规范规定进行这些测试。当要求终点测试，但在订购文件中并未指明任何参数时，应将 \(100\%\) 筛选的最终电测试参数作为终点测试。  
若器件封装的密封内腔周长大于或等于 \(51\mathrm{mm}\) ，或器件的质量大于或等于 \(5\mathrm{g}\) ，恒定加速度试验可按适用情况，自行选择以下的规定进行处理：删除试验条件E,采用适用的器件规范中规定的试验条件来代替。除另有规定外，大型单片器件的应力水平，不应低于试验条件D。如果规定的应力水平低于试验条件D,承制方应提供数据，说明降低应力水平的合理性，并取得标准化机构或订购方的确认，允许的最小应力水平为试验条件A。  
当初始鉴定或产品设计更改时，器件应在筛选之前按GJB 597B- 2012和方法3015的规定进行ESDS等级鉴定。ESD的标志及处理应按GJB 597B- 2012的规定。  
辐射强度保证器件的鉴定和质量一致性检验要求，除正常的S级和B级要求之外，还要满足表6对每一种规定的辐射等级(M、D、P、L、R、F、G和H)的要求。
