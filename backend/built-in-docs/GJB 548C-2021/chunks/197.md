# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 2.3 漏率转换因子 leak-rate conversion factor

---

采用不同气体媒质表征漏率时的定量转换计算因子。  
注：不同气体媒质的漏率转换因子如下:a) \((\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}(\mathrm{氮}85)\times 1.71 = (\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}(\mathrm{空气})\)  
b) \((\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}(\mathrm{氮}85)\times 4.61 = (\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}(\mathrm{氮})\)  
c) \((\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}(\mathrm{氮})\times 0.37 = (\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}(\mathrm{空气})\)
