# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.7.1 室温辐射

---

由于辐射效应与温度有关，被试器件在辐射时，从试验箱中靠近夹具的位置测得的环境温度应为 \(24^{\circ}\mathrm{C}\pm 6^{\circ}\mathrm{C}\) 。应在 \(24^{\circ}\mathrm{C}\pm 6^{\circ}\mathrm{C}\) 的环境温度下对被试器件进行电测试。如果采用移位测试，不允许被试器件的测试环境温度比辐射温度高 \(10^{\circ}\mathrm{C}\) 。若要求其他温度范围，应作出规定。  
注：在高于辐射温度的环境温度中，退火可能非常明显，特别是对于低剂量率条件(条件D)下的长时间试验。为了使退火效应减少到最低程度，将器件温度保持在上述温度范围内非常重要。
