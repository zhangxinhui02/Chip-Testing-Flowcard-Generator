# 中华人民共和国国家军用标准
## 方法2005.1 振动疲劳
### 3.1 试验过程

---

把器件牢固地安置在振动台上， 引线或电缆也应适当固定。器件做等幅简谐振动， 其峰值加速度应符合表 1 规定的试验条件。对于试验条件 A, 频率为 \(60\mathrm{Hz}\) (误差为 \(\pm 20\mathrm{Hz}\) ), 振动两倍幅值为 \(1.52\mathrm{mm}\) (总偏移) 的等幅谐振运动可用来代替规定的峰值加速度。振动时间至少要在 X、Y 和 Z 每个方向上维持 \(32\mathrm{h} \pm 8\mathrm{h}\) ， 且总时间至少为 \(96\mathrm{h}\) 。当有规定时， 对于壳内所含部件或元件在振动时易移动和易受到破坏的器件， 应按方法 2012 用 X 射线检查方法进一步检查， 或去掉封盖或打开外壳， 放大 30 倍进行目检， 检查部件或元件是否遭到破坏或错位。当本试验作为一个试验组或试验分组的一部分来进行时， 在本试验结束时不必专门进行试验后测试或检查， 可在该组或分组结束时进行。
