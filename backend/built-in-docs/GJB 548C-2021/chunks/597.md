# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 5.3.3.2 准备

---

应按5.2要求准备样品。  
a）对于焊片、转接片、接线端、直径大于 \(1.14\mathrm{mm}\) 的实心导线及直径大于 \(1.02\mathrm{mm}\) 的绞合线使用标准可焊线时，所有样品应用标准导线缠绕待试验部分1.5匝。  
b）用4.7规定的标准缠绕线缠绕后，在浸焊料期间不应相对移动。图5是这种导线缠绕方式的示例。  
c）若需要，在规范中应单独规定涉及样品缠绕部分的特殊说明。  
d）对设计用于直径小于 \(0.6\mathrm{mm}\) 引线的焊片、转接片，4.7规定的标准铜缠绕线应与所设计的焊片、转接片的尺寸相同。
