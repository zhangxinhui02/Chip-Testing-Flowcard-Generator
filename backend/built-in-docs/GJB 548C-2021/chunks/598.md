# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 5.3.3.3 步骤

---

具体步骤如下：a）焊剂应处于室温，并符合4.1要求；  
b）引出端浸入焊剂中的最小深度应能覆盖被试表面；  
c）被试表面应浸入焊剂 \(5\mathrm{s}\sim 10\mathrm{s}\) ，并允许沥干 \(10\mathrm{s}\sim 60\mathrm{s}\)  
d）在引出端浸焊料前，应从熔融焊料表面撤去焊剂浮渣及焦渣；  
e）浸入和提出速度为 \(25\mathrm{mm / s}\pm 6\mathrm{mm / s}\) ，停留 \(7\mathrm{s}\pm 0.5\mathrm{s}\)  
f）样品系吊到浸焊料装置，已涂焊剂的引出端浸入熔融焊料中一次，深度与5.3.3.3b)相同；  
g）浸完后，允许被试件在空气中冷却；  
h）检验前，应按4.1.3要求，去除所有引出端上的全部可见残余焊剂。
