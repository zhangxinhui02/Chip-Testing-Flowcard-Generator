# 中华人民共和国国家军用标准
## 方法1009.2 盐雾
### 4.8.1 通则

---

除 \(4.8.2\mathrm{b}\) 和 \(4.8.2\mathrm{c}\) 另有规定外，所有检查都应在放大10倍 \(\sim 20\) 倍的情况下进行。其中：a) 腐蚀色斑不应认为是4.8.2a)中所指的缺陷；  
b) 由引线腐蚀产生的淀积在引线以外部位的腐蚀生成物，不应认为是4.8.2a)中所指的缺陷；  
c) 引线端头的腐蚀和由此腐蚀产生的腐蚀生成物，不应判为不合格；  
d) 若由于设计或几何形状尺寸(例如针栅阵列封装的引线底部或陶瓷双列封装的钎焊部分)不能按 \(4.8.2\mathrm{b}\) 作进一步试验的引线，应根据4.8.2a)失效判据进行判定。
