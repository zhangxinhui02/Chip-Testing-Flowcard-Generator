# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.11.2 需要进行补充室温退火试验的情况

---

应根据如下标准来确定是否需要进行补充室温退火试验的情况:a) 退火试验程序适用于MOS或双极工艺器件。  
b) 退火试验程序仅适用于参数失效(而非功能失效)情况。试验的相关方应采取适当步骤以确定被试器件在整个总剂量试验过程中只发生了参数失效。  
c) 退火试验程序适用于用被试器件自然退火响应来修正参数响应超出规范值的情况。而且，对预定应用环境辐射剂量率很低，而辐射诱生陷阱正电荷的室温退火会导致器件性能的明显改进，该退火试验程序会导致较高的器件接收率。当预定应用环境辐射剂量率比试验剂量率低，且低于 \(0.001\mathrm{Gy(Si)} / \mathrm{s}\) 时，应考虑使用该程序。试验相关方应采取合适的措施确保其试验技术在整个总剂量试验范围内，能获得所要求的退火响应。
