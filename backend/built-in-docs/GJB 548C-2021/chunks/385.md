# 中华人民共和国国家军用标准
## 方法1020.2 剂量率感应锁定
### 5.5.1 通则

---

在生产试验之前，对于新的或不熟悉的器件类型(即新的设计或工艺过去很少或未进行过锁定试验的不熟悉的或很复杂的器件)至少要进行一次表征试验。表征试验的结果将用于确定生产试验的要求(见5.4)。在适用的试验方案或程序中应规定这些要求(包括第3章中列出的条款)。  
除另有规定外，每个脉冲的剂量应为5Gy(Si) \(\pm 2\mathrm{Gy}\) (Si)，脉宽为 \(20\mathrm{ns} \sim 100\mathrm{ns}\) (包括上述范围两端的边界值)。应按试验方案或程序中的规定，(对逻辑器件)至少应使电路以两种不同的工作状态接受辐射脉冲的作用。除另有规定外，应根据被试器件电源电流、在规定时间范围内输出信号电压的恢复以及辐射后的现场功能测试这几方面的综合情况确定是否发生了锁定。在辐射后的现场测试结束之前，不得中断被试器件电压供给。应在辐射脉冲开始作用之前以及作用后的规定时刻测量被试器件电源电流，以确定电源电流是否已返回至规定的范围。在恢复以后立即进行功能测试，以确定器件功能是否正常。除另有规定外，器件的试验温度应取为其在系统应用时的最高器件工作温度或比器件最高额定温度低 \(15^{\circ}C\) (取其小者)。只有事先已得到订购方的批准，才可以使用限流电阻。且阻值应小于或等于在系统应用中的大小。除另有规定外，应在锁定试验前后均进行终点电测试(至少要按方法5005或GJB2438进行A1分组和A7分组测试)这些测试通常都不是在现场进行。对进行测试的时间也没有限制。如果这些测试是作为另一个测试的组成部分(例如：老炼后测试、最终电特性接收测试)，只要试验顺序是：A组测试- 锁定试验- A组测试，不需要重复进行A组测试。
