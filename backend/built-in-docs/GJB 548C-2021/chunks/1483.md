# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5010.3 复杂单片微电路检验程序
### 3.3.4.3 鉴定用器件

---

应采用鉴定试验来证明工艺的稳定性和可靠性，鉴定所用器件，可以是标准评价电路(SEC)，也可以是实际器件(最坏情况设计)，这些器件应包含本方法所详述的基本信息，并应提交鉴定机构批准。鉴定用器件制造所用的设计规则和工艺应与常规生产所采用的设计规则和工艺相同。鉴定用器件的设计应利用单元库，还要考虑评价底层结构(如扩散层)的可靠性，以及体现不同层次最坏情况下的设计规则。适用时，工艺加工中还应采用可以检测金属与金属间短路或开路、高通孔电阻及介质针孔(适用时)等测试结构。建议使用表3的测试结构。  
|  |  |
| --- | --- |
| 参数 | 测试结构 |
| 功能及性能 | 大功能块（如ALU）、环行振荡器 |
| 接触电阻／电迁移 | 接触孔链 |
| 热电子／空穴 | 短沟道器件 |
| 氧化层击穿电压 | 电容器 |
| 电阻（电迁移试验） | 金属条 |
