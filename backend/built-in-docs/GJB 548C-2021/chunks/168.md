# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 2.6 热响应时间 thermal response time

---

\(t_{\mathrm{JR}}\)  
结温变化达到最终值 \(90\%\) 所需要的时间。结温变化是在器件规定参考点温度保持不变的情况下，由于功耗的阶跃变化造成的。为了与测量结温的特定方式相一致，热响应时间也分为 \(t_{\mathrm{JR}(\mathrm{峰值})}\) 、 \(t_{\mathrm{JR}(\mathrm{平均})}\) 和 \(t_{\mathrm{JR}(\mathrm{区域})}\) 。
