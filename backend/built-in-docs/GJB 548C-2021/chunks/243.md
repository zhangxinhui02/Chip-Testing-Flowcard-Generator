# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.6.3 失效判据

---

对内腔体积小于或等于 \(0.01\mathrm{cm}^3\) 的器件，如果器件重量增量大于或等于 \(1.0\mathrm{mg}\) ，该器件应拒收。对内腔体积大于 \(0.01\mathrm{cm}^3\) 的器件，重量增量大于或等于 \(2.0\mathrm{mg}\) 为拒收。对已按重量分组的器件，若器件增加了足够的重量，使其按重量划分的级别增加一级以上，则该器件为拒收。失去了一定重量的器件(结果增加了该重量就会引起器件拒收),可在 \(125^{\circ}\mathrm{C}\) 下烘烤 \(8\mathrm{h}\) 后重新试验。
