# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2024 玻璃熔封盖板的扭矩
### 3.1 概述

---

试验时固定器件的基座，将扭矩加在器件盖板上；反过来也可以。安装盖板的扭转装置是为了保证扭矩只加在封装盖板、基座或其间垫层的侧面，避免接触密封玻璃。盖板扭转装置可以接触封装外引线，但不能让大部分转矩直接通过外引线而转移。扭矩应逐渐平缓地施加，直到封装发生分离，或达到 \(12.8\mathrm{N}\cdot \mathrm{m}\) 的极限扭矩为止。记录分离封装所需要的扭矩或达到 \(12.8\mathrm{N}\cdot \mathrm{m}\) 的极限扭矩。施加扭矩时应使旋转轴与密封区平面垂直，且位于密封区的中心(见图1)。
