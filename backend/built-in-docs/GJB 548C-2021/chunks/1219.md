# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2032.1 无源元件的目检
### 4.3.2.1 金属化划伤

---

呈现下列情况的元件不得接收：a）H级：除键合区外，使金属化中的划伤或探针划痕沿导电带宽度方向上暴露出下层材料（下层材料不一定在沿整个划伤的长度方向上全部暴露），并使未受影响的部分小于初始宽度的 \(50\%\) （见图38）。本判据不适用于电容器。  
K级：同H级。  
![](assets/95d41aedb2094d052e165d6c33cd73da88a70a0e88d9f87cc0a7e1af5532037e.png)  
b) H级：在键合点区域内的划伤，暴露了下层材料并使得金属化走线进入键合点的位置处金属化通路宽度减少至其初始金属化宽度的 \(50\%\) 以下，如果有两个或多个金属化走线进入一个键合点，对每一条走线应分别考虑(见图39a)。  
K级： \(75\%\) 以下。  
c) H级：在键合点区域内的划伤或探针划痕使暴露出的下层材料超过初始金属化区域的 \(25\%\) 。K级：同H级。
