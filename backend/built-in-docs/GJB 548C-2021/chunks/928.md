# 中华人民共和国国家军用标准
## 方法2017.2 内部目检(混合电路)
### 4.2.10 封装情况在10倍 \(\sim 60\) 倍放大倍数下检查

---

呈现下列情况的器件不得接收：a) 在封装内部或凸缘上存在没有固定住的多余物。  
注：多余物应视为未固定的，除非能证明其是固定的。当多余物最大长度大于最邻近的未被玻璃钝化的导体间距的 \(75\%\) 时， 可用适当的机械器具(即针、探针、钓针等)轻轻触动它来核实附着情况， 可以采用订购方批准的清洗工艺来验证较小的外来物附着情况， 所有的多余物或颗粒均可用气流吹(大约 \(138\mathrm{kPa}\) )以验证其附着情况。  
b) 附着的多余物使金属化层通路、封装的两引线、封装金属化层与引线、各功能电路元件、各个结或者以上各部分之间桥接。  
c) 液滴或任何化学沾污使未被玻璃钝化的工作金属表面之间桥接。  
d) 阻碍充分密封的密封面的物理损伤或污染(共晶或聚合材料)。  
e) 存在残余助焊剂(使用10倍\\~15倍放大)。  
f) 看不到轮廓线的熔融多余物。
