# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.11 对文件和数据的要求

---

A.11.1 鉴定机构需要时可以要求承制方提交缺陷表征结果、筛选方法有效性评估、抽样和检验方法、更改控制程序和体系等有关情况。  
A.11.2 检验和筛选程序应纳入文件和更改控制管理。数据记录应予保存，在鉴定机构要求时应提交审查。应按采购规范要求保存数据。  
A.11.3 偏离控制程序应形成文件，并受控。适用时，按要求对偏离的根本原因进行的分析、控制、处理和纠正措施等记录(通过MRB或经认可的其他管理机构)应予保存，鉴定机构要求时应提交。规定允许差异的程度对每条生产线来说都是非常重要的，因此，处理权限实施可以从责任个体到MRB，文件的范围包括从批次随工单到MRB报告。承制方应有用于各种处理方法的指南和文件(如：产品偏离在一定范围内，未超出行动极限，生产线工程师有处理权，如果超出这些行动极限，则需要更高的处理权限)。应按采购规范要求保存数据。
