# 中华人民共和国国家军用标准
## 方法1023.1 线性微电路的剂量率响应和翻转阈值
### 运算放大器试验示例

---

A.1 本附录给出了一个例子， 用来说明如何规定一个运算放大器的试验条件。由于试验条件同时取决于器件类型和特定的使用情况， 因此不能认为本例适用于任何情况， 它只为了说明如何使用本试验方法。  
A.2 方法1023试验规范:a) JF741型运算放大器， 8引线金属圆形封装；  
b) 试验电路如图A.1所示， 其中外引线1、5和8未接；  
c) \(V_{+} = 9.0\mathrm{V}\pm 0.2\mathrm{V}\) ; \(V_{-} = -9.0\mathrm{V}\pm 0.2\mathrm{V}\)  
输入信号电压峰峰值为 \(280\times (1\pm 5\%) \mathrm{mV}\) ； 频率为 \(2000\mathrm{Hz}\pm 50\mathrm{Hz}\) ；  
d) 监测6号引出端和电源电流；  
e) 采用标准的噪声极限值要求；  
f) 脉冲宽度: \(20\mathrm{ns}\) ;  
g) 总剂量不得超过 \(10\mathrm{Gy(Si)}\) ；  
h) 试验中使用的剂量率为 \(10^{5}\times (1\pm 30\%) \mathrm{Gy(Si)} / \mathrm{s}\) ；  
i) 试验环境温度为 \(25^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) ；  
j) 接收或拒收失效判据： 电源电流和输出信号应在辐射脉冲的 \(1\mathrm{ms}\) 以内回到辐射前水平的 \(10\%\) 以内；  
k) 本试验是破坏性试验。
