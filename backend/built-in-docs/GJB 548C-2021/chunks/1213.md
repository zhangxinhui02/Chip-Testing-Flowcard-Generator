# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2032.1 无源元件的目检
### 4.2.5 基板缺陷(高放大倍数)

---

呈现下列情况的元件不得接收:a) H级：工作金属化和元件边缘之间间隔小于 \(3\mu \mathrm{m}\) ，设计要求除外(见图10)。本条款不适用于包含有周边导体的元件，或与元件边缘对齐的射频/微波元件的键合区(在设计图上标注);当键合区金属化与元件边缘对齐时，元件边缘处的键合区金属化和其他任何非同一导电表面之间应至少有 \(25\mu \mathrm{m}\) 的间隔。  
K级：同H级。  
![](assets/2561135d9790bb85ffbc09496e84125f97c98e7fb042f1deda74d0a68078b3ca.png)  
b) H级：扩展到有源电路区的基板缺口(见图10)。  
K级：同H级。  
c) H级：长度超过 \(127\mu \mathrm{m}\) 的任何裂纹(见图11)。对于融凝石英或晶体不能有龟裂。  
K级：同H级。  
d) H级：任何与有源电路区或工作金属化距离小于 \(3\mu \mathrm{m}\) 的裂纹(见图11)。  
K级：小于 \(7\mu \mathrm{m}\) 。  
e) H级：由元件边缘直接扩展向有源电路区或工作金属化的裂纹长度超过 \(25\mu \mathrm{m}\) (见图11)。K级：同H级。  
![](assets/9da8d7acaeec5e6be2e09d3f84e2e44ce7320f9ff66d74174722d55da8fdf43c.png)  
f) H级：无要求。  
K级：总长度大于等于两个键合点间最小间隔的 \(75\%\) 的半圆形裂纹或元件边缘方向上的裂纹组合(见图12)。  
![](assets/488379e89ddadba33e2457eb7d6172cdc43b98bc4ec83e0541c4982560f95159.png)  
g) H级：有源电路区上来自相邻元件的任何附着部分。  
K级：同H级。  
h) H级：不是起源于边缘上的任何裂纹。  
K级：同H级。  
i) H级：贯穿基板的孔洞(有设计要求的除外)。K级：同H级。
