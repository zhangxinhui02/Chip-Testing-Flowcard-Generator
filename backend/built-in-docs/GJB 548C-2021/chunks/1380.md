# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.1 目的

---

本附录的目的旨在解决以下两个问题：第一，方法2010的目检判据对复杂晶圆制造工艺不适用，因为在多数情况下，即使放大200倍也难以观察到在这种工艺技术条件下产生的缺陷。第二，目前尚没有一种对复杂晶圆生产工艺带来的缺陷进行控制的通用方法。A.3叙述了在什么情况下实施本程序。本附录在保留和扩展方法2010目检判据的同时，实施了一些用于控制和消除晶圆制造过程中产生缺陷的新方法。  
本附录实质包含了这样一种概念，即承制方有责任确定和用文件规定(并经鉴定机构认可)减少缺陷和控制缺陷的方法，这种方法对于承制方及鉴定机构都可接受，见A.5的规定。该部分包括了晶圆在制造过程中产生的缺陷对可靠性影响的认知。承制和鉴定机构应进行良好沟通，以便制定双方满意的缺陷控制程序。本附录慎重地使用了一些非专用的度量标准，如缺陷尺寸、缺陷密度、相关性和风险性等，以便用于不同的工艺技术、不同的制造控制方法和持续改进。本附录规定的程序有这种目的，即通过承制和鉴定机构之间的协商，使度量标准及其量值更加明确。  
本附录A.6介绍了缺陷表征方法。本章的关键部分是如何获知工艺缺陷对最终产品可靠性的影响。为达到这一目的可通过多种方法，包括做实验、查阅有关文献和一些半导体经典数据。关于缺陷表征的深度和广度都将由承制方及鉴定机构来确定。  
在本附录的很多章节中都讨论了缺陷验证的概念。并且给出了尽可能多验证缺陷的方法，以满足使用的需要。  
正如A.11所述，缺陷表征结果以及监测和控制缺陷水平的方法应以文件的形式加以规定。所采用的任何一种(在线或离线)筛选的有效性也应形成文件。对以验证和记录的最后要求需由承制方和鉴定机构商定。鉴定机构会关心承制方，对缺陷产生、控制、减少、预防以及缺陷对产品可靠性的影响等方面知识和水平的掌握情况。  
本附录是基于下述假设：承制方在晶圆生产过程中努力采取措施不断改进缺陷水平(即降低缺陷等级)。作为该假设的一部分，承制方按A.7所述，采用检验手段来获得减少缺陷的有关信息。虽然这些方法可能是承制方综合缺陷控制系统的一部分，但产品的质量不是靠检验出来的。相反，其目的是提供一种缺陷预防、控制和减少在晶圆加工过程中产生缺陷的有效方法。理想情况是，承制方不断努力改进自己的缺陷控制系统。  
A.8、A.9、A.10论述了偏离控制、成品率分析和防止失效的方法。  
本附录使用大量的应用实例说明某些关键点和方法。这些应用实例更进一步说明了在给定的条件下，如何执行本附录中的一些条款。给出的实例不作为规定的条款。
