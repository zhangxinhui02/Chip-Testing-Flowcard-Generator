# 中华人民共和国国家军用标准
## 方法1023.1 线性微电路的剂量率响应和翻转阈值
### 7 说明

---

有关的订购文件应规定以下内容：a) 被试器件型号和数量。b) 追溯性(器件编号、晶圆/批编号等)要求和数据报告及提交的要求。c) 被试器件在试验中(包括结构框图)的电气结构图。d) 试验条件顺序和逻辑测试图形。e) 需要监测和记录的输出。f) 剂量率水平和脉冲宽度。g) 翻转和恢复判据，恢复的输出稳态值和(或)电源电流。包括采样波形(若需要)。h) 翻转阈值和失效水平(适用时)。i) 试验后，验证存储图形所需的功能测试，以及在辐射脉冲和功能测试开始之间的最大时间间隔。j) 对每个器件型号的总电离剂量极限和烧毁水平。k) 若适用，电源串联的最大限流电阻，以及在测试线路(见4.3.6)中允许的电阻。l) 适用时，辐射前后A组电测试要求。m) 测试仪器要求(若与上述规定不同)。n) 表征试验和分析要求。
