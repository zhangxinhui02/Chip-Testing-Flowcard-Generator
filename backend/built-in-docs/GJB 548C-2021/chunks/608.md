# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### B.2 圆柱形引线

---

圆柱形引线的缺陷测量或引线全部表面区域缺陷面积百分比估算比扁平矩形引线表面更为困难。例如，观察圆柱形引线的圆柱表面，一个测量直径为圆形的缺陷变成平面时显示是椭圆形，并且宽度比引线横向可见表面更窄。  
为帮助可焊性试验检验员估算可焊性试验后引线表面百分比，图B.3是不同直径引线 \(5\%\) 允许针孔面积评价辅助图，当检验直径 \(0.5\mathrm{mm}\) 、长 \(25.4\mathrm{mm}\) 的引线表面焊料涂覆，10个直径等同于引线直径的缺陷的面积等于全部引线表面积的 \(5\%\) ，图B.1中也列出了缺陷直径为1/2引线直径及1/4引线直径情况下的最大缺陷允许数目，这些不同尺度缺陷的组合也能容易地计算。
