# 中华人民共和国国家军用标准
## 方法1023.1 线性微电路的剂量率响应和翻转阈值
### 5.4 表征试验

---

应进行表征试验或数据获取， 以决定器件在不同剂量率下的特性， 并且规定生产试验的要求(适用时)。下述是表征试验信息获得的示例:a) 在剂量率和脉冲宽度作用下的参数特性；  
b) 不同辐射剂量率和脉冲宽度下的翻转阈值；  
c) 确定敏感性线路条件；  
d) 器件最敏感的线路识别， 以及被监测的合适输出端；  
e) 温度对翻转或失效的影响；  
f) 规定在详细规范中的翻转、恢复时间和失效判据；  
g) 剂量率试验后A组电参数的退化；  
h) 最坏情况电源电压；  
i) 最大浪涌电流和时间间隔， 以及光电流烧毁水平。
