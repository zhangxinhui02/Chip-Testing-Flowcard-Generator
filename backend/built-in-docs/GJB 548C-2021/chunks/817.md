# 中华人民共和国国家军用标准
## 方法2011.2 键合强度(破坏性键合拉力)
### A.4.5 键合剪切设备

---

A.4.5 键合剪切设备键合剪切设备必须能保证剪切刀位置精确(基体之上的误差在 \(\pm 2.54\mu \mathrm{m}\) 之内)，应高于键合面最高部位的规定距离 \((h)\) 以保证剪切刀不接触到芯片表面，但应小于从键合面的最高部位至球形或楔形键合点的中心线 \((C_{\mathrm{L}})\) 之间的距离(见图A.7)。
