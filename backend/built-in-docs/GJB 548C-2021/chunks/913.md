# 中华人民共和国国家军用标准
## 方法2017.2 内部目检(混合电路)
### 4.2.2 有源和无源元件

---

所有集成电路应按方法2010进行检查。  
H级按B级单片，K级按S级单片，具体条款如下：  
a) 方法2010中4.2.2 金属化层缺陷；b) 方法2010中4.2.3 扩散和钝化层缺陷；c) 方法2010中4.2.4 划片和芯片缺陷；d) 方法2010中4.2.5 玻璃钝化层缺陷；e) 方法2010中4.2.6 介质隔离；f) 方法2010中4.2.7 膜电阻器；g) 方法2010中4.2.8 激光修调的膜电阻器；h) 方法2010中4.3.6 多余物。  
所有晶体管和半导体二极管应按方法2010或GJB128B- 2021中的适用方法检查(见表1)。无源元件应按方法2032检查。
