# 中华人民共和国国家军用标准
## 方法2010.2 内部目检(单片电路)
### 4.3.2.6 单片器件的重新键合

---

单片器件的重新键合受到下列条件限制。呈现下列情况的器件不得接收：a) 条件A(S级)和条件B(B级):在暴露的钝化层上或有明显起皮的金属上面的重新键合。在任一设计键合区上试图进行一次以上的重新键合。重新键合触及因金属剥落而暴露氧化层的区域。  
b) 条件A(S级):键完全或部分位于另一键、键尾或引线残留部分上。  
条件B(B级):键完全或部分位于另一个键、键尾或引线残留部分上或与其相交，交叠宽度大于 \(25\%\) 。  
c) 条件A(S级)和条件B(B级):在器件中重新键合超过了该电路全部键数目的 \(10\%\) 。  
注：例如，对于28引线键合封装，有56个键合点，引线一端的键合计为一次键合；替换一根两端已键合的引线计为两次重新键合。仅对条件B,在第一次不成功的键合后，用于清除键的键剥离可不计为重新键合，只要能证实确实为键剥离。  
d) 条件A(S级)和条件B(B级):  
引线的短缺或多余的引线。
