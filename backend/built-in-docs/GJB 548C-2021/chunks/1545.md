# 附录B (方法5010) (规范性附录) 晶圆批验收
### B.3.5 批验收

---

应按表B.4进行晶圆批验收。  
|  |  |
| --- | --- |
| PM 类型 | 在PM 规范限内的数量 （若5 个中少于3 个合格） |
| 插入式 | 拒收 |
| 位于划片区 | 拒收 |
| 每一芯片中均有 | 拒收 |  
|  |  |  |
| --- | --- | --- |
| 要求 | 条件 | 验收 |
| 生产线认证 | 相关规定 | 按规范控制 |
| 批随工单检查点 | 相关规定 | 按批验收规范100 ％进行 |
| PM 测试数据 | 每片晶圆 | 批中75 ％的晶圆应通过PM 评估，否则拒收 |
| 目检 | 每片晶圆 | 逐个晶圆验收 |
| 扫描电镜检查 | 方法2018 | 按方法2018 的判据 |
