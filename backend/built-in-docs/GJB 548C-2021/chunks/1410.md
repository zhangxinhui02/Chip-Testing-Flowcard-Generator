# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.7 检查和试验程序

---

A.7.1 缺陷的控制和减少依靠检查和试验程序、工艺和产品监控以及筛选。应在晶圆加工的整个工艺流程(在线和/或离线)中使用检查和试验程序。希望采用切实有效的检验和测试程序来避免在交付的产品中出现致命性缺陷。A.14是关于检验和测试程序的实例。  
A.7.2 检查和试验程序应能形成一套完整的控制和减少缺陷的方法。该方法应考虑到以下判据(适用时)。  
A.7.2.1 供货方应确定在适当的点执行检查和试验程序，以监控致命缺陷和关键性缺陷(见A.6)。  
A.7.2.2 检查和试验程序应根据对缺陷产生的根源及其出现规律的认识(例如：随机出现，一个晶圆中不同芯片之间的差异，同一批中不同晶圆间的差异，批次间的差异，加工日期间的差异等)来确定抽样方案。抽样方案应符合统计规律(分布形式和 \(\alpha\) 、 \(\beta\) 风险)。被抽样的母体应是同一分布的母体。"同一分布"要求的实例如下:除非另有说明，由于分批或发生变化而返工的批次，均不能认为是同一母体。因此，对于非同一分布(返工)的产品，应单独抽样。同样，如果在同一工序，采用多台工艺设备(例如对同一晶圆批采用多台晶圆步进光刻机),必须证明这些加工设备对产生致命和关键性缺陷在统计上的一致性，才能将该晶圆批视为同一分布的母体。  
A.7.2.3 检查和试验应(但不限于)考虑最坏情况位置(如A.6所示)。实例:a) 对低压化学淀积(LPCVD),缺陷表征可能确定在炉管末端或其附近的晶圆上始终存在较多的颗粒，则对LPCVD抽样应包括在此位置的样本； b) 在表征金属桥接时，芯片上的一特定位置可能始终比其他位置更易发生桥接，抽样检验应包括对这一位置的样品。  
A.7.2.4 检查和试验程序应利用"追溯"方法，追溯检验可以检查当前工序和一个或多个之前的工序。该方法能够检验/测试被后续加工(放大或加强的)延伸的缺陷和/或被后续加工所缓饰的缺陷。这种技术为检验和测试前面膜层中存在的致命性和关键性缺陷提供了额外的机会。实例:a) 检验场氧化层，有可能追溯到前几层的图形生成工序；  
b) 在后续的低温氧化物(LTO)沉积(缺陷尺寸延伸)以后，缺陷更加明显，因此对LTO检验可对以前产生缺陷的工序进行有效的追溯。  
A.7.2.5 检查和试验程序应规定行动极限和适当的数据记录。数据记录应考虑到对晶圆、各晶圆批次或产品的处置和纠正措施等要求(如：数据可根据设备编号、加工工具、晶圆批次、操作者等来区分)。这些数据类型和行动极限来自于按A.6所确定的缺陷表征结果，并且应该考虑有关的缺陷属性(即：尺寸、颜色、重量、成分、密度等)。行动极限应包括按A.6所确定的安全余量。  
A.7.2.6 依据缺陷表征(见A.6),应监测非关键性缺陷，除非证明该非关键性缺陷，不管其缺陷程度或缺陷密度如何，均对最终产品无任何影响，理由是:  
a) 非关键性缺陷可能影响对致命缺陷和关键性缺陷的检查(如：颜色的改变可能影响对致命缺陷或关键性缺陷的目标);  
b) 由于缺陷密度的增加，使非关键性缺陷转化成了关键性缺陷(如：由于非关键性缺陷的增加，形成了一个缺陷链，从而形成了关键性缺陷);  
c) 非关键性缺陷和关键性缺陷发生的情况不同于常规情况，表明工艺有了变化，这时应引起重视并作出解释。  
A.7.3 任何在线或离线进行的筛选，当用于对致命性和关键性缺陷进行替代或补充的检验和测试时，都应形成文件并予以实施。筛选的母体(如：晶圆数、芯片数、晶圆分批、晶圆批等)应确定且形成文件。这些程序只需包括那些已经被证明(按A.6的要求)有效的筛选项目。筛选结果的记录(接收/拒收数据)应保存。  
A.7.4 分析工具以及生产、工艺可靠性和设备监测应有足够的能力确定A.6定义的缺陷属性，包括关键性缺陷密度的变化(例如：对给定掩蔽层，如果缺陷表征表明 \(0.1\mu \mathrm{m}\) 的颗粒是关键性缺陷，检验程序应能够检测和量化这种尺寸和更大尺寸的颗粒)。分析工具包括但不限于A.15中所列。
