# 中华人民共和国国家军用标准
## 方法1018.2 内部气体成分分析
### 3.2 预处理

---

所有的器件都要在试验前进行 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 下 \(16\mathrm{h}\sim 24\mathrm{h}\) 的预烘烤。烘箱应有方法显示烘烤期间是否发生断电及温度低于 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 的时间。如果器件温度低于 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 超过 \(1\mathrm{h}\) ，则必须至少再烘烤 \(12\mathrm{h}\) 。
