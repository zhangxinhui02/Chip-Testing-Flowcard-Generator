# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2019.3 芯片剪切强度
### 3 设备和工具

---

本试验所需设备是一台能施加负载的仪器，要求其准确度达到满刻度的 \(\pm 5\%\) 或 \(0.5\mathrm{N}(50\mathrm{gf})\) (取其大者);一台用来施加本试验所需力的带有杠杆臂的圆形测力计或线性运动加力仪。试验设备应具有以下能力:  
a) 被试件接触工具，能把力均匀地加到被试件的一条棱边(见图1)。可使用合适的辅助器材(如光滑的爪状物、线带等),以确保被试件接触工具能将应力均匀地施加到被试件的一条棱边。  
b) 保证被试件接触工具与底座或基板上安放芯片的平面垂直。  
c) 被试件接触工具与基板/底座夹具具有相对旋转能力，这有利于与被试件边沿线接触，即对被试件加力的工具应从一端到另一端接触被试件的整个边沿(见图2)。  
d) 一台放大倍数至少为10倍的双筒显微镜，其照明应有利于在试验过程中对被试件与接触工具的界面进行观察。
