# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 5.3.2.3 步骤

---

具体步骤如下：a) 在浸焊料前，应从熔融焊料表面撤去焊剂浮渣及焦渣；b) 已涂焊剂样品浸入熔融焊料深度最少为 \(0.10\mathrm{mm}\) (见图4)，浸入和提出速度为 \(25\mathrm{mm / s}\pm 6\mathrm{mm / s}\) ，停留 \(5_{-0.5}^{+0}\mathrm{s}\) ，大体积器件可能需要在熔融焊料中停留更长时间(大体积器件指有厚热沉封装的器件，此类器件的停留时间应按有关规定执行)；c) 提出后，在维持样品试验条件的同时，焊料允许在空气中冷却固化；d) 检验前，应按4.1.3要求，去除所有引出端上的全部可见残余焊剂。
