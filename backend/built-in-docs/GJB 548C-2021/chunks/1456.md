# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5006 极限试验
### 3.2.3 持续热冲击

---

本试验的目的是测定器件抗热疲劳作用的能力。应按方法1011，对器件至少进行100次热冲击循环试验。应按表2规定顺序进行热冲击循环试验。  
|  |  |  |
| --- | --- | --- |
| 步进 | 循环次数 | 试验条件 |
| 1 | 15 | A |
| 2 | 15 | B |
| 3 | 70 | C |  
应分别在第15、30、40、70和100次热冲击循环试验后测量电参数，并分别记录失效数。  
若适用的订购文件中有规定时，可以采用方法1010温度循环代替持续热冲击试验。应按表3规定顺序进行温度循环试验。  
|  |  |  |
| --- | --- | --- |
| 步进 | 循环次数 | 试验条件 |
| 1 | 20 | B |
| 2 | 20 | C |
| 3 | 20 | D |  
应在完成每一次步进规定的循环次数之后测量电参数，并记录相应的失效数。
