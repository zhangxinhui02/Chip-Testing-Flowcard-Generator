# 中华人民共和国国家军用标准
## 方法2009.2 外部目检
### 4 说明

---

有关的订购文件应规定以下内容:a) 标志、引线(引出端)或引出端识别要求；  
b) 材料、设计、结构和工艺质量的其他详细要求。
