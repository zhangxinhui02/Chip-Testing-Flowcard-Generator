# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 6.3.8.2 J形引线校直

---

具体操作如下：a) 把封装安装在类似图3的夹具上；b) 使用合适的钳子将引线肩部和过渡区拉直(见图4)；c) 将封装引线的J形弯曲段小心拉直，使其与引线最宽方向近似垂直；d) 应小心操作以免引线受损，如裂缝、扭结或深划痕。
