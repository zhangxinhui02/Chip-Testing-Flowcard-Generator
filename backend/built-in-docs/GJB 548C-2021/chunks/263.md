# 中华人民共和国国家军用标准
## 方法1015.1 老炼
### 3.4.1 通则

---

当有规定时，或承制方自愿时，应在施加老炼试验条件前进行老炼前测试。老炼后测试应在器件移除规定老炼试验条件后(即撤除加温或去除偏置)的96h内完成。测试应包括全部 \(25^{\circ}\mathrm{C}\) 时的直流参数测量(方法5005或GJB2438中A1分组)和作为中间(老炼后)电测试的一部分而规定了变化量极限的全部参数。当采用时，应根据这些测试确定参数的变化量极限是否超过了允许的范围。无论对于常规老炼试验或加速老炼试验，如果不能在96h内完成这些测试，那么在作老炼后的测试条件之前器件应按原先的老炼条件(见3.2)和原来采用的温度至少再作表1中所规定的时间的老炼。
