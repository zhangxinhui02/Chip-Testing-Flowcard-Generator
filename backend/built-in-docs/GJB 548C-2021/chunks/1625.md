# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.17.2 吸附牢固性(长期)

---

无论是新制备的样品或者是经上述短期试验的样品，应首先在 \(150^{\circ}\mathrm{C}\) 条件下贮存 \(1000\mathrm{h}\) ，再对样品按方法2002试验条件B在Y2方向进行机械冲击，然后采用与5.8.17.1相同的要求对样品进行PIND试验。  
PIND试验之后，去除封盖，按下述要求进行目检：a) 检查粒子是否与吸附剂材料分离或者已落入封装腔体内；b) 检查吸附剂覆盖状态是否已经散开或变形；c) 检查封装内部是否存在起皮和/或吸附剂与封装分离的现象。
