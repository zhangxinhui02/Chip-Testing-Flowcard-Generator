# 中华人民共和国国家军用标准
## 方法1023.1 线性微电路的剂量率响应和翻转阈值
### 模拟-数字转换器试验示例

---

B.1本附录给出了一个例子，用来说明如何规定模拟-数字转换器(ADC)的试验条件。由于试验条件同时取决于器件类型和特定的使用情况，因此不能认为本例适用于任何情况，它只为了说明如何使用本试验方法。  
B.2方法1023试验规范:a)ADC型 \(\scriptstyle (\mathrm{n} = \# \mathrm{bit} = 12)\) ，40引出端陶瓷双列封装。  
b)试验电路按图B.1规定，试验装置图见图B.2。存储RAM必须以超过器件时钟频率的速度写入，并且对控制器提供一个接口，能够贮存来自辐射源的触发脉冲。如果使用ADC的数据准备线，由于可能出现翻转，也必须分别监测。  
c)至少应试验3个输入电压。调节输入偏置，使中心输出码在:  
1) \(2^{n} / 2\)  
2) 满量程减去 \(10\% (2^{n} - 0.1 \times 2^{n})\)  
3) 零加上 \(10\% (0 + 0.1 \times 2^{n})\)  
d) 至少要在10MHZ和1MHZ( \(F_{\mathrm{max}}\) 和 \(0.1 \times F_{\mathrm{max}}\) )下进行测试。  
e) 翻转判据：翻转阈值的确定应通过统计分析将辐射前ADC输出码与在辐射后立即得到的数据进行比较来得到，恢复时间( \(20\%\) 以内)也应通过比较辐射前后的数据来确定。  
f) 脉冲宽度: \(20\mathrm{ns}\) (半高宽)。  
g) 在 \(10^{2}\mathrm{Gy}(\mathrm{Si}) / \mathrm{s} \sim 10^{7}\mathrm{Gy}(\mathrm{Si}) / \mathrm{s}\) 的剂量率范围进行试验以确定翻转阈值。  
h) 总电离剂量应不超过 \(500\mathrm{Gy}(\mathrm{Si})\) 。  
i) 在环境温度 \(25^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}\) 试验。  
j) 在翻转试验完成之后，试验达到设备最大剂量率以确定器件是否会烧毁。本试验是破坏性试验。
