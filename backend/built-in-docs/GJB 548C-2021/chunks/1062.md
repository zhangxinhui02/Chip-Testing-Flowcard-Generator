# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2023.3 非破坏性键合拉力
### 3.3.2.6 键合前 \(100\%\) 自检程序

---

应在有关文件中规定键合前对芯片键合区和封装引线柱进行 \(100\%\) 自检的程序。自检应在放大100倍 \(\sim 200\) 倍、100级洁净环境中进行。允许对可能导致拒收的沾污进行清洗处理。器件不得出现下述情况：  
a）设计的键合区有玻璃钝化层；b）芯片键合区或外壳引线柱有化学物质薄膜、光刻胶或液体污染物；c）在芯片键合区或外壳引线柱的有关键合区域上有直径大于 \(6.4\mu \mathrm{m}\) 的微粒或多余物；d）相应的芯片键合区和外壳引线柱不完全满足方法2010条件A或方法2017的所有相应要求。注：在进行了 \(100\%\) 清洁预处理，且随机抽取5个器件对其所有芯片键合区和引线柱进行自检合格后，键合前 \(100\%\) 自检可省略。如果以往的数据证明清洁预处理是不必要的，则清洁预处理可以省略。在样品自检过程中，不允许作任何清洁处理。 \(100\%\) 清洁预处理和5(0)抽样方案的目检最多可重复2次。如果第二次清洁预处理后出现样品拒收，则应按3.3.2.6对整批进行键合前 \(100\%\) 自检。应对该批次中和样品中出现拒收的原因进行分析，并采取必要的纠正措施。同时，对后续产品要进行键合前 \(100\%\) 自检。一旦确定了纠正措施的有效性，键合前 \(100\%\) 自检可取消。
