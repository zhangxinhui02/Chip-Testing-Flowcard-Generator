# 中华人民共和国国家军用标准
## 方法2018.2 扫描电子显微镜(SEM)检查
### 4.6.3.1 通则

---

如果任何几种缺陷(见图 23)的共同作用或由于金属化层的变薄使主载流方向横截面上金属条的横截面的面积减少了 \(50\%\) 以上，该钝化层台阶上的金属化层不应接收。对于钝化层台阶上的金属化层，当其结构是圆形或多边形通孔或接触结构，并有足够的环绕金属(大于金属线条宽度的 \(10\%\) ),允许电流流过通孔或接触孔的所有边时， 在主载流方向上只要有 \(20\%\) 以上的金属层覆盖就是允许的 (包括阻挡层金属， 见图 24)。金属化层必须满足 GJB 597B- 2012 中的电流密度要求。在看不到能明显表示有效覆盖的棱边、平缓过渡或斜坡的情况下， 应通过横截面来验证金属覆盖。
