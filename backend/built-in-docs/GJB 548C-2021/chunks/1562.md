# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.3.3 材料放气

---

5.3.3 材料放气应按5.8.7确定固化聚合物材料的放气情况。接收要求为三个封装(0失效)或五个封装(1失效)。失效判据为按5.8.7.2确定的水汽含量不大于 \(0.5\% (5000\mathrm{ppm})\) 。对于其他气体，若其含量不小于 \(0.01\% (100\mathrm{ppm})\) ，应以百分数(V/V或ppm)为单位出具报告。按5.8.7.3要求得到的数据也应以同样的方式报告，但是只是作为一种信息，而不作为确定是否接收的判据。应按5.8.7确定已固化吸附剂材料的放气。采用吸附剂的封装器件，在 \(150^{\circ}\mathrm{C}\) 下烘焙24h后，其水汽含量不应超过 \(0.2\% (2000\mathrm{ppm})\) ，在 \(150^{\circ}\mathrm{C}\) 下烘焙 \(1000\mathrm{h}\) 后其水汽含量不应超过 \(0.3\% (3000\mathrm{ppm})\) 。
