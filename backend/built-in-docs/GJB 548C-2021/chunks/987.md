# 中华人民共和国国家军用标准
## 方法2018.2 扫描电子显微镜(SEM)检查
### 4.7 样品文件

---

对每批样品中受检金属化层中的每一层应至少拍摄三张照片，并在检查后至少保存五年。其中两张照片反映最坏的钝化层台阶情况，第三张照片反映一般金属层中的最坏情况。如果某张照片上在所观察的范围出现了一种明显缺陷，就应另外再拍摄一张照片来确定该明显缺陷的程度(见表2)。  
注：其他形式的图像存贮方法也可采用(例如：光盘),但应首先得到鉴定机构的认可。  
每张照片都应附有下述信息:a) SEM照片的日期；  
b) 器件或电路识别(型号或器件号);  
c) 照片范围；  
d) 电子束加速电压；  
e) 放大倍数；  
f) 承制方；  
g) 生产批号；  
h) 计算/测试的台阶覆盖百分比的记录；  
i) SEM操作员或检查员的识别号；  
j) 观察角度。
