# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2020.2 粒子碰撞噪声检测
### 3.4.1 试验条件A

---

试验条件A的正弦峰值加速度为 \(20\mathrm{g}\) ，其振动装置试验频率 \((f)\) 由封装最大自由活动腔高 \((h)\) 决定，按公式(1)计算：式中：  
\[f = 99.706 / \\sqrt{h} \\quad (1)\]  
\(h\) ——最大自由活动腔高， \(\mathrm{mm}\)  
\(f\) ——振动装置的试验频率， \(\mathrm{Hz}\)  
公式(1)的使用要限于 \(40\mathrm{Hz} \sim 150\mathrm{Hz}\) 频率范围内，对于计算出的频率在该范围以外的封装高度不应使用本公式，除非该频率能够获得相应的认可。
