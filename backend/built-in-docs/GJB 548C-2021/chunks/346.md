# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.9 偏置和负载条件

---

辐射或加速退火试验期间，器件偏置条件应在试验方案规定的 \(\pm 10\%\) 以内。应按如下原则为器件施加偏置：给器件施加的偏置、负载、内部辐射图形应使器件产生最大的辐射诱生损伤，或产生在预定应用环境下的最坏情况损伤(如果已知)。在整个分步方式的辐射试验和退火过程中，偏置、负载、内部辐射图形应保持一致。如果偏置、负载、内部辐射图形等条件的任何一个不是处于最坏情况条件，在试验方案和试验报告中应给出条件选择的合理理由。最大电压通常是最坏情况条件，然而对于一些双极线性器件参数(例如输入偏置电流或最大输出负载电流),在0V偏置时的退化更明显。应按试验方案要求给每一个器件施加规定的偏置条件。应在辐射即将开始前和辐射刚刚结束后检查偏置情况，并注意在选择和施加偏置时使结温的升高减至最小。
