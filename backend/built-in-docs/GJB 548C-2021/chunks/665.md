# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 8.3.2 具有矩形截面引线或引出端器件的试验程序

---

8.3.2 具有矩形截面引线或引出端器件的试验程序对器件本体作刚性固定，除另有规定外，在离器件本体 \(3.05\mathrm{mm}\pm 0.76\mathrm{mm}\) 处，或当引线不足 \(3.05\mathrm{mm}\) 长时则在引线末端处，对引线(引出端)施加 \(14.21\mathrm{N}\cdot \mathrm{mm}\pm 1.42\mathrm{N}\cdot \mathrm{mm}\) 的扭矩。以引线的轴线为轴，顺时针方向和逆时针方向各施加一次扭矩。如引线在靠近器件本体处作了成形处理，则在离成形部位 \(3.05\mathrm{mm}\pm 0.76\mathrm{mm}\) 处加扭矩。对于施加的扭矩小于规定值就扭转明显的器件引线，应继续扭转直至扭转角度达到 \(30^{\circ}\pm 10^{\circ}\) ，或达到规定的扭矩，取两者中首先出现的条件。然后应把引线恢复到原来位置。
