# 中华人民共和国国家军用标准
## 方法1015.1 老炼
### 3.4.2 常规老炼后的冷却

---

去除偏置之前，所有器件应冷却到与器件处于室温下功耗稳定状态时的温度相差不超过 \(10^{\circ}\mathrm{C}\) 。为了把器件转移到与老炼试验箱不在一处的冷却位置而中断偏置不超过1min，不应看作去除了偏置(在冷却位置处加的偏置应与老炼时的偏置相同)。对于线性或MOS(CMOS、NMOS、PMOS等)电路以外的器件(除另有规定外)，如果在撤除试验条件后的30min内被试器件的外壳温度能够降到 \(35^{\circ}\mathrm{C}\) 以下，并且能够在去除偏置后的5min内将器件从老炼试验箱内取出的话，也可以在冷却期间去除偏置。在重新加热器件之前应完成全部 \(25^{\circ}\mathrm{C}\) 的直流参数或其他参数测量(见3.4.1)。
