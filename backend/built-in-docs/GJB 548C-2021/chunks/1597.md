# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.7.1 概述

---

应采用镀金或镀镍可伐封装或陶瓷封装制备10个样品。对介质材料，制备样品时可采用镀铝的硅材料作为基板。允许采用无引线封装，以减少由封装结构引起的水汽影响。材料的固化应采用最短固化时间再加上配套文件中规定的密封前最短的烘焙时间（见5.3.1)。密封前烘焙后，进行气密性封装，并进行检漏试验。只有那些通过方法1014规定的细检漏和粗检漏试验要求的封装才能交付作水汽含量分析。若密封试验后剩下的样品不足10个，应采用与上述相同的封装和检漏方法，选取满足气密性要求的封装样品替代失效的封装样品。
