# 中华人民共和国国家军用标准
## 方法1004.1 耐湿
### 3.8 最终测试

---

在最后一次循环的第6步之后(或者在第10次循环期间完成3.5的子循环，则在第7步之后)，器件应在室温环境条件下放置24h，然后按方法1003中试验条件A进行绝缘电阻测量，或在 \(25^{\circ}C\) 下进行规定的终点电测试。绝缘电阻测量或 \(25^{\circ}C\) 下终点电测试应在器件移出试验箱后的48h内完成。电测试也可以在放置的24h期间进行，但应记录测试中出现的所有失效器件，且禁止为了得到可接收的结果，在24h之后对失效器件重新进行测试。在24h放置期间不应进行其他试验(如密封试验)。进行绝缘电阻测量时，测得的电阻值不得小于 \(10\mathrm{M}\Omega\) ，并且应记录测量结果，其数据作为终点数据的一部分提交。如果把封装外壳设计成与芯片基板电连接，则不必进行绝缘电阻测量，而应在器件移出试验箱后的48h内完成规定的 \(25^{\circ}C\) 终点电测试，还应进行目检和任何其他规定的终点电参数测量(见第4章c))。
