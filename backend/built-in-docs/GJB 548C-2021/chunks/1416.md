# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.13 缺陷对可靠性影响的评价方案

---

栅氧化层预清洗操作中产生的颗粒特性表明，在这步操作过程中产生的颗粒往往很小 \((0.2\mu \mathrm{m})\) ，其密度在 \(0.02\mathrm{d} / \mathrm{cm}^2\sim 0.8\mathrm{d} / \mathrm{cm}^2\) 之间变化，其密度主要取决于对预清洗站的利用频度。随着预清洗站所处理的晶圆数目的增加，缺陷密度也随之增加。  
该步工序处理后挑选出两组晶圆：一组晶圆具有较低的缺陷密度(大约为 \(0.3\mathrm{d} / \mathrm{cm}^2\) )，另一组晶圆具有较高的缺陷密度(约为 \(0.8\mathrm{d} / \mathrm{cm}^2\) )。这些晶圆经过相同的生产线加工后，分别抽取芯片进行高电压应力试验。试验结果表明：对于中低密度缺陷而言，栅漏与工艺基线的历史数据相当；而对于高密度缺陷芯片，栅漏是历史数据的3倍，并且其失效率高得不可接受。  
根据颗粒的这一特性，在栅氧化预清洗工序进行颗粒监测，其缺陷密度上限设置为 \(0.6\mathrm{d} / \mathrm{cm}^2\) ，该设置与导致栅漏严重的颗粒密度 \((0.8\mathrm{d} / \mathrm{cm}^2)\) 间留有一定的安全余量。然而，由于监测资源有限，只能每工作班次(大约12h)监测一次，由于生产线上物料的影响有可能两次监测的之间发生清洗站里的颗粒密度超过控制极值的情况，第二个预清洗站约3个月内安装完工。这个预清洗站将提供足够的能力，以防止在栅氧化预清洗操作中出现与晶圆生产量相关的颗粒密度失控状况。在新的预清洗站投入使用之前，为了确保栅氧晶圆的质量，在出现监测结果超标时，承制方应对上次监测结果正常以后到目前这段期间加工的所有栅氧晶圆批实施高电压应力筛选试验。  
为了表明筛选方法的有效性，在预清洗站对每一批晶圆都进行颗粒监测。这种试验应该在能够生产具有不同缺陷密度的多个晶圆批的期间进行。从每个晶圆批中抽取芯片进行高电压应力筛选试验。结果表明该筛选方法对于检测缺陷密度大于 \(0.6\mathrm{d} / \mathrm{cm}^2\) 的晶圆批 \(100\%\) 有效。筛选结果还表明栅氧化预清洗的缺陷密度与栅漏电大小密切相关。在第二个预清洗站安装和验收之前，用该方法能很好地反映预清洗站的颗粒状况数据。
