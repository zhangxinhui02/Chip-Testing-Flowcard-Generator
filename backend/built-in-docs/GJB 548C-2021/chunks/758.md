# 中华人民共和国国家军用标准
## 方法2010.2 内部目检(单片电路)
### 4.2.2.7 金属化层的对准偏差

---

呈现下列情况的器件不得接收：a）条件A(S级)：被金属化层覆盖的接触窗口面积小于整个接触窗口面积的 \(75\%\) 。  
条件B(B级):被金属化层覆盖的接触窗口面积小于整个接触窗口面积的 \(50\%\) 。  
b) 条件A(S级):连续被金属化层覆盖的接触窗口周边长度小于接触窗口周长的 \(50\%\) 。  
条件B(B级):连续被金属化层覆盖的接触窗口周边长度小于接触窗口周长的 \(40\%\) 。  
c) 条件A(S级)和条件B(B级):接触窗口两个相邻边上被金属化层覆盖的长度小于该两边总长度的 \(75\%\) (仅适用于MOS结构)。  
注：若按设计要求，金属层应完全包含在接触窗口内或不需要完全覆盖整个接触窗口的周边时，只要满足设计要求， \(4.2.2.7a\) )、 \(4.2.2.7b\) 和 \(4.2.2.7c\) 要求可以不予考虑。  
d) 条件A(S级)和条件B(B级):不应该覆盖接触窗口的金属化层通路与接触窗口之间看不到明显间隔线。  
e) 条件A(S级)和条件B(B级):栅氧化层的任何暴露(即在源和漏扩散区之间的氧化层未被栅电极覆盖，仅适用于MOS结构)(见图17)。  
f) 条件A(S级)和条件B(B级):对含有扩散保护环的MOS结构，栅金属化层未与扩散保护环重合或超越保护环(见图17)。
