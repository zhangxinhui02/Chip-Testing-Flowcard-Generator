# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5012 数字微电路的故障覆盖率测量
### 4.4.3 块逻辑

---

对块逻辑分割应间接测定其故障覆盖率。对给定的块逻辑分割，只有在下述情况下才应给出对该块逻辑分割测定的故障覆盖率和测试效率:a) 加在器件上的测试向量序列也是用于该块逻辑分割的确定测试算法；  
b) 从块逻辑分割中产生的临界输出值在初始输出端是可观察的。  
否则，认为该块逻辑分割的故障覆盖率是 \(0\%\) 。用这种方法测试过的每一个块逻辑分割，应在故障仿真报告中给出采用的测试算法、其成功应用的证明、及得到的故障覆盖率或测试效率。
