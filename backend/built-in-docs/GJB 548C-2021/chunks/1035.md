# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2021 玻璃钝化层的完整性
### 3.2 方法A

---

把去盖的样品完全浸没在铝腐蚀液中： \(\mathrm{H}_3\mathrm{PO}_4(85\%):\mathrm{H}_2\mathrm{O}:\mathrm{HNO}_3(70\%) = 40:19:4\) (体积比)  
该溶液温度保持为 \(50^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 。  
在腐蚀过程中应采用适于观察浸入溶液中样品的光学系统(如单筒、双筒或立体显微镜)检查器件。器件的腐蚀时间是从裸露的键合点处完全去掉铝金属层所需总时间的两倍。将经腐蚀的器件从加热溶液中取出，在蒸馏水中漂洗后用压缩空气或其他合适的气流吹干。腐蚀干燥之后至少用100倍进行最后光学检查。
