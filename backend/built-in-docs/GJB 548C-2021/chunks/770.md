# 中华人民共和国国家军用标准
## 方法2010.2 内部目检(单片电路)
### 4.3.2.2 金丝球焊键合

---

\(4.3.2.2\mathrm{a})\sim 4.3.2.2\mathrm{c})\) 同时适用于条件A(S级)和条件B(B级)。呈现下列情况的器件不得接收：  
a）芯片或外引线键合区上的金丝球焊键合，其球型键合的直径小于引线直径的2.0倍或大于5.0倍；  
![](assets/b5bc798faacea9d8325e0de0efe33c3a674c01b04d9b477dd8c518aabe9c66b9.png)  
![](assets/929deae8b575766381e1d158544d456eec913d813acc06b76b4010087ea295b8.png)  
b) 金丝球焊键合的引出线不完全在球的周界内； c) 金丝球焊键合引出线中心不完全在未被玻璃钝化层覆盖的键合区界限内。
