# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
### A.2 设计控制

---

A.2.1 设计包括器件设计和工艺开发。器件设计包括将一个器件的功能描述转换成图形生成数据库的所有步骤和支持系统。工艺开发包括材料的选择、加工和工艺条件，这些有可能极大影响金属化层的完整性。在保证金属化层完整性中应主要考虑设计工艺。  
A.2.2 承制方的设计系统应包括基于承制方工艺能力受控的和文件化的规则。这些规则应规定特征尺寸和间距要求，并应考虑在加工过程中可能发生的尺寸变化。承制方应有能力证明其规则是基于所期望的工艺改变。此外，还应形成文件化的可靠性规则，用以考虑加工材料、公差和极限，以确定每项技术的电特性。承制方还应具备一个检查规则违背的系统，以及纠正违背的系统。设计规则应考虑最大电流密度(按GJB597B- 2012的规定进行计算)，其是由最坏工作条件所决定，并考虑了接触窗口和通孔的电流密集情况。承制方应保证最坏情况的工艺条件(如对齐、金属化厚度、线宽和接触区/通孔尺寸)不会违背电流密度的规定。对于每项技术所设计的电流密度应有足够的余量，以确保在器件的规定寿命期内不会由电迁移引起失效。  
A.2.3 承制方的设计必须考虑工艺中的已知缺陷水平。承制方开发的工艺所生产的金属化层，其电特性和机械特性应和A.2.2的设计规则和技术的可靠性目标相一致。在最后加工完成后，应能明确推导出金属层中的机械应力。利用适当的设计试验结果，承制方应能证明已达到了所要求的电特性和机械特性，并说明其已了解其他工艺参数对金属化层完整性参数(在A.3中所列的最低要求)的影响。初始的工艺规范限制应这样选择：金属化完整性参数都在加工能力之内。承制方应具备更改控制系统，使那些新的或更改的工艺若不进行适当的可靠性评估不能投入生产。
