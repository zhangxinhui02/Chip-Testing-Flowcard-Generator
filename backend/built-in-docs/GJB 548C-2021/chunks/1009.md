# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2019.3 芯片剪切强度
### 4.3.1.1 环氧树脂类材料粘接

---

符合以下任一条判据的芯片均应视为失效。  
a) 达不到图4中1.0倍曲线所表示的剪切强度要求；  
b) 发生分离的力大于图4中表示的最小强度(1.0倍力)曲线所对应的力，但是小于图4中2倍力曲线所对应的力，同时有效粘接面积小于附着区面积的 \(75\%\) 。  
![](assets/70a98f29020d32e159853716b9ac2200c64c291b414782813791048b5e6cd019.png)  
注1:若附着区面积大于 \(4.13\mathrm{mm}^2\) ，被试件承受的最小作用力应为 \(24.5\mathrm{N}\) 或其倍数(见4.3)。  
注2:若附着区面积小于或等于 \(4.13\mathrm{mm}^2\) ，被试件承受的最小作用力应为 \(6.1\mathrm{N / mm}^2\) (1.0倍时)、 \(7.6\mathrm{N / mm}^2\) (1.25倍时)或 \(12.2\mathrm{N / mm}^2\) (2.0倍时)。
