# 中华人民共和国国家军用标准
## 方法2018.2 扫描电子显微镜(SEM)检查
### 4.1.1 概述

---

表1中给出晶圆抽样方案以及在晶圆样品中规定的特定芯片位置可在保证整体试验置信度的同时， 使试验样品减至最少。这些芯片代表了金属化结构中的最典型情况或是最坏位置情况。  
当芯片或已封装器件用作晶圆批接收评估， 不能满足根据表1进行的晶圆选择要求， 则应使用下列样本大小:  
a) 如果芯片/已封装器件来自于已知的同类晶圆批(能追溯到特定的晶圆或晶圆批， 并有可查证的客观证据), 则可以从中随机抽取8只器件；  
b) 如果芯片/已封装器件来自于非同类晶圆批(可追溯性是未知的， 或没有可查证的客观证据), 则必须从中随机抽取22只器件。  
芯片上作SEM评估检查的区域不应位于或靠近晶圆边缘。被检查的区域应是完全没有受到污染，以保证是在一个未被破坏的金属化区域实施规定的检查。互连金属化的接收应基于对所选芯片区域的检查，来确定单个晶圆的接收与否或一个晶圆批的接收与否。  
本方法中涉及的芯片指对一个具有完全功能的器件芯片的评价。如果鉴定机构批准，也可代之为一个特定设计的SEM测试图形的评价。这些测试图形可位于划片区、包含在每个器件芯片内或是用插入式方法位于晶圆上某几个特定芯片位置上。
