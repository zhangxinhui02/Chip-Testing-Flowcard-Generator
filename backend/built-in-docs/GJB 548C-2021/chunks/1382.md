# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.3 范围

---

A.3.1 当满足下述任一情况时，本程序可用于复杂微电路:  
a) 晶圆上任一载流导体的最小平面尺寸不大于 \(1.5\mu \mathrm{m}\) ；  
b) 传导电流的互连层(如金属、多晶硅)不少于3层，且逻辑门数超过4000;  
c) 采用的不透明材料掩蔽了设计特征。  
A.3.2 本程序可供用户审查。  
A.3.3 复杂微电路的承制方可将本程序扩展到不满足A.3.1规定的其他器件(任选的器件)。这种扩展只适用于那些同一晶圆工艺线上采用基本相同的工艺基线生产的器件(用来生产在A.3.1中规定产品的主要设备和工艺基线也用于扩展产品)。扩展产品采用的所有不同工艺要素应符合这里所述的要求。  
A.3.4 工艺改进后，本程序允许删除、修改或减少检验和筛选项目。进行更改前，加工工艺(和/或工步)必须经过充分的表征。在有要求的情况下，应向用户和鉴定机构提交支撑这些更改的数据。  
A.3.5 本程序只适用于晶圆制造过程中产生的有关缺陷。采用本程序时，承制方可以不考虑方法2010中条件A和条件B的4.2.2(后面注明的除外)、4.2.3、4.2.5、4.2.6、4.2.7和4.2.8的内容。组装导致的缺陷(如分片损伤、探针损伤、键合完整性、芯片表面划伤和多余物等)应根据试验方法2010中的4.2.2.1、4.2.2.6、4.2.4和4.3.6的规定(条件A或条件B),在较低放大倍数(30倍\\~60倍)下进行检验。  
A.3.6 除非本程序明确规定，不能将本程序作为其他有关规范的要求不执行的理由。
