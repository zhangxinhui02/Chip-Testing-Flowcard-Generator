# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
### A.4 可靠性试验

---

尽管可靠性是由设计和制造出来，而不是通过筛选成品来实现的。但从筛选和可靠性试验中可获得具有价值的信息。筛选试验如老炼等不仅能提出该批产品中的有缺陷器件，而且还能提供有关失效机理的信息，这些信息能用来改进设计、材料、工艺或电测试。与此类似，对器件的加速试验是使那些在正常工作条件下可能出现的失效机理提早出现。对这些失效机理进行分析，从而为改进提供基础。承制方所使用的加速试验包括(但不限于)电迁移试验、寿命试验、温度- 湿度偏置试验、温度循环试验。加速试验中所使用的结构对所代表的技术应具有典型性。在加速试验中所出现的失效机理，应为器件在正常工作时出现的典型失效机理。  
承制方应具备一个对金属化系统可靠性进行评估的系统。该系统应使承制方能确定器件在给定寿命时间内出现失效的概率，使用户能随时了解与给定技术相关的有关金属化系统的寿命和失效率数据。承制方系统收集的数据必须能显示它具有极大的概率满足规定的寿命和/或失效率。
