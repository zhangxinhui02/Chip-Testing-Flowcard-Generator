# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5005.3 鉴定和质量一致性检验程序
### 3.6.4 E组样本

---

根据承制方的意愿，E组样本无需经受方法5004筛选的所有试验，但应采用合格的外壳装配，并在辐射前至少要通过A1及A7分组 \(25^{\circ}C\) 的电动试。同时，应服从下述准则：  
a）E组检验的样品应经过老炼试验；b）如果过去的试验表明老炼试验对器件总剂量效应的改变可以忽略，则可不作老炼试验；c）如果老炼引起的总剂量效应可以修正，E组检验的样品也可不进行老炼，应采用辐射试验相关方认可的方式进行修正。
