# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2037 铅锡焊料成分分析- X 射线荧光检测
### 3.1 校准

---

每次检测前，应使用标准样品对检测设备进行校准。为了排除镀层厚度对检测造成影响，可将标准样品箔片放在与被测样品类似的基片上对系统进行校准。校准检测结果必须与标准样品的标称值一致，但也应考虑标称值的不确定度和实验室操作的不确定度(即如果标准样品标称含铅量为 \(3\%\) (重量百分比)，容差为 \(\pm 10\%\) ，则当校准检测结果在 \(2.7\% \sim 3.3\%\) 范围内，可认为设备正常工作)。可建立控制图对校准检测结果进行对比。
