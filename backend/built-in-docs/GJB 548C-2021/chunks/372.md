# 中华人民共和国国家军用标准
## 方法1020.2 剂量率感应锁定
### 4.1 辐射源

---

试验可采用下述两种辐射源中的一种： 闪光X射线机(FXR)或电子直线加速器(LINAC)。FXR应用于X射线模式， LINAC用于电子束模式。FXR峰值(终点)能量应大于或等于2MeV, LINAC束能量应大于或等于10MeV, 脉宽应为 \(20\mathrm{ns} \sim 100\mathrm{ns}\) (或按订购文件中的规定)。在器件接受辐射的空间中， 用剂量测试系统测得的辐射场均匀性应为 \(\pm 15\%\) 。每次辐射暴露的剂量应在试验方案或程序中规定(关于生产试验的要求见5.5.1)。
