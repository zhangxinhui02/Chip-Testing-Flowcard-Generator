# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.7.1.1 标准漏孔

---

本程序的设备类似于条件 \(\mathrm{A}_1\) 中质谱检漏仪，质谱检漏仪最适宜的标准漏孔为 \(5\times 10^{- 5}(\mathrm{Pa}\cdot \mathrm{cm}^3) / \mathrm{s}\) 由于累积氨的测量值与测量时间是线性函数，当信噪比大于100时，测量范围可达到 \(10^{- 7}(\mathrm{Pa}\cdot \mathrm{cm}^3) / \mathrm{s}\) 当需要检测更小漏率时，需要一个 \(5\times 10^{- 7}(\mathrm{Pa}\cdot \mathrm{cm}^3) / \mathrm{s}\) 的标准漏孔。当需要检测较大漏率时，需要一个 \(1\times 10^{- 3}(\mathrm{Pa}\cdot \mathrm{cm}^3) / \mathrm{s}\) 的标准漏孔。  
用一个经校准的扩散型标准漏孔，每个工作班次校准检漏仪的指示器至少一次。
