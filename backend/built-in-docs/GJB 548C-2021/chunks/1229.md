# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2032.1 无源元件的目检
### 4.3.4 厚膜电阻缺陷

---

呈现下列情况的元件不得接收:a) H级：电阻末端的空洞使电阻宽度减至初始电阻宽度的 \(50\%\) 以下(见图46)。K级：同H级。  
![](assets/38f4ed28e68c4849653c369422bd658ea1220b8de7c1ae0b88d0e1a734d01ebb.png)  
b) H级：电阻末端出现颈状缩小，使电阻宽度减至初始电阻宽度的 \(50\%\) 以下(见图47)。K级：同H级。  
![](assets/98c7d6560feb534329a946f64427fc52788d562378bade1686c64a5209719cfc.png)  
c) H级：电阻膜的任何剥落、脱皮或起泡。  
K级：同H级。  
d) H级：电阻上长度大于 \(25\mu \mathrm{m}\) 的裂纹。  
注：电阻材料在烧制过程中产生的缝隙等缺陷，只要未暴露下层材料则不认为是裂纹。  
K级：同H级。  
e) H级：用叠加印刷或其他任何方法进行电阻修补的痕迹。  
K级：同H级。  
f) H级：两个电阻之间的间隔小于原始间隔的 \(50\%\) 。  
K级：同H级。  
g) H级：电阻和导体组合键的间隔小于原始间隔的 \(50\%\) 。  
K级：同H级。  
h) H级：电阻宽度的增量超过初始设计宽度的 \(25\%\) 。  
K级：同H级。  
i) H级：电阻与基板边缘的距离小于 \(25\mu \mathrm{m}\) 。  
K级：同H级。  
j) H级：由空洞、划伤、切割或其他的组合引起电阻宽度减小至最小电阻宽度的 \(50\%\) (见图48)。K级：同H级。  
![](assets/53acefb553b8cee5ad48c6f38158e08d73ce5122489942a5b1872998d64a281d.png)  
k) H级：金属化和电阻之间接触重叠部分的宽度尺寸"y"小于初始电阻宽度的 \(50\%\) (见图49a))。K级：小于 \(75\%\) (见图49b))。  
![](assets/8c24c8b0c4f7a67aa61d0f0df51882f0fe0c61700df3f5c4a454defcf7da550f.png)  
1) H级：金属化和电阻之间接触重叠部分的长度尺寸"x"小于 \(76\mu \mathrm{m}\) (见图50)。K级：同H级。  
![](assets/1ef943602a2d992c41993fcf46138641328fb8a70a79209081b0e8bd777714b6.png)  
m) H级：由于空洞或对准不良，使玻璃钝化层所覆盖的电阻区域小于 \(90\%\) (见图49a))。K级：同H级。  
n) H级：电阻上玻璃钝化层出现龟裂。  
K级：同H级。  
0) H级：玻璃钝化层中出现划伤、剥落或脱皮，暴露了一部分电阻。  
K级：同H级。
