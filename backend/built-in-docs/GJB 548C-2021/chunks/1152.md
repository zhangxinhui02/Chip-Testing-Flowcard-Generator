# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2032.1 无源元件的目检
### 2.7 受控环境 controlled environment

---

湿度和空气洁净度受控。除允许的最大相对湿度不超过 \(65\%\) 外，100级受控环境每立方米空气中微粒(直径大于等于 \(0.5\mu \mathrm{m}\) )应小于3520个，1000级受控环境每立方米空气中微粒(直径大于等于 \(0.5\mu \mathrm{m}\) )应小于35200个，100000级受控环境每立方米空气中微粒(直径大于等于 \(0.5\mu \mathrm{m}\) )应小于3520000个。
