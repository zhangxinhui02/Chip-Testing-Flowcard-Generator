# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2023.3 非破坏性键合拉力
### 附录A (方法2023)

---

(资料性附录)  
工艺能力研究示例以最小芯片、100根键合丝的封装作为最坏情况的芯片/外壳组合的例子。所谓最坏情况是以键合线的几何尺寸和数量来表征的。直径为 \(32\mu \mathrm{m}\) 的铝丝，其密封后键合强度合格判据为 \(0.02\mathrm{N}(2\mathrm{gf})\) 。推荐的要求是失效率 \(0.01\% (100\mathrm{ppm})\) 。  
器件的键合强度分布通过每根键合线的键合强度来检验。本示例假设键合强度服从正态分布，对每根键合线的键合强度作正态统计。可以确定最坏情况下键合强度在规范规定的 \(0.02\mathrm{N}\) 左右变化并接近 \(0.02\mathrm{N}\) ，这种最坏情况分布的平均值是 \(0.0426\mathrm{N}(4.26\mathrm{gf})\) ，标准偏差 \(\sigma\) 是 \(0.005\mathrm{N}(0.5\mathrm{gf})\) 。  
因此对于该分布， \(0.02\mathrm{N}\) 的规范对应于 \(4.52\sigma (0.0426 - 0.02) / 0.005\mathrm{N})\) ，而相当于近似 \(3.1\times 10^{- 6}(3.1\mathrm{ppm})\) 水平。如果将分布移动至使 \(0.02\mathrm{N}\) 相当于 \(0.01\% (100\mathrm{ppm})\) 水平(即： \(0.02\mathrm{N}\) 的规范对应于 \(3.719\sigma)\) ，则要求分布由当前键合强度平均值 \(0.0426\mathrm{N}\) 移动大约 \(0.8\sigma (4.52 - 3.719)\) (见图A.1)。采用这个数据来确定能力研究所需器件的数量。  
若数据为正态分布，可用公式(A.1)获得:式中:  
\[n = \\left[(Z_{\\mathrm{a}} + Z_{\\mathrm{p}})^{2}\\right] / (d^{2}) \\quad (A.1)\]  
\(d\) (标准偏差移动) \(= 0.8\)  
\[Z_{\\mathrm{a}} = -1.645(\\alpha = 0.05)\]  
\[Z_{\\mathrm{p}} = -3.09(\\beta = 0.001)\]  
则 \(n = \left[(- 1.645 - 3.09)^{2}\right] / (0.8)^{2} = 22.42 / 0.64 = 35\) 。因此，在本示例能力研究中应使用35个器件(不同标准偏差移动至 \(0.01\% (100\mathrm{ppm})\) 所需的 \(n\) 值见表A.1)。  
采用标准键合工艺，对35个器件(每个器件100根键合线)进行外壳密封，测定密封后的键合强度。  
对35个器件的每个键合线位置的键合强度计算平均值 \(X_{\mathrm{bar}}\) 和标准偏差 \(\sigma\) 。  
对键合强度的分布进行评估，确认其分布与正态分布偏离不明显。  
确定键合拉力的下限(LSL),本示例中的下限为 \(0.02\mathrm{N}(2\mathrm{gf})\) 。  
对每个键合线位置，按公式(A.2)计算 \(Z\) :对每根键合线位置，通过判定键合强度低于 \(Z\) 值的概率来确定键合线失效的概率。因为是正态分布，在本示例中可利用正态分布表，对本示例有100个值。  
\[Z = (X_{\\mathrm{bar}} - \\mathrm{LSL}) / \\sigma \\quad (A.2)\]  
通过计算键合线位置失效的100个 \(P\) 值的总和来估算器件失效的概率。  
\(P\) (器件失效率)为总的 \(P\) (键合线失效率)之和，等于0.00005(50ppm)。  
对本示例已证明器件不满足密封后最小键合强度概率小于0.0001的要求。
