# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5007.2 晶圆批验收
### 3 程序

---

应按表1规定的条件进行晶圆批验收试验。如果根据抽样方案的试验结果判断某一批不合格，作为对整批拒收的一种替代方法，生产厂家可对该批中每一晶圆都进行抽样试验中未能通过的那些试验项目。成功地通过了这些试验项目的所有晶圆应视为构成继续进行其余试验的晶圆批。通不过任一试验项目的所有晶圆都应从该批中剔除。应记录所有试验中得到的数据。不必严格按表1中的顺序进行试验。但是，如果表1“条件”一栏内有明确规定的话，应在规定的工艺点进行试验。如果极限值是按照与批准的设计标称值的偏差确定，则应按GJB1208在提交批准的试验方案中给出“标称值”。如果表1中的极限值是按照与平均值的偏差确定，则首先应该至少对五批晶圆进行电测试，计算其平均值，并应按GJB1208在提交批准的试验方案中给出“平均值”。在任何情况下都不允许设计“标称值”或“平均值”超过表1中规定的绝对极限值。
