# 中华人民共和国国家军用标准
## 方法1022 MOS场效应晶体管阈值电压
### 4.2.2 栅压的测量

---

以 \(0.25\mathrm{V}\) 为增量，依次减小栅压 \(V_{\mathrm{G}}\) ，并测量相应的漏极电流 \(I_{\mathrm{D}}\) 和栅压 \(V_{\mathrm{G}}\) ，直到其中有一达到其最大值。如果在两个极限值均未达到之前，栅压变到 \(0\mathrm{V}\) ，这时必须变换栅压极性转换开关，继续以 \(0.25\mathrm{V}\) 为增量，依次减小栅压，测量 \(I_{\mathrm{D}}\) 和 \(V_{\mathrm{G}}\) ，直到其中一个达到其极限值。
