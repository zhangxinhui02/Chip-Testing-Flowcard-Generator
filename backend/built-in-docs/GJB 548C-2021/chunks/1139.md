# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2031 倒装片拉脱
### 3.1 通则

---

应按下列步骤进行试验。应统计所有的芯片拉力， 且应遵循规定的抽样接收和附加的样品制备， 适用时由规定的LTPD确定被试芯片数目， 而不是键合数。对于混合或多片器件应采用最少四个芯片； 若器件不足四个芯片， 应采用至少两个最终器件的所有芯片。如果器件采用了提高键合强度的密封剂、粘合剂或其他材料， 应在采用这些材料之前进行所有的拉脱试验。  
当倒装芯片所键合的基板不是最终器件的基板时， 应该使用下列条件:  
a) 本试验的被试芯片应从其代表的最终器件中所使用的同一芯片母体中随机抽取；  
b) 在对最终器件进行键合期间， 本试验所用芯片键合设备应与最终器件中使用的设备相同；  
c) 在对最终器件基板进行处理的同一时期内， 对被试芯片基板应采用与最终器件基板一样的处理、金属化和加工。
