# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2023.3 非破坏性键合拉力
### 3.3.2.7 键合设备参数

---

3.3.2.7 键合设备参数必须对每种芯片/外壳组合确定键合设备参数(如：温度、压力、时间、夹具、键合线尺寸、键合线材料、高度设定等)，并通过设计试验来优化键合设备参数范围。这些试验应考虑到键合线几何尺寸的变化程度(如整个键合线的高度、键合线长度、梁的高度等)。这些试验应能确定预计的键合强度和键合操作引起的公差。由受控试验确定的键合参数允许的性能变化范围应在文件中注明。设备参数的变化超出允许的极限时，必须进行评估并在文件中注明仍然满足预计的键合强度和公差。
