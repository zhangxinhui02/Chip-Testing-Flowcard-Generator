# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.17.1 吸附牢固性(短期)

---

应采用代表最大尺寸且使用吸附剂材料的封装类型，制备气密性封装样品。样品中仅包含吸附剂材料和散布在吸附剂材料上的试验用可动粒子。应按正常产品的使用情况，在封装中的规定位置施加规定量的吸附剂材料。应测量并记录吸附剂材料的覆盖范围。除非用户与鉴定机构同意采用其他的试验用可动粒子，试验中采用的可动粒子应包括：a) 球状焊料颗粒：直径 \(76.2\mu \mathrm{m} \sim 152.4\mu \mathrm{m}\) ，2粒。  
b) 铝条：厚 \(50.8\mu \mathrm{m}\) 、宽 \(127\mu \mathrm{m}\) 、长 \(254\mu \mathrm{m}\) ，1条。可用一根直径为 \(50.8\mu \mathrm{m} \sim 152.4\mu \mathrm{m}\) 的铝丝代替。  
c) 金丝：直径 \(25.4\mu \mathrm{m}\) 、长 \(0.4\mathrm{mm} \sim 0.5\mathrm{mm}\) ，1根。  
应按产品的正常生产流程顺序应用吸附剂材料以及进行固化处理。样品经受的环境条件作用顺序应与正常产品相同。应按方法2020试验条件A或试验条件B对样品重复进行三次PIND试验，总共进行四个循环，以检验吸附剂材料吸附外来可动粒子的牢固性。进行PIND试验时应注意样品的放置方向，使得试验中在冲击脉冲作用下吸附的可动粒子最易于从吸附剂材料中脱离出来。最少应评价三个样本，且均应通过规定的PIND判据。
