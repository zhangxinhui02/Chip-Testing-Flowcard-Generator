# 中华人民共和国国家军用标准
## 方法1005.1 稳态寿命
### 3.3.2.1 通则

---

除另有规定外，试验条件A至试验条件E（见3.6）的环境寿命试验温度至少应是 \(125^{\circ }C\) ，对于混合集成电路，试验条件可以根据表1的规定进行变动。当采用规定的试验线路和偏置条件时，供货方可自行确定将试验条件A至试验条件E的环境温度按表1的规定升高以缩短试验时间。因为在正常情况下，外壳温度和结温将明显地高于环境温度，所以对于器件结构应作如下考虑，当器件在试验或工作时，其外壳或结的最高额定温度，对于B级不得超过 \(200^{\circ }C\) ，对于S级不得超过 \(175^{\circ }C\) （见3.3.2.2）。
