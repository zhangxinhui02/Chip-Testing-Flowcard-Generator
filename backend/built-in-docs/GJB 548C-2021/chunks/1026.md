# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2020.2 粒子碰撞噪声检测
### 3.4.2 试验条件B

---

试验条件B的正弦峰值加速度为 \(10\mathrm{g}\) ，其振动装置试验频率 \((f)\) 由封装最大自由活动腔高 \((h)\) 决定，按公式(2)计算：式中：  
\[f = 70.503 / \\sqrt{h} \\quad (2)\]  
\(h\) ——最大自由活动腔高， \(\mathrm{mm}\)  
\(f\) ——振动装置的试验频率， \(\mathrm{Hz}\)  
公式(2)使用的最小频率为 \(60\mathrm{Hz}\) (对应的最大自由活动腔高 \(h\) 为 \(1.381\mathrm{mm}\) )，当器件的最大自由活动腔高大于 \(1.381\mathrm{mm}\) 时，其振动装置的试验频率均为 \(60\mathrm{Hz}\) 。对混合或多芯片组件封装，最大自由活动腔高应从封装空腔的底部或从基板的顶部算起，且不应包括安装在封装内部芯片的厚度。
