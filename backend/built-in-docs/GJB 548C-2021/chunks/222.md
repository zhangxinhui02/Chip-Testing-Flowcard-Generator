# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.2.4 程序

---

器件应放在放射性示踪气体激活箱内。为了减少抽真空的时间，可在激活箱内填充部分惰性材料。将激活箱抽真空到 \(0.07\mathrm{kPa}\) 。然后使器件至少受到 \(202\mathrm{kPa}\) 的氦- 85与干燥氮气混合气体的作用，作用时间至少 \(12\mathrm{min}\) ，实际压力大小和吸收时间应按5.2.1的计算结果。用“每分钟的计数”表示的 \(R\) 值应高出背景600以上，氦- 85与干氮混合气体应抽出存贮，直至激活箱内真空度达到 \(0.07\mathrm{Pa}\sim 0.27\mathrm{Pa}\) 。应在 \(3\mathrm{min}\) 内完成这一气体的抽出存贮过程。该时间从激活过程结束时算起，若采用高压力作用，则从激活压力达到 \(414\mathrm{kPa}\) 时算起。接着使激活箱充满空气(气洗)。然后将器件移出激活箱。在受气体作用后1h内，用装有闪烁晶体的计数台进行泄漏试验。对5.2.3要求的器件封装应先暴露在环境空气中不少于5.2.3规定的“等待时间”。但器件从激活箱中取出到进行试验之间的间隔时间决不能超过1h。应在受到气体作用之后，但又在用计数台确定漏气率之前进行这样的暴露。对5.2.3中未提到的那些器件封装可直接进行试验而不需要“等待时间”，从增压室中一次取出的器件数应受下列要求的限制：即它们中最后一个接受试验的器件应能在1h内完成试验。器件实际漏率按公式(4)计算：式中：  
\[Q = \\frac{n\\cdot Qs}{R} \\quad (4)\]  
\(Q\) ——实际的漏率， \((\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}\)  
\(n\) ——每分钟的实际净读数；  
\(Qs\) 、 \(R\) ——见5.2.1。  
注：排入大气中的氦- 85限量应符合有关标准的规定。
