# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2027.1 芯片粘结强度
### 2 设备

---

试验设备应包括抗拉强度试验台，该试验台应能提供7MPa乘以最大被试芯片面积的力，其准确度为 \(\pm 5\%\) 或 \(0.5\mathrm{N}(50\mathrm{gf})\) 的力(取较小者)。试验设备应具有下述功能：a) 可互换的芯片接触工具，每次试验接触面范围应是被试芯片面积的 \(60\% \sim 100\%\) ；  
b) 能够保证芯片接触工具与安放芯片的底座或基板平面垂直；  
c) 芯片接触工具和底座/基板固定装置之间能够旋转。
