# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 2 术语和定义

---

下列术语和定义适用于本方法。  
2.1 电离辐射效应 ionizing radiation effects由于辐射诱生电荷引起的器件电参数的改变，也称之为总剂量效应。  
2.2 辐射中测试 in-flux test  
辐射期间对器件进行的电测试。  
2.3 内部辐射图形 internal dose pattern同状态向量相关的一种测试条件，用于指定在辐射期间逻辑电路内部所有单元的逻辑条件。  
2.4 非辐射中测试 not in-flux test  
除辐射期间以外的任何时间对器件进行的测试。  
2.5 移位测试 remote test将器件从辐射位置移开后对器件进行的测试。  
2.6 原位测试 in-situ test  
器件在辐射位置进行的测试。  
2.7 时变效应 time dependent effect(TDE)  
辐射后，由于辐射诱生陷阱电荷的积累和(或)退火引起电参数明显退化。在辐射期间也会发生类似的效应。  
2.8 加速退火试验 accelerated annealing test  
利用提高温度加速时变效应的试验程序。  
2.9 低剂量率增强效应 enhanced low dose rate sensitivity(ELDRS)  
某一产品在剂量率低于 \(0.50\mathrm{Gy(Si)} / \mathrm{s}\) 时辐射产生的增强破坏。  
2.10 过辐射试验 overtest根据规范规定的剂量和过辐射因子确定一个新的辐射剂量等级，器件必须通过该辐射剂量等级，才能按规范规定的剂量接收。如一个过辐射因子为1.5,则意味着器件必须要用规范规定剂量的1.5倍来进行试验。  
2.11 参数 \(\Delta\) 值设计容限 parameter delta design margin(PDDM)  
指应用于电参数辐射诱生变化的设计容限。PDDM2指在一个特定剂量率下辐射后的变化值乘以2再加上辐射前的值来判断器件是否超出了辐射后的参数极限。例如，辐射前的 \(I_{\mathrm{b}}\) 值是 \(30\mathrm{nA}\) ，辐射 \(200\mathrm{Gy(Si)}\) 后是 \(70\mathrm{nA}(I_{\mathrm{b}}\) 相差 \(40\mathrm{nA}\) ),那么对于PDDM2来说，其值应为 \(110\mathrm{nA}(30\mathrm{nA} + 2\times 40\mathrm{nA})\) 。如果可接收的辐射后参数极限为 \(100\mathrm{nA}\) ，那么这个器件为失效。
