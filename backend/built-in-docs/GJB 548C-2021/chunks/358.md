# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.13.3 确定 ELDRS 器件生产或批接收辐射试验条件的表征试验

---

如果器件具有 ELDRS 现象或按 4.13.2 进行判定试验时表现出 ELDRS 现象，则可直接(不经过表征)使用剂量率小于或等于 \(0.0001\mathrm{Gy}(\mathrm{Si}) / \mathrm{s}\) (条件 D) 的低剂量率试验，或采用加速试验(条件 E) 进行生产或批接收试验。选择的加速试验方法，应经过详细的表征以确定试验条件。加速试验的方式可选下列方法之一：a) 剂量率大于 \(0.0001\mathrm{Gy}(\mathrm{Si}) / \mathrm{s}\) 的室温低剂量率辐射试验；  
b) 高温辐射试验；  
c) 高剂量率辐射和高温退火的组合试验；  
d) 变剂量率试验；  
e) 其他形式加速试验。  
ELDRS 的表征试验必须表明，使用剂量率小于或等于 \(0.0001\mathrm{Gy}(\mathrm{Si}) / \mathrm{s}\) 、附加过辐射因子和/或参数 \(\Delta\) 值设计裕量的组合方法下，生产或批接收试验的辐射试验程序可界定所有关键电参数的低剂量率响应。因此，作为比对的基准，表征试验应包括以小于 \(0.0001\mathrm{Gy}(\mathrm{Si}) / \mathrm{s}\) 的剂量率达到规范中要求的总剂量水平的辐射试验。
