# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法5004.3 筛选程序
### 4 说明

---

有关的订购文件应规定以下内容:a) 如果不按 3.1,应规定程序和器件等级；  
b) 当无规定或与规定不同时(见第 3 章),应规定试验顺序、试验方法、试验条件、极限值、循环次数、温度、轴向等；  
c) 中间(老炼前和老炼后)电测试参数(见 3.5.1);  
d) 老炼试验条件(见 3.1.10)和老炼试验线路；  
e) 变化量 \(\Delta\) 参数测量或对允许的不合格品率(PDA)的规定，应包括可追溯性程序(见 3.5.1);  
f) 最终电测试(见 3.5.2);  
g) 恒定加速度等级(见 3.2);  
h) 对数据记录和数据报告的要求(适用时)(见 3.6);  
i) 对失效分析(见 3.7)的要求。
