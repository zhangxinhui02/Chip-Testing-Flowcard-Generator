# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2029 陶瓷片式载体焊接强度(破坏性推力)
### 3.3 试样制备

---

基板应按如下方法制备:a) 利用对被试基板进行端铣钻孔或其他适当方法使包含每个片式载体样品 \((50^{+5}_{0})\%\) 焊接面的近似圆形区域暴露出来，若不能以这种方式暴露陶瓷，那么应把片式载体焊接到已经制造了合适的孔和孔径的基板上(假定满足了3.2的条件)。  
b) 必须为被试基板提供适当的支撑，以使在试验期间基板的弯曲减至最小。若需要，这种支撑可以把基板焊接到刚性金属板上。该金属板具有与基板匹配的圆形小孔。  
c) 必须为每个孔径制备一个圆柱形刚性金属试验杆，它将通过支撑板插入基板孔中。试验杆将用来把应力源设备的规定应力传输到暴露的封装表面。试验杆的直径应是相应试验孔直径的 \((85^{+5}_{0})\%\) ，试验杆的长度应足够长，以便当它完全插入孔内时，露出试验孔开端的部分长度达到 \(25.4^{+25.4}_{0}\mathrm{mm}\) 。
