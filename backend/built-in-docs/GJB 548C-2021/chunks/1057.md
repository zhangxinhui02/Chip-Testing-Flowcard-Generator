# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2023.3 非破坏性键合拉力
### 3.3.2.1 通则

---

承制方为了使用替代程序，键合过程中应按GJB 3014的要求实施统计过程控制(SPC)。特性鉴定后发生任何对键合强度有重大影响的变化时，均应开展键合工艺变化对键合完整性影响程度的特性鉴定。对于QML,SPC大纲和本程序所列出的要求都应得到鉴定机构的批准，并随时接受鉴定机构的监督审查。在有要求时，SPC大纲和本程序列出的要求应接受订购方的评审。所有统计评估、特性鉴定和设计试验都应能够提交评审。
