# 中华人民共和国国家军用标准
## 方法1018.2 内部气体成分分析
### 3.3 试验准备

---

器件从烘箱到安装在设备的时间不大于 \(5\mathrm{min}\) ，如果超过 \(5\mathrm{min}\) ，应重新烘烤到 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 。  
设备应能保持一个等同或高于器件温度的稳定温度。在开口真空室使用的夹具应按2.3描述的穿刺装置的要求安装器件，穿刺前器件在 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 下至少保持 \(10\mathrm{min}\) 。  
器件装好后，真空室需要抽真空，且在 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 一直烘烤，直到背景压力水平不会影响获得规定的测量准确度和灵敏度。需要采集背景真空的质谱，并在稍后器件质谱中将其扣除。
