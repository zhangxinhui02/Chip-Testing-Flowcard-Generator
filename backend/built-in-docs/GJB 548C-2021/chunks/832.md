# 中华人民共和国国家军用标准
## 方法2012.2 X射线检查
### 3.1.1 准备

---

为了在灵敏度要求的范围内获得满意的曝光， 并得到X射线检查的器件或缺陷特征图像最详细的细节， 应选择或调整X射线曝光指数、电压、电流和时间设置。在满足要求的前提下， X射线电压应尽量低， 除另有规定外， 不应超过 \(200\mathrm{kV}\) 。尽管高一些的电压能穿透特定封装的器件， 但会损伤一些器件的性能， 必须征得承制方的认可方可使用。实时成像X射线检查系统用剂量率表征， 应鉴定其剂量率， 并建立安全时间限制以确保被试器件不遭受过量辐射。  
注： 实时成像X射线设备应具有同胶片技术一样的成相质量， 与在胶片上显影相比， 在显示器上观察数码相片应具有等效或更好的放大倍数。
