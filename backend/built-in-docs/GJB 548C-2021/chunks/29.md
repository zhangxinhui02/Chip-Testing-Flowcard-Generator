# 中华人民共和国国家军用标准
## 微电子器件试验方法和程序
### 4.5.2 电测试设备的准确度

---

在确定测量值极限时，应考虑测量误差(包括由于偏离标称试验条件而引起的误差)，使器件参数的真值(标称的试验条件下)在规定的极限值之内。  
除另有规定外，以下电测试容差均应适用于器件的测量：a) 电源电压和偏压准确度应保持在规定值的 \(1\%\) 之内。b) 输入调整电压准确度应保持在规定值的 \(1\%\) 或 \(1\mathrm{mV}\) 之内，取其大者。c) 输入脉冲特性、重复率、频率等准确度应保持在 \(10\%\) 之内。应当这样选择标称值，使其 \(\pm 10\%\) 的变化(如果测试设备的变量小于 \(10\%\) ，则选用实际值)不影响规定值测量的准确度。d) 击穿试验施加的电压准确度应保持在规定值的 \(1\%\) 之内。e) 对于数字器件而言，应注意保证只有在其适当的高或低逻辑电平或其他规定电平时才施加最大输出负载电流。f) 阻性负载的容差为 \(\pm 1\%\) 。  
g) 容性负载的容差为 \(\pm 5\%\) 或 \(\pm 1\mathrm{pF}\) ，取其大者。  
h) 感性负载的容差应为 \(\pm 5\%\) 或 \(\pm 5\mu \mathrm{H}\) ，取其大者。  
i) 静态参数应测到 \(1\%\) 以内。  
j) 开关参数应测到 \(5\%\) 或 \(1\mathrm{ns}\) 以内，取其大者。
