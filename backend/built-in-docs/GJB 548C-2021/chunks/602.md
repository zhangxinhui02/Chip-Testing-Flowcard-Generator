# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 5.3.5.1 步骤

---

具体步骤如下：a) 将试验基板和丝网印刷模板放在基座夹具上。b) 将足够量的焊膏涂到丝网印刷模板上。将焊膏涂覆到模板/丝网上，经一次平稳的推刮焊膏，将引出端图形印刷至试验基板。c) 小心地将丝网印刷模板移开，确保焊膏已完全覆盖金属焊盘。d) 小心地将器件的焊柱放入已印刷好的焊膏中。e) 小心地将器件和试验基板一起从基座夹具上移开。f) 在适当的放大倍数下确认器件是否对准。g) 对器件和试验基板实施回流焊接。根据采用的回流焊方法，从表2中选择回流焊参数。h) 冷却后在清洗液中清洗器件和试验基板。  
i) 使用切割设备从试验基板上切下器件， 小心保护与试验基板界面处的焊接点。
