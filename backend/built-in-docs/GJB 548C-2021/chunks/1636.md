# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5012 数字微电路的故障覆盖率测量
### 2.5 故障层次 failure hierarchy

---

对故障的不同等级与划分，包括失效机理、物理失效、逻辑故障、错误等。采用故障层次可以将物理缺陷及其起因与故障仿真器以及可观察到的效应联系起来。其中:失效机理是物理失效的真正起因，例如器件中铝互连线的电迁移。  
物理失效(简称失效)是失效机理引起的实际物理缺陷，例如金属互连线开路。  
逻辑故障(简称故障)是对失效直接反应的逻辑抽象，例如金属互连线开路导致逻辑门输入端的“固定1”行为。  
错误是故障器件相对于无故障器件在一个或多个可观察的初始输出的行为差异。
