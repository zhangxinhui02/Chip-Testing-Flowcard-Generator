# 中华人民共和国国家军用标准
## 方法2012.2 X射线检查
### 2.2 胶片成像X射线检查

---

本试验所用设备和材料包括:a) X射线检查设备： 其电压范围应足以使X射线穿透器件。焦距应适当， 使得主要尺寸为 \(25.1\mu \mathrm{m}\) 的物体能清晰成像。  
b) X射线成像胶片： 颗粒精细的工业级X射线胶片， 单乳胶或双乳胶均可。  
c) X射线照片观察器： 主要尺寸分辨率为 \(25.4\mu \mathrm{m}\) 。  
d) 固定夹具： 能把器件固定在要求的位置上， 而不影响图像的准确性和清晰度。  
e) X射线检查质量标准： 具备分辨检测全部规定缺陷的能力。  
f) 胶片容器： 表面覆盖厚度至少 \(1.6\mathrm{mm}\) 铅材料的工作台或者背面为铅材料的胶片盒， 以防止射线的背散射。
