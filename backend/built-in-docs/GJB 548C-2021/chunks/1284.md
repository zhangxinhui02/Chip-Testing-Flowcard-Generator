# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2036 耐焊接热
### 4.5.7 试验条件I、试验条件J、试验条件K: 红外/对流回流焊

---

操作步骤如下:a) 器件应装在夹具或安装板上(见2.3或2.4), 通孔插装器件的引出端应插入引引出端孔， 表面安装器件应放置在安装板上表面。  
b) 应使用2.6规定的回流炉。  
c) 应在远离边缘的合适位置将小质量热电偶紧密贴在器件的引出端上。  
d) 安装板或夹具应放入试验炉内， 器件温度以 \(1^{\circ}\mathrm{C} / \mathrm{s} \sim 4^{\circ}\mathrm{C} / \mathrm{s}\) 速率上升(用电偶测量)。安装板或夹具应在 \(183^{\circ}\mathrm{C}\) 以上温度保持 \(90 \mathrm{~s} \sim 120 \mathrm{~s}\) ， 并保持在规定试验条件下的最后温度及时间， 然后冷却至室温。按表1构成一次热循环。
