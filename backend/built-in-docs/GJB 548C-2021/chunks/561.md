# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 3.2 浸焊料工具

---

除另有规定外，5.3规定的试验条件A、试验条件B和试验条件C应使用与图1类似的机械或机电式的浸焊料工具，它能够按5.3的规定控制引出端出入焊料槽的速率，并控制在焊料槽内的停顿时间及浸入深度。样品的夹具不要与焊料槽接触，应通过正确设计，使粘附的焊剂减到最少，并能使热损耗最小化并保证试验结果的重复性。应将摇动、振动及其他无关运动减至最小。
