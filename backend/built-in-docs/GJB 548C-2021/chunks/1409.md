# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.6 缺陷和筛选有效性的表征

---

A.6.1 采用本程序生产的产品应经过对工艺过程的表征，以确定缺陷类型，即非关键缺陷、关键缺陷和致命缺陷，以及这些缺陷对可靠性的影响。缺陷表征应考虑到缺陷之间有可能以一定概率发生的相互影响(如由于接触关键尺寸的变化和介质层厚度变化的“相互作用”，影响了接触电阻值)。缺陷表征应确定已知缺陷的类别、每种缺陷类型的根源(以保证足够的缺陷控制)和缺陷的分布(例如，随机的、一个晶圆上不同芯片间的差异、同一批次中各晶圆间的差异、各批次间的差异、不同加工时间的差异)。  
A.6.2 用于评估缺陷对可靠性影响的方法和技术包括(但不限于)：专门设计的实验、失效模式和影响分析(FMEA)、表征数据、对现场失效的分析、承制方对意外失效的分析、可用的历史数据(如公开发表的文献和专利资料、现有的可靠性数据、器件/工艺模型等)。无需了解每种缺陷或缺陷组合对可靠性的影响。但是，对于交付产品重复出现的可靠性问题，必须弄清楚晶圆生产线上产生相关缺陷的来龙去脉。  
A.6.3 缺陷应至少包括表A.1所述类型(由于工艺能力或其他基本原因而不可能产生的缺陷除外)。  
为了验证金属化层的完整性，承制方在工艺控制程序中应考虑到以下金属化隐患：a) 硅消耗；b) 结尖峰；c) 硅析出(球状、结节状突起)；  
d) 铜核化； e) 非平面化； f) 钻蚀； g) 缺口； h) 隧道(贯穿); i) 尖楔。  
A.6.4 缺陷表征应能识别和量化每一掩蔽层的非关键缺陷、关键缺陷和致命缺陷，并在适当的检验点确定合格判据。如果用在线或离线的工艺检验进行 \(100\%\) 筛选，以去除特定的缺陷，就无需对受影响的掩蔽层进行检验和规定合格判据。缺陷表征应能确定变化的主要原因和对缺陷属性(如大小、质量、组成和数量)的影响。缺陷表征应强调缺陷对正在被表征的掩蔽层的影响和对后面的掩蔽层直到最终产品的影响。缺陷表征还应包括在最坏情况条件下缺陷行为，这些条件分别是最坏条件加工位置(如产生关键缺陷的最坏情况物理位置)、最坏情况外围条件(如厚度、温度、气体流量等)和最坏情况设计规则。参见A.12的实例。  
A.6.5 根据缺陷表征的结果，允许的缺陷应小于下述水平，即缺陷不能对器件的可靠性和性能产生不良影响(应采用工艺"安全裕量"的概念，如：如果清楚缺口达 \(25\%\) 的铝条会缩短器件的寿命，则应考虑缺口的裕量极限。即可允许的缺口极限应小于 \(25\%\) )。根据定义，只要观察到任何致命性缺陷(一个或多个),就认为超过了行动极限。  
A.6.6 应根据缺陷表征结果制定检验抽样方案(即样本大小和抽样频度),也可用于在线和离线工艺检验分析技术(见A.7)。  
A.6.7 承制方应在缺陷表征、在线检验及离线检验的基础上建立工艺基线，并对工艺更改进行严格的控制。如果出现了可能加剧缺陷产生的变化，都需进行重新表征。  
示例1:改变工作服，可能影响颗粒的产生，所以应确定这种改变对颗粒的影响是否与已经表征过的相当或是更好，如果更好，就无需进一步开展工作，否则，就应进行重新表征。  
示例2:改换盐酸供应商，需要对新老供应商供应的试剂中的杂质进行对比分析，而后进行工程评价，以确定对产品的影响。如果发现有过量的(超过规范值)、或新的杂质，在更换供应商之前需要进行重新表征。  
A.6.8 承制方所采用的在线或离线筛选，应被证明对消除(超过允许行动极限)的致命性和关键性缺陷的有效性。  
A.6.9 从控制不当分析、成品率分析、用户反馈和检验程序等途径发现的任何新缺陷都应根据A.6的规定进行表征。
