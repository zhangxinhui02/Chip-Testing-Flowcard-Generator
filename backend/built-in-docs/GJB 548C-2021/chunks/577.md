# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 4.7 铜标准缠绕线

---

5.3.3 规定的标准缠绕线应采用软的或拉制的且退火未涂覆的引线来制造，引线直径应为 \(0.635\mathrm{mm} \pm 0.127\mathrm{mm}\) 。引线制备应按以下规定:a) 拉直引线，并将引线切割成适宜长度(最短为 \(50\mathrm{mm}\) )。  
b) 浸入适当清洗剂(如：异丙醇)中 \(2\mathrm{min}\) ，以去除油脂。  
c) 在室温下，在含有氟硼酸(HBF4) \(10\%\) (体积)的水中清洗 \(5\mathrm{min}\) ，并搅拌，操作时要小心。  
d) 漂洗去除酸，步骤如下:  
1) 用常温去离子水或蒸馏水漂洗两次；  
2) 用异丙醇漂洗两次；  
3) 自然干燥。  
e) 浸入焊剂(GB/T9491中规定的RMA型)。  
f) 对于锡铅焊料，浸入 \(245^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}\) 的熔融焊料中 \(5\mathrm{s}\) 。  
按4.1.3的要求清洗或漂洗，以去除或溶解残留的焊剂。  
标准缠绕导线如果未立即使用时，要将其贮存在干净、有盖的容器中。在涂覆后，标准缠绕导线的使用期限不应当超过30d。
