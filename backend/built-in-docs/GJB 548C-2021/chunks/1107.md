# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2029 陶瓷片式载体焊接强度(破坏性推力)
### 3.4 试验

---

试验应按如下方式进行:a) 在试验中应对单个封装施加推力。  
b) 在试验之前，应把一层聚四氟乙烯带安放在暴露的片式载体表面和试验杆之间。  
c) 把试验杆插入试验孔。试验杆与片式载体的接触应无明显碰撞(不大于 \(2.5\mathrm{mm / min}\) )。当试验设备的施力装置以 \(0.5\times (1\pm 1\%) \mathrm{mm / s}\) 的恒定速率推进时，通过试验杆将足够的力施加到片式载体上，使片式载体与基板彻底分开(至少使试验中的片式载体的三边撕开)。发生失效时应记录失效时的作用力和失效类别。导致片式载体或基板破碎的试验应被看作是不可接收的，应不采纳该试验数据，并重新进行试验。
