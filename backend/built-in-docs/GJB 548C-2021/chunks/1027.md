# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2020.2 粒子碰撞噪声检测
### 3.5 试验顺序

---

下述 \(3.5\mathrm{a}) \sim 3.5\mathrm{i}\) 操作顺序构成了一组试验过程：a) 试验前冲击3次；b) 振动 \(3\mathrm{s} \pm 1\mathrm{s}\)  
c) 与 \(3.5\mathrm{~b}\) 振动同时进行3次冲击；d) 振动 \(3\mathrm{~s}\pm 1\mathrm{~s}\) ；e) 与 \(3.5\mathrm{~d}\) 振动同时进行3次冲击；f) 振动 \(3\mathrm{~s}\pm 1\mathrm{~s}\) ；g) 与 \(3.5\mathrm{~f}\) 振动同时进行3次冲击；h) 振动 \(3\mathrm{~s}\pm 1\mathrm{~s}\) ；i) 根据试验结果决定器件接收或拒收。
