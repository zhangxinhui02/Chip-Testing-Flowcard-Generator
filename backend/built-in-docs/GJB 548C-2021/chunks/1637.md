# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5012 数字微电路的故障覆盖率测量
### 2.6 故障覆盖率 fault coverage

---

对于器件的一个逻辑模型、逻辑模型的一个故障群、以及给定的一个测试向量序列，故障群中由该测试向量序列检测到的故障数与故障总数之比。  
注：故障覆盖率也可表示为一个百分比。在本测试方法中，故障覆盖率是基于可测故障的等效分类(见4.3)。对故障覆盖率分数或百分比进行“舍入”处理时是只“舍”而不“入”。例如，如果从仿真的10000个故障中检测出9499个故障，故障覆盖率是 \(94.99\%\) ，如果舍入成两位有效数字，则结果是 \(94\%\) ，而不是 \(95\%\) 。
