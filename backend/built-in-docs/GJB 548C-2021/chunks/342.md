# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.8.1 通则

---

试验方案里应规定需要进行的电参数测量和功能测试。在器件规范中提供的测试条件下，至少测试一个比对器件，以验证测试系统和辐射前后数据的有效性。对自动测试仪器，如果已对器件结温上升进行了最小化处理，则可不限制测试顺序。对手动测试，参数测量顺序应使测试时间最短。当进行一系列测试时，应最先进行功耗最低的测试项目，并按功耗逐渐增加的顺序安排测试。应在同一测试系统上完成辐射前后的电测试。对一系列辐射试验的测试，其测试顺序应保持一致。应视情况进行脉冲方式的电参数测试以使发热和后续的退火效应减至最小。对需要进行加速退火试验(见4.12)的器件，辐射前应进行老炼试验以消除与老炼相关的失效。
