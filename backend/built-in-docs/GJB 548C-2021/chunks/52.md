# 中华人民共和国国家军用标准
## 方法1001 低气压(高空工作)
### 3 程序

---

样品应按规定安装在密封室内，将室内气压降到下述试验条件中的规定值(见表1)。把样品保持在规定的气压下，对它们进行规定的试验。在试验期间及试验前的 \(20\mathrm{min}\) 内，试验温度应为 \(25^{\circ}\mathrm{C}\pm 10^{\circ}\mathrm{C}\) 。器件应施加规定的电压，在从大气压力到规定的最低气压并恢复到大气压力的整个过程中监测器件是否出现故障。器件如出现飞弧、有害的电晕或其他任何影响器件工作的缺陷或退化，都应视为失效。  
连接器件，进行测量，并且在整个抽气过程中施加规定的电压。用微安表或示波器监视施加最大电压(见第4章c))的器件引出端，从直流到 \(30\mathrm{MHz}\) 范围内看其是否出现电晕放电电流。应制定适用试验条件下的数据处理程序(试验电路中的电流减去器件正常工作条件下的电流)，以保证获得被试器件的低气压特性。  
除另有规定外，试验样品在低气压条件下的试验时间，可从下列数值中选取：5min、30min、1h、2h、4h、16h。
