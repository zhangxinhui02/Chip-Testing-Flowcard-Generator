# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2036 耐焊接热
### 4.5.6 试验条件H: 气相回流焊

---

操作步骤如下:a) 器件应装在夹具或安装板上(见2.3或2.4), 通孔插装器件的引出端应插入引出端孔， 表面安装器件应放置在安装板上表面。  
b) 回流炉(见2.6)应足够大以悬挂安装板或夹具而不会碰到炉壁或溶液。气相回流炉液剂应放在试验炉内并加热至沸腾。悬挂安装板和夹具前应让溶液沸腾5min。  
c) 温度、暴露时间、加热次数等规定组合条件应按表1的规定。  
d) 炉腔平衡后， 安装板或夹具应水平悬挂在蒸汽中， 安装板或夹具不得接触溶液。  
e) 加热后， 应让器件在室温条件下冷却稳定。若使用焊膏， 应使用合适的清洗液清洗器件。
