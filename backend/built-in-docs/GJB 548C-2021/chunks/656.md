# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 7.3.7 半易弯曲和易弯曲引线的可选程序

---

作为所有尺寸引线的一种可选试验，除另外规定外，按公式(1)确定的力将施加到每根被试引线上，使外壳本体作 \(90^{\circ} \pm 5^{\circ}\) 的旋转。7.3.4和7.3.5的所有其他条件都适用。  
式中:  
\(W\) ——试验力，N;  
\(S\) ——引线横截面积， \(\mathrm{mm}^{2}\)  
\(K\) ——具体材料的极限抗拉强度， \(\mathrm{N} / \mathrm{mm}^{2}\) 。  
可伐材料的极限抗拉强度典型值为 \(519.4 \mathrm{~N} / \mathrm{mm}^{2}\) ，42号合金的极限抗拉强度典型值为 \(490 \mathrm{~N} / \mathrm{mm}^{2}\) 。其他材料的极限抗拉强度可向供货商索取或通过其他途径得到。计算结果应四舍五入取整。
