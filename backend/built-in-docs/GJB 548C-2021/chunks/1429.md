# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.15.2.5 其他监测

---

用于测量关键工艺参数。包括但不局限于以下部分:a) 金属反射率和电阻率(用于检测金属的异常现象：例如丘状凸起、台阶变薄、粒度改变、空洞等)。  
b) 离子沾污。  
c) 折射率，用于介质层厚度的测量。  
d) 晶圆探针测试后目检。对进行过探针测试的晶圆进行随机抽样监测，先进行高倍数目检，后续进行更详细的分析(电子扫描显微镜、EDX、逐层剥离分析等)。主要用于验证在线监测的有效性。  
e) 酸洗(用于玻璃钝化层缺陷、裂纹和孔的快速检测/染色)或声学显微镜检查(检查玻璃钝化层的完整性)。
