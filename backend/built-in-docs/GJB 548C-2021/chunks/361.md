# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 5 说明

---

有关订购文件中应规定以下内容:a) 器件型号、数量和产品规范(见4.2);  
b) 剂量测量要求(见4.4);  
c) 包括剂量和剂量率的辐射等级(见4.5和4.6);  
d) 若与4.7规定不同，则应规定辐射、电测试和运输温度；  
e) 待测电参数和测试期间的器件工作条件(见4.8.1);  
f) 试验过程中的电测试方式，即采用辐射中测试或非辐射中测试(见4.8.2);  
g) 辐射期间器件的偏置、负载和内部辐射图形条件(见4.9);  
h) 辐射后测试的时间间隔(见4.10);  
i) 需要时，补充室温退火试验要求(见4.11);  
j) 需要时，加速退火试验要求(见4.12);  
k) 需要时，ELDRS判定试验要求(见4.13.2);  
l) 需要时，ELDRS表征试验要求(见4.13.3);  
m) 与器件一起提交的其他文件(见4.14)。
