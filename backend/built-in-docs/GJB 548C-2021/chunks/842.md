# 中华人民共和国国家军用标准
## 方法2012.2 X射线检查
### 3.2.5 胶片和标志

---

X 射线照相胶片应放在背面被铅的胶片盒中，或者把普通胶片盒放在有铅覆盖的工作台上，铅厚至少 \(1.6\mathrm{mm}\) 。应在 X 射线照片上，用照相方法印出下述内容来识别胶片:a) 器件承制方名称或识别代码；  
b) 器件型号或器件编号；  
c) 生产批号或日期代码或检验批号；  
d) X 射线检查胶片观察编号和日期；  
e) 适用时，器件序号或对照号；  
f) X 射线实验室识别标志(如果不是由器件承制方进行试验)。
