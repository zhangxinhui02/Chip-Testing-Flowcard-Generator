# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 10.5 说明

---

有关的订购文件应规定以下内容:a) 若与规定不同(见10.3)应规定抽样准则；  
b) 若与规定不同(见10.4)应规定失效判据；  
c) 若不是采用2.22N,应规定试验中所施加的拉力；  
d) 若不是采用30s,应规定所施加拉力的时间。
