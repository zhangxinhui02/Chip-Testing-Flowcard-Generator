# 中华人民共和国国家军用标准
## 方法1022 MOS场效应晶体管阈值电压
### 4.2.1 测试线路

---

4.2 P沟器件4.2.1 测试线路按图2所示电路连接测试线路，接通测量仪器。将电压源 \(\mathbf{V}_{\mathrm{S}1}\) 和 \(\mathbf{V}_{\mathrm{S}2}\) 置为 \(0\mathrm{V}\) 把被测MOS场效应晶体管插入测试线路。栅压极性转换开关置于合适位置。将电压源 \(\mathbf{V}_{\mathrm{S}1}\) 置于比预计的阈值电压 \(V_{\mathrm{TH}}\) 约高 \(1.0\mathrm{V}\) 的数值。调节电压源 \(\mathbf{V}_{\mathrm{S}2}\) 的大小，直到由电压表示的漏极电压 \(V_{\mathrm{D}}\) 达到规定的数值。这时用电流表 \(\mathbf{A}_1\) 和电压表 \(\mathbf{V}_{1}\) 分别测量并记录漏极电流 \(I_{\mathrm{D}}\) 和栅极电压 \(V_{\mathrm{G}}\) 。
