# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5010.3 复杂单片微电路检验程序
### 3.5.3.1 通则

---

对适用的器件类型及等级，本程序适用于所有质量一致性检验要求。在一个检验组中，各分组可以按任何顺序进行。但在一个分组（B2分组除外）内的各个试验应按本方法所规定的B组、C组、D组和E组的组内试验顺序进行。在B组、C组、D组和E组各分组试验中要求的终点电测试，应按适用的详细规范规定进行。对于要求终点电测试但在订购文件中并未指明具体参数时，则应将100％筛选中规定的最终电参数作为终点电测试参数。同一检验批中电特性筛选测试不合格的器件可用作不要求终点电测试分组的试验。  
对B级质量一致性检验，每一检验批（子批）应通过A组、B组和E组（适用时）检验，或按3.5.3.5作接收检验，定期进行的C组和D组检验按GJB 597B-2012进行，终点电测试应按3.5.3.3中的规定。对S级，每个QCI检验批应按GJB 597B-2012的S级要求组合，S级的质量一致性试验应按表6、表7、表8和表9的规定进行。
