# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 3.5 试验电路板

---

在辐射期间或原位测试中，被试器件应安装或连接在带有给器件提供必要偏置的试验电路板上。除另有规定外，辐射期间器件所有的输入端以及其他可能影响辐射响应的引出端都应电连接，即不得浮空。电路板的几何尺寸和材料应能使被试器件受到均匀辐射。应使用良好的设计和结构以防止振荡、使漏电流减到最小、防止电损伤，并得到准确的测量结果。只有能抗辐射的并且没有显著漏电(相对于被试器件而言)的插座才能用来安装器件和连接带有相关电路的试验电路板。所有在辐射场中反复使用的设施应定期检查其物理和电学性能退化。试验电路板上除被试器件之外放置的所有元件应对累积辐射不敏感，或者对其进行辐射屏蔽。测试夹具应由不影响被试器件附近辐射场强度均匀性的材料制成。应在辐射场外测试夹具的漏电流。插座上不插装器件的情况下，使试验电路板与测试系统相连接，保证所有可能起作用的噪声源和干扰源存在的状态下，施加器件的最大偏置，任意两个引出端之间的漏电流应不超过器件规范中辐射前最小电流极限值的 \(10\%\) 。加速退火试验中为器件施加偏置的试验电路板应能承受加速退火试验的温度，在试验前后均应对试验电路板的物理和电学退化情况进行检查。
