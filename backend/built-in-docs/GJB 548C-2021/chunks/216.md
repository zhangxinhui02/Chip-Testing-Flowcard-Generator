# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.1.2.6 试验条件 \(\mathbf{A}_5\) -He/O₂干法粗检漏和He细检漏(A₁或A₂)的组合方法

---

每次使用前需要对检漏仪(可配有适合尺寸的金属块)进行校准，且在每8h使用后或更换金属块时，以及设备维护后重新进行校准。检漏仪校准时，应基于当前使用温度和氦损耗率，根据相应的可追溯标准漏孔的校准证书，对每个氦标准漏孔的漏率进行校正。通过测量两个经校正的标准漏孔的氦信号，并创建一个线性校准函数，将检漏仪所测量的氦信号转换为定量测量的氦漏率。至少每年应对标准漏孔重新进行校准。  
使用本试验条件的粗检部分可检测的样品最小内腔体积应是在器件盖板上具有直径为 \(0.13\mathrm{mm}\) 或更大漏孔的且可检测到粗漏信号的内腔体积。基准测量应在试验一批器件之前及时记录，并作为确定粗漏拒收依据的定性参考。基准应包括与时间相关的样品室压力，以及在粗漏试验期间使用已知密封性良好的样品或金属块模拟被试样品外部体积的氦和氧基线信号。  
被试器件应放置在密封的氦气压力室中。基于器件细检漏试验所需的等效标准漏率 \(L\) 值，根据候检时间(最长72h)选择加压压力和加压时间，从而使按公式(1)计算得到的示踪气体最大允许测量漏率 \((R_{1})\) 读数比质谱仪的最小可检漏率(背景读数)高出半个数量级。也可使两者差值在半个数量级之内，前提是质谱仪需要有程序文件控制背景信号抬升，以避免干扰测量结果准确性，使得质谱仪测量的 \(R_{1}\) 信号不小于背景信号标准偏差的3倍。  
将被试器件从氦气压力室取出后，应在1h内完成对这些器件的初始粗检漏和细检漏，除非可以证明较长的时间(不超过计算 \(R_{1}\) 所用的候检时间)能够发现粗泄漏和中程细泄漏。应将被试器件依次放入样品室，然后用氮气或氩气吹洗样品室一段时间，以便去除样品室内的空气。开始抽真空时，样品室内的压力数据和采集的氦氧粗漏试验数据，以及氦细漏试验数据均需永久保存。将器件的试验数据与基准数据进行比较，以确定器件是否通过粗检漏。  
在试验过程中，应按以下次序对被试器件进行处理:a) 剔除粗漏失效的器件。  
b) 通过粗检漏和细检漏判据的样品应予接收。  
c) 其余器件(已通过粗检漏但由于氦气吸附或可能存在的中程细泄漏而造成细检漏超标)则应置于空气中，隔一段时间重新检测一次，以确定是否通过细漏判据。如果所测漏率是由于器件密封区域泄漏通道的实际泄漏而非氦气吸附效应，那么细检漏的最大候检时间(FLDT)应通过计算器件内部氦气含量消耗 \(1\%\) 所需的时间来确定。应通过公式(2)计算每个器件的FLDT(单位为小时):式中:  
\[FLDT = K \\cdot P \\cdot V / LMLR \\quad (2)\]  
\(K\) —— \(2.8 \times 10^{- 4}\) , \(\mathrm{h} / \mathrm{s}\) ;  
\(P\) —— \(0.01\) (氦损失 \(1\%\) );  
\(V\) ——被试器件内腔体积， \(\mathrm{cm}^{3}\) ;  
LMLR——初始检漏后立即记录的测量漏率。  
FLDT不应超过72h,且不应大于计算 \(R_{1}\) 时所用的候检时间，未能在规定的FLDT时间内达到细检漏判据的器件应予以拒收。与典型的良好器件相比，在初始试验期间检测到显著泄漏，而又在候检时间内通过了细检漏判据的那些样品，应作进一步的目标以排除粗漏可能。内部体积大于 \(0.1\mathrm{cm}^{3}\) 的器件可置于真空中或在 \(100^{\circ}\mathrm{C}\) 下烘烤不超过20min,以缓解封装表面氦气吸附。
