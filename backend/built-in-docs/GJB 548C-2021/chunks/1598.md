# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.7.2 水汽及其他气体成分的短期放气试验

---

从按5.8.7.1要求制备的封装样品中选取5个，按方法1008的要求，在 \(150^{\circ}C\) 下加热24h，然后立即将这些封装(间隔时间不得超过5min)置入内部气体分析仪器中，按方法1018进行内部气体分析。除水汽外，含量不小于 \(0.01\% (100\mathrm{ppm})\) 的其他气体成分也应以百分数(V/V或ppm)为单位予以报告。本试验应满足5.3.3规定的要求。(按方法1018的要求，只需要从5个封装样品中抽取3个样品进行内部气体分析试验，接收判据为3(0)(见5.3.3)。但是，如果在试验中，出现一个样品失效的情况，则需要再对另两个样品进行试验，采用的接收判据为5(1)。）  
所有被试聚合物材料的质量和暴露面积应与实际应用情况等效。可以采用镀金可伐片和氧化铝片代替实际器件。几种不同用途的聚合物材料可以组合在一起进行本项试验，但总的水汽含量不应超过 \(0.5\%\) (5000ppm)。
