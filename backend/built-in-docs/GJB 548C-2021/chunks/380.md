# 中华人民共和国国家军用标准
## 方法1020.2 剂量率感应锁定
### 4.3.7 温度控制系统

---

如果不是在室温下进行试验，应采用温度控制系统将被试器件的温度控制在规定温度的 \(\pm 10^{\circ}\mathrm{C}\) 范围内。在非室温时，锁定试验的温度应取为器件在系统应用中的最高器件工作温度或是比器件最高额定温度低 \(15^{\circ}\mathrm{C}\) (取低者)(见下注)。如果不知道或提供不了应用温度，则应在比最高额定温度低 \(15^{\circ}\mathrm{C}\) 下进行试验。为了保证器件工作温度不会高于最高额定温度，可以要求采用散热器。  
注：通过锁定试验插座的热导通常比通过焊接板上外引线的热导小得多。
