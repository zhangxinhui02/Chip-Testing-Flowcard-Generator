# 附录A（方法5010）（规范性附录）计算机辅助设计(CAD)的认证要求
### A.1.1 附加的生产线认证要求

---

本附录规定了按GJB1208进行生产线认证时的附加要求。除GJB1208的要求之外，对本附录中问题的回复应提交鉴定机构批准。承制方应以文件形式记录下列附加信息供评审：  
a) 作为承制方内控资料的设计/版图规则文件。b) 承制方单元库中的单元清单、单元性能数据及仿真验证数据(适用时)。c) 承制方采用的工艺监测图形设计；承制方用于鉴定及质量一致性检验(QCI)的标准评价电路(SEC)。d) 标准宏单元集、延迟仿真数据(适用时)。e) 承制方在电路设计过程中采用的软件包清单，包括名称以及当前版本号。f) 设计规则检查(DRC)验证。应对已知包含有违背设计规则的设计进行DRC检查。g) 电学规则检查(ERC)验证。应对已知包含有违背电学规则的设计进行ERC检查。h) 版图与电原理图一致性(LVS)的检查。i) 如果承制方的设计方法是基于“结构修正”技术，则无需采用独立的DRC、ERC和LVS软件。在这种情况下，对f)、g)和h)中的内容不作要求。但承制方应提供适用的实例，表明“结构修正”软件的修正特性。
