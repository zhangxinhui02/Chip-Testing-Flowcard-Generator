# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2026.1 随机振动
### 3.1 谱密度曲线

---

用图来表示振动装置输出的功率谱密度与频率的关系。功率谱密度是在一窄带滤波器的每个单一滤波通带中所通过的一个波幅的均方值。此处以 \(G^{2} / f\) 表示。 \(G^{2} / f\) 是加速度的均方值，以滤波通带每一周波数的重力单位表示。分贝数 \(dB\) 按公式(1)定义为:在 \(f_{1}\) 和 \(f_{2}\) 之间频带内加速度均方根值 \(G_{\mathrm{rms}}\) 由公式(2)给出:式中:  
\[dB = 10\\lg \\frac{G^{2} / f}{G_{\\mathrm{r}}^{2} / f} = 20\\lg \\frac{G / (f)^{1 / 2}}{G_{\\mathrm{r}} / (f)^{1 / 2}} \\quad (1)\]  
\[G_{\\mathrm{rms}} = \\left(\\int_{f_{1}}^{f_{2}}G^{2} / f\\mathrm{d}f\\right)^{1 / 2} \\quad (2)\]  
\(G_{\mathrm{r}}^{2} / f\) ——是给定的功率谱密度参考值，通常是规定的最大值。  
谱密度值在规定的下限频率与 \(1000\mathrm{Hz}\) 之间应为规定值 \(^{+40\%}_{- 50\%}\) ( \(\pm 1.5\mathrm{dB}\) )之内，在 \(1000\mathrm{Hz}\) 与规定的上限频率 \((2000\mathrm{Hz})\) 之间应为规定值 \(^{+100\%}_{- 50\%}\) ( \(\pm 3\mathrm{dB}\) )之内。滤波器带宽是1/3倍频的最大值或 \(25\mathrm{Hz}\) 频率(取较大者)。
