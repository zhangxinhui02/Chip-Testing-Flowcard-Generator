# 中华人民共和国国家军用标准
## 方法2004.3 引线牢固性
### 7.3.11 失效判据

---

去掉应力后放大10倍 \(\sim 20\) 倍检查，任何完全的断裂(如引线脱离本体)都被视为失效。  
注：对于QFJ封装来说，按弯曲应力试验(试验条件 \(\mathrm{B}_{1}\) )6.3.8.5的规定进行试验，其失效判据仅适用于从引线最宽部位到最窄部位的过渡区域。对于QFJ封装弯曲来说，按弯曲应力试验(试验条件 \(\mathrm{B}_{1}\) )6.3.8.5的规定进行试验，其失效判据适用于整根引线，而对于6.3.8.6则仅适用于引线最宽部位。
