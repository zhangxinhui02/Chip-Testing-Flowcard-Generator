# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
### A.5 鉴定机构批准

---

A.5 鉴定机构批准A.5.1 承制方执行本程序应由鉴定机构审查批准。在合格产品交货之前， 承制方应向鉴定机构提交执行本程序的书面材料，该书面材料应严格规定所包含的内容(如晶圆生产线的详细说明，包括：地点、工艺基线、任选的器件和工艺技术等)。  
A.5.2 为了使生产线满足本要求，鉴定机构应确定是否有必要派有专业经验的审核员开展半导体晶圆制造方面的评价，以确认该生产线是否满足本程序的要求。有关半导体晶圆制造方面的专业经验应包括以下内容：晶圆生产工艺流程、晶圆生产工艺与测量工具、晶圆生产工艺的化学与物理特性、可靠性物理特性以及缺陷的产生与控制等。
