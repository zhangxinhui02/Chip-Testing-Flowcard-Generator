# 中华人民共和国国家军用标准
## 方法1015.1 老炼
### 3.3.4 温度加速试验

---

若采用试验条件F，器件在 \(175^{\circ}\mathrm{C}\sim 250^{\circ}\mathrm{C}\) (该温度大大超过最高额定结温)下加上偏置。通常发现在这样的高温下，器件不能按有关的订购文件中的规定正常工作，因此必须特别注意选择偏置电路和条件，以保证电路的重要部位得到充分偏置又不使电路的其他部位受到破坏性的过应力。为了适当的选择加速试验条件，建议将适当选取的器件样品置于预定的高温下，测量各个器件引出端上的电压和电流，以确信施加的电力未导致损坏性的过应力。除详细规范中另有规定外，应采用表1确定的最小时间一温度组合。最短试验时间应为 \(12\mathrm{h}\) 。在任一或全部引出端上的外加电压等于推荐的 \(125^{\circ}\mathrm{C}\) 时的工作电压。若在规定电压下工作时会引入过量电流或过大的功率耗散，相应引出端上的外加电压最低可以降到规定电压的 \(50\%\) ，同时按方法1005中3.6.6给出的公式确定试验时间。带有内部过热保护电路的器件应按方法1005中3.6.6处理。在任何时候都必须防止热失控状况的出现。
