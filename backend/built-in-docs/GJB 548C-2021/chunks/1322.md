# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法3015.1 静电放电敏感度的分级
### 4.4 试验的引出端组合

---

组合方式如下:a) 将器件的地引出端连接到B端，其他每条引出端依次连接到A端。除了进行试验的一个引出端和地引出端以外，其余所有引出端应开路(若器件存在多个地引出端(组),按表2脚注c进行试验)。  
b) 将所有命名为电源引出端(例如 \(V_{SS1}\) 或 \(V_{SS2}\) 或 \(V_{SS3}\) 或 \(V_{CC1}\) 或 \(V_{CC2}\))的每个分组依次连接到B端，其他每个引出端依次连接到A端。除了进行试验的引出端和电源引出端(组)外，其余所有引出端应开路。  
c) 将每个输入或输出引出端依次连接到A端，其余的所有输入和输出引出端组合连接到B端。除了被试验的输入或输出引出端及其余的输入和输出引出端组合外，所有其他的引出端都开路。  
注1: 允许在A端和B端之间放置一个旁路电阻(10kΩ或更大)来消除脉冲前的电压现象，这种现象主要发生在高阻态引出端上。该旁路电阻的存在不能影响HBM波形参数、或设备验收、校准和波形验证。旁路电阻可置于HBM模拟器或试验夹具系统上。  
注2：在每次放电结束后，允许将所有的引出端同时接地处理，然后施加下一次静电脉冲或V/I曲线扫描。
