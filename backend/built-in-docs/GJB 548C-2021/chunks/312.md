# 中华人民共和国国家军用标准
## 方法1018.2 内部气体成分分析
### 3.4 器件刺穿

---

抽真空后，对器件的外壳或盖板进行刺穿，然后用质谱仪测量所释放出的气体含量。检测出的水汽含量，按单位体积的百分比或总体气体成分的 \(10^{- 6}(\mathrm{ppm})\) 表示。  
除水汽外其他气体的比例(按体积):氮、氨、质量数69的碳氟化合物、氧、氩、氢、二氧化碳、甲烷、氨和其他溶剂(必要时)。所有出现的气体都应计算并在检测报告中给出。应对数据做简化处理，以消除其他气体裂解图谱对计算水汽含量的影响。对任何与系统相关的影响(如内部环境中氢气的存在)都应进行数据的修正。  
刺穿器件封装释放气体时真空室压力将增加。当出现实际的压力变化比预期的器件空腔体积释气增压改变 \(\pm 25\%\) 情况时，可能是以下原因造成:a) 器件外壳未完全刺穿；  
b) 器件外壳不密封；  
c) 器件内未包含常规的内部压力。
