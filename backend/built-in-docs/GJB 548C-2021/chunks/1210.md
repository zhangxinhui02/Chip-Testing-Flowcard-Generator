# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2032.1 无源元件的目检
### 4.2.2.6 金属化对准

---

呈现下列情况的元件不得接收:a)H级：被金属化覆盖的面积少于 \(50\%\) 的接触窗口。  
K级：少于 \(75\%\) 。  
b)H级：连续被金属化覆盖的周边少于 \(40\%\) 周长的接触窗口(见图7a))。  
注：当按设计要求，金属化完全包含在接触窗口之内或未覆盖整个接触周长时，如果满足设计要求，则可以不考虑4.2.2.6a)面积覆盖或4.2.2.6b)周长覆盖条款。  
K级：少于 \(50\%\) (见图7b))。  
c)H级：不应覆盖接触窗口的金属化通路，与接触窗口间隔小于 \(3\mu \mathrm{m}\) (设计要求除外)。K级：同H级。
