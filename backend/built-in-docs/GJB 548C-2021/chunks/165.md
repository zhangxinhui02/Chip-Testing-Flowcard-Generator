# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 2.3 结温 junction temperature

---

\(T_{\mathrm{J}}\)  
器件中产生主要热量的半导体结的温度。 \(T_{\mathrm{J}(\mathrm{峰值})}\) 是非均匀电流分布下工作结的峰值温度， \(T_{\mathrm{J}(\mathrm{平均})}\) 是非均匀电流分布下工作结的平均温度， \(T_{\mathrm{J}(\mathrm{区域})}\) 是最接近工作结的六个等效半径处的温度(等效半径是面积与结界面区域面积相同的等效圆的半径)。通常， \(T_{\mathrm{J}(\mathrm{区域})} \leq T_{\mathrm{J}(\mathrm{平均})} \leq T_{\mathrm{J}(\mathrm{峰值})}\) (即使工作结中的电流分布是均匀的)。
