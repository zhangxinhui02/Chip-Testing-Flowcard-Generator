# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 4.2.3.2 测量 \(T_{\mathrm{J}(\mathrm{平均})}\) 的开关方法

---

本测量采用下列符号:  
\(I_{\mathrm{M}}\) : 测量电流 \((\mathrm{mA})\) ；  
\(V_{\mathrm{MD}}\) : 在施加 \(P_{\mathrm{D}}\) 的相应结温下，测量电流为 \(I_{\mathrm{M}}\) 时测得的温敏参数值 \((\mathrm{mV})\) ；  
\(T_{\mathrm{MC}}\) : 在参考点上测得的校准温度(℃);  
\(V_{\mathrm{MC}}\) : 在规定的 \(T_{\mathrm{MC}}\) 值时测量电流为 \(I_{\mathrm{M}}\) 时测得的温敏参数值 \((\mathrm{mV})\) 。  
采用结正向电压作温敏参数的 \(T_{\mathrm{J}(\mathrm{平均})}\) 测量，应按下列步骤进行:步骤1:测量温敏参数的温度系数(校准):在烘箱内或在温度可控的散热器上对被试器件进行外部加热，规定恒定的测量电流 \(I_{\mathrm{M}}\) 和集电极电压。通过测量温敏参数与参考点温度的关系，得出温敏参数的温度系数。规定参考点被校准的温度范围应不少于施加功率试验(见步骤2)中所涉及的温度范围。通常，选择的测量电流使得在感兴趣的范围内，随着温度的增加，温敏参数线性地减少，并且在测量期间内部产生的热量可以忽略不计。测量温敏参数时，根据被试器件的额定条件和工作条件，通常采用的测量电流范围为 \(0.05\mathrm{mA}\sim 5\mathrm{mA}\) 。在本试验中采用的具体测量电流和集电极电压的情况下，温敏参数温度系数值 \((V_{\mathrm{MC}} / T_{\mathrm{MC}})\) 可从校准曲线 \((V_{\mathrm{MC}}\) 与 \(T_{\mathrm{MC}}\) 的关系曲线)中计算得到。  
步骤2:施加功率的试验：施加功率的试验分两部分进行。两部分试验的规定参考点温度均应保持在固定的预置值上。首先进行测量的是温敏参数 \(V_{\mathrm{MC}}\) ，它是在校准期间所用的测量电流 \(I_{\mathrm{M}}\) 和集电极电压下测量得到的。而后用占空系数不少于 \(99\%\) 的加热功率间断地施加到被测量的器件上。应采用在校准期间所用的测量电流 \(I_{\mathrm{M}}\) 和集电极电压(见步骤1),在加热脉冲间隔(小于或等于 \(100\mu \mathrm{s}\) )的时间内测量温敏参数 \(V_{\mathrm{MD}}\) 。  
由于某些半导体单元的温度在去掉加热电源和测温敏参数之间的时间里会降低，所以要将 \(V_{\mathrm{MD}}\) 回推到加热电源终止时刻的值。对于冷却开始后的前 \(100\mu \mathrm{s}\) ，可用公式(2)计算:式中:  
\(V_{\mathrm{MD}}(t = 0)\) ——外推到加热功率终止时刻的温敏参数， \(\mathrm{mV}\) ;  
\(t\) ——加热功率终止后的延迟时间， \(\mu \mathrm{s}\) ;  
\(V_{\mathrm{MD1}}\) —— \(t = t_{1}\) 时刻的温敏参数， \(\mathrm{mV}\) ;

如果在线性坐标纸上绘制冷却开始后的前 \(100\mu \mathrm{s}\) 的 \(V_{\mathrm{MD}}(t)\) 与 \(t^{1 / 2}\) 关系曲线，绘出的将是一条直线(非热开关瞬变占优势的初始部分除外)。 \(t_{2}\) 是可进行温敏参数测量的最短时间，它由 \(V_{\mathrm{MD}}(t)\) 与 \(t^{1 / 2}\) 冷却关系曲线上的线性部位确定。 \(t_{1}\) 至少应等于 \(t_{2} + 25\mu \mathrm{s}\) ，但应小于 \(100\mu \mathrm{s}\) 。对多数据件而言，可测量温敏参数之前的延迟时间范围为 \(1\mu \mathrm{s}\sim 50\mu \mathrm{s}\) 。对于作为敏感器件的半导体(结)单元，若直径大于或等于 \(0.2\mathrm{mm}\) ，延迟时间超出 \(1\mu \mathrm{s}\sim 50\mu \mathrm{s}\) 范围，这个外推方法是正确的。  
由于采用 \(V_{\mathrm{MD2}}\) 代替外推值 \(V_{\mathrm{MD}}(t = 0)\) 而引起热阻计算误差超过 \(5\%\) 时，应采用外推的 \(V_{\mathrm{MD}}\) 值来计算平均结温。  
加热功率 \(P_{\mathrm{D}}\) 的选择应使得计算的结与规定参考点之间的温差(由 \(V_{\mathrm{MD2}}\) 的测量)不小于 \(20^{\circ}\mathrm{C}\) 。在施加功率的试验中，应记录 \(V_{\mathrm{MD}}\) 、 \(V_{\mathrm{MC}}\) 和 \(P_{\mathrm{D}}\) 值。  
应记录和计算下列数据:a) 温敏电参数，即 \(V_{\mathrm{F}}\) ， \(V_{\mathrm{EB}}\) (仅发射极开关), \(V_{\mathrm{FB}}\) (发射极和集电极开关), \(V_{\mathrm{CB}}\) , \(V_{\mathrm{F}}\) (基板)或其他温敏参数。  
b) 平均结温 \(T_{\mathrm{J}}\) (平均),按公式(3)计算:式中:  
\(T_{\mathrm{R}} = T_{\mathrm{C}}\) 或 \(T_{\mathrm{M}}\) 。  
c) 外壳温度 \(T_{\mathrm{C}}\) 或安装表面温度 \(T_{\mathrm{M}}\) (通常是 \((60\pm 0.5)^{\circ}\mathrm{C}\) )。  
d) 功耗 \(P_{\mathrm{D}}\) ，此处 \(P_{\mathrm{D}} = P_{\mathrm{D}}\) (封装)或 \(P_{\mathrm{D}}\) (单结)。  
e) 安装布局。
