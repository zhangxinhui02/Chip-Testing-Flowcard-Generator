# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2036 耐焊接热
### 4.5.1 概述

---

除另有规定外，应对器件上的全部焊接引出端进行试验(如图1)。本试验条件包括六种焊接技术，试验条件见表1，具体如下：试验条件A：烙铁一焊帽、通孔插装器件、焊片及焊接头、焊孔的手工焊接；  
试验条件B：浸焊料一模拟有引线器件的热焊料浸涂(浸锡)；  
试验条件C：波峰焊一模拟安装在印制板上表面产品的波峰焊接；  
试验条件D：波峰焊一模拟安装在印制板下表面产品的波峰焊接；  
试验条件H：气相回流焊一没有预热的气相回流环境；  
试验条件I、试验条件J、试验条件K：红外/对流回流焊一模拟红外回流、自然对流和热风对流回流环境(见表1的温度要求)。
