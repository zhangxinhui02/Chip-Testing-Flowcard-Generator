# 中华人民共和国国家军用标准
## 方法1005.1 稳态寿命
### 3.4.1 试验温度低于或等于 \(150^{\circ}\mathrm{C}\) 的测试

---

除另有规定外，全部规定的中间和终点测试应在器件去掉规定的试验条件(即去掉加热条件或者去掉偏置)后的96h内完成。如果这些测试不能在96h内完成，则器件在进行中间或终点测试之前，还应至少再承受24h的寿命试验，试验条件(见3.6)和温度应与先前采用的相同。如果有规定或承制方自愿时，应在 \(168^{+72}\mathrm{h}\) 和 \(504^{+68}\mathrm{h}\) 进行中间测试。如果规定试验时间超过 \(1000\mathrm{h}\) ，附加的中间测试点应是 \(1000^{+168}\mathrm{h}\) 和 \(2000^{+168}\mathrm{h}\) ，以后每次相隔时间为 \(1000^{+168}\mathrm{h}\) 。中间测试主要是规定的参数和条件，这些参数和条件应包括能足以揭示致命失效和退化失效的主要功能特征。器件应至少冷却到与器件处于室温下功耗稳定状态时的温度相差不超过 \(10^{\circ}\mathrm{C}\) ，才能去掉偏置。  
为了把器件从进行寿命试验的试验箱中移到与其隔开的冷却位置上，对偏置中断不超过1min时不应视作去掉偏置。除线性器件、MOS(CMOS、NMOS、PMOS等)器件或其他规定的器件外，只要能使被试器件的外壳温度在去掉试验条件后的30min内降到低于 \(35^{\circ}\mathrm{C}\) ，并且被试器件在去掉偏置后的5min内移出加热的试验箱，器件就可以在冷却期间去掉偏置。在器件重新加热之前，应完成全部规定的 \(25^{\circ}\mathrm{C}\) 条件下的电测试。
