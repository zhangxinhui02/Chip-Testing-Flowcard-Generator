# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法3015.1 静电放电敏感度的分级
### 3.3.3 有效电容

---

有效电容是在试验插座中不插器件且试验开关断开的情况下，将 \(\mathrm{C_1}\) 充电到规定电压(见表1),然后将 \(\mathrm{C_1}\) 的电荷释放到图1中置于A、B两端之间的静电计、库伦表或标准电容来确定，在规定电压范围内有效电容应是 \(100\times (1\pm 10\%) \mathrm{pF}\) ，且在 \(1000\mathrm{V}\) 下定期验证。  
注：为了缓慢放电以便得到正确的测量值，可接一个串连电阻。  
![](assets/7639d44d4024b61c9adc306ed9600c19b15b7b20487b358a3f60788255c7461f.png)  
\(C_{1}\) - 有效电容(见3.3.3), \(C = 100 \times (1 \pm 10\%) \mathrm{pF}\) (绝缘电阻不小于 \(10^{12} \Omega\) );  
\(\mathrm{R}_{1}\) - 电阻， \(R = 10^{6} \Omega \sim 10^{7} \Omega\) ;  
\(\mathrm{R}_{2}\) - 电阻， \(R = 1500 \times (1 \pm 1\%) \Omega\) ;  
\(\mathrm{S}_{1}\) - 高压继电器(无回跳、水银继电器或等效的继电器);  
\(\mathrm{S}_{2}\) - 常闭开关(脉冲放电和电容测量期间开路)。
