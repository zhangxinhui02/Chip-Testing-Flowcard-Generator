# 附录A（方法5010）（规范性附录）计算机辅助设计(CAD)的认证要求
### A.1.2 功能延迟仿真

---

对每一项最终的专用电路设计和版图(即完成布线的设计)进行仿真，并由承制方存档。仿真时应采用从生产中实际使用的器件布局及版图中计算得到的寄生参数和实际延迟。实际延迟应包括门延迟、实际金属互连线电容及输出端器件负载的贡献。利用这些实际延迟，专用集成电路(ASIC)的设计人员应确保ASIC电路没有违反时序约束。这种时序上的违背应包括(但不限于)由于工艺、温度及电源电压波动引起的建立、维持、关键延迟路径及电路竞争等情况。两种最坏极端情况下(针对温度、工艺、辐射(适用时)及电源电压)关于电路工作情况的仿真结果应完全一致(即在每一时钟周期内，各信号的最终状态应完全相同)。
