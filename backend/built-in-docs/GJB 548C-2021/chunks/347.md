# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.10 辐射后测试的程序

---

除另有规定外，应遵循下述时间间隔:a) 对条件A,辐射结束到电测试开始的间隔时间应不超过1h。对条件B、条件C、条件D和条件E,如果间隔超过1h,辐射结束到电测试开始的间隔时间可以等于辐射时间的10%,最多不能超过72h,否则也不应超过1h。对条件A、条件B、条件C、条件D和条件E,如果进行移位测试，可以选择将器件保存到干冰中，但是必须在辐射后15min内将器件包裹好。保存在干冰环境内的器件引线应短接，并维持器件的温度不高于 \(-60^{\circ}\mathrm{C}\) 。电测试应在器件温度恢复到室温的情况下进行，但温度的恢复时间必须控制在30min以内，并且从辐射结束到电测试开始的时间间隔不得超过72h,电测试应按4.7.1的要求进行。选择干冰保存的测试方式时，如果经过表征试验，处于室温的时间、以及在干冰中保持的温度和时间也可以与上述要求不同。具体见4.10c)。  
b) 对条件A,前一次辐射结束到后一次辐射开始的时间间隔应不超过2h。对条件B、条件C、条件D和条件E,如果间隔超过2h,前一次辐射结束到后一次辐射开始的时间间隔可以等于前一次辐射时间的20%,最多不能超过120h,否则也不应超过2h。对条件A、条件B、条件C、条件D和条件E,当分步辐射的中间电测试采用移位测试时，可以选择将器件保存到干冰中，但是必须在电测试后15min内将器件包裹好。保存在干冰环境内的器件引线应短接，并维持器件的温度不高于 \(-60^{\circ}\mathrm{c}\) 。辐射应在器件恢复到室温的情况下进行，但温度的恢复时间必须控制在30min以内，并且从上次电测试结束到下一次辐射开始的时间间隔不得超过72h。选择干冰保存的测试方式时，如果经过表征试验，处于室温的时间、以及在干冰中保持的温度和时间也可以与上述要求不同。具体见 \(4.10\mathrm{c}\) )。  
c) 如果使用干冰保存器件，应通过针对性的退火表征试验，证明退火对所有关键参数的变化影响小于辐射后1h内室温参数测量值的10%。如果采用4.10a)和4.10b)中所列的其他时间和温度条件，这些条件应经过表征。但是，辐射后到电测试开始的间隔时间不应超过72h,并且对分步辐射试验，上次辐射结束到下一次辐射开始的间隔时间不应超过120h。例如，可以允许承制方的低温规范限高于 \(-60^{\circ}\mathrm{C}\) 。另例，如果器件按条件A进行辐射后，在室温下退火不明显，也可以对1h和2h的限制加以适当放宽。对任何除4.10a)和4.10b)中所列的其他时间和温度条件，必须能够证明，对所有的关键参数，同辐射后1h内的室温测试数据比较，这些条件产生的退火变化在10%以内。如果采用1h以后的测试数据作为比较基准，则应对辐射试验时间进行适当地加严限定。表征试验的结果应包含在4.14的试验报告中。  
为使时变效应减至最小，以上时间间隔应尽可能的短。对一系列试验(分步辐射方式),每次辐射试验的参数测量顺序应保持一致。
