# 中华人民共和国国家军用标准
## 方法2010.2 内部目检(单片电路)
### 4.3.5.2 梁式引线

---

\(4.3.5.2\mathrm{a})\sim 4.3.5.2\mathrm{f})\) 同时适用于条件A(S级)和条件B(B级)。呈现下列情况的器件不得接收:a) 空洞、裂痕、凹陷或划伤使未受破坏的部分小于梁宽的 \(50\%\) ；  
b) 梁与芯片分离；  
c) 除设计规定外的梁式引线损伤或不完整；  
d) 未键合的梁式引线；  
e) 键合区离钝化层边沿小于 \(2.5\mu \mathrm{m}\) ；  
f) 从梁的两边看去， 在芯片和梁之间的钝化层小于 \(2.5\mu \mathrm{m}\) (见图38和图39)。
