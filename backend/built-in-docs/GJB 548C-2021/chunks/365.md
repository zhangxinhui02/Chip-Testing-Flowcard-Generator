# 中华人民共和国国家军用标准
## 方法1020.2 剂量率感应锁定
### 2.2 剂量率感应锁定 dose rate induced latchup

---

由于电离辐射脉冲产生的光电流作用，使器件内寄生结构(例如 \(\mathrm{p} - \mathrm{n} - \mathrm{p} - \mathrm{n}\) 或 \(\mathrm{n} - \mathrm{p} - \mathrm{n} - \mathrm{p}\) 四层通路)导通，而且在光电流消失后仍维持一段不确定时间的导通状态。这是一种可恢复的器件效应。只要电源电压大于维持电压、电流大于维持电流，器件就维持锁定。锁定破坏了电路中某一部分的正常工作，并可能由于半导体区域、金属化层或键合线的局部过热引起器件致命损坏。
