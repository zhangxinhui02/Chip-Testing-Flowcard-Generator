# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.12.2 需要进行加速退火试验的情况

---

试验各方应采取合适的步骤确定是否需要进行加速退火试验。应采用下列判断标准:  
a) 任何含有MOS电路单元(如晶体管或电容器)的器件或电路应进行4.12.3规定的试验。  
b) 在如下任一情况可不考虑TDE:  
1) 已知器件的电路设计中不包含MOS单元。  
2) 已知应用环境的总剂量不超过 \(50\mathrm{Gy(Si)}\) 。  
3) 已知器件在辐射环境服役的总寿命小于TDE时间。  
4) 按预定应用环境的剂量率进行辐射试验。  
5) 通过表征试验， 业已证明， 该器件类型或IC工艺， 并未表现出器件参数TDE变化大于试验误差(或大于规定的上限), 而且也已证明影响TDE响应的变量对于特定的生产工艺过程已处于受控状态。作为起码要求， 上述表征试验应包括评价TDE对传输延迟、输出驱动、最小工作电压参数的影响、对影响TDE变量的连续工艺控制状态可通过MOS测试结构的辐射加固批取样试验得到证明。  
c) 对不包含MOS单元器件的加速退火问题， 本方法未提供指导意见。  
如果器件按试验方案或相关规范中规定的总剂量等级通过了标准条件或4.11(若使用了该程序)试验， 且不属于4.12.2中规定可不进行TDE试验的情况， 应按4.12.3进行加速退火试验。
