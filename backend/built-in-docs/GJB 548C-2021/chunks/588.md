# 中华人民共和国国家军用标准
## 方法2003.2 可焊性
### 5.3.1.3 步骤

---

具体步骤如下：a) 在浸焊料前，应从熔融焊料表面撤去焊剂浮渣及焦渣；b) 已涂焊剂的样品浸入熔融焊料，焊料表面距有引线通孔安装器件本体或安装支撑面应小于 \(1.27\mathrm{mm}\) （取距离器件本体较远者，见图3）；c) 浸入速度和提出速度为 \(25\mathrm{mm / s}\pm 6\mathrm{mm / s}\) ，停留 \(5_{-0.5}^{+0}\mathrm{s}\) ；d) 提出后，在维持样品试验条件的同时，焊料允许在空气中冷却固化；e) 检查前，应按4.1.3要求，去除所有引线上的全部可见残余焊剂。
