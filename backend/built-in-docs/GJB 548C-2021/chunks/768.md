# 中华人民共和国国家军用标准
## 方法2010.2 内部目检(单片电路)
### 4.3.1 通则

---

对每个器件都应进行 \(4.3.2 \sim 4.3.4\) 要求的内部自检，低放大倍数的检查是在30倍 \(\sim 60\) 倍范围。此外，4.3.5可适用于梁式引线工艺，4.3.6的检查应在所规定的高放大倍数和低放大倍数下进行，高放大倍数与4.2.1规定的一样，且遵循4.1a)中的条件。
