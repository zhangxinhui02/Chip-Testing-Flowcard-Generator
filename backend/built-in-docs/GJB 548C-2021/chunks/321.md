# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 3.3 铅/铝 \((\mathrm{Pb / Al})\) 容器

---

被试器件应放在 \(\mathrm{Pb / Al}\) 容器内，以减小由低能散射辐射引起的剂量增强效应。容器外层要求至少为 \(1.5\mathrm{mm}\) 厚的铅，内层屏蔽层至少为 \(0.7\mathrm{mm}\) 厚的铝。这种 \(\mathrm{Pb / Al}\) 容器对Si以及对如 \(\mathrm{CaF}_2\) 产生近似的带电粒子平衡。在下述三种情况下应测量 \(\mathrm{Pb / Al}\) 容器的内部辐射场强度：a) 开始时；  
b) 源变换时；  
c) 源的方向或结构变化、容器和试验夹具变化时。  
测量时应将剂量计(例如一个TLD)放置于器件辐射容器中靠近被试器件所在位置。如果能证明，低能散射辐射足够小，以至不会因剂量增强效应引起剂量测定误差，也可不使用 \(\mathrm{Pb / Al}\) 容器。
