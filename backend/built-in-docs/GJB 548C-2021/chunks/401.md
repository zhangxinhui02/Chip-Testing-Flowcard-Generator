# 中华人民共和国国家军用标准
## 方法1021.1 数字微电路的剂量率翻转
### 4.1 辐射源

---

试验可采用下述两种辐射源中的一种：闪光X射线机(FXR)或电子直线加速器(LINAC)。FXR应用于X射线模式，LINAC用于电子束模式。LINAC束能量应大于 \(10\mathrm{MeV}\) 。在被试器件和剂量测试仪器所在的区域内，辐射源应提供一个均匀的(偏离小于 \(20\%\) )的辐射等级。用于窄脉冲测量的辐射脉宽应为 \(10\mathrm{ns}\sim 50\mathrm{ns}\) 。对窄脉冲测量，可采用FXR或LINAC。对宽脉冲测量(典型值为 \(1\mu \mathrm{s}\sim 10\mu \mathrm{s}\) ),应采用LINAC。对LINAC辐射的脉宽应作出规定。被试器件所在处剂量率对窄脉冲应在 \(10^{4}\mathrm{Gy(Si)} / \mathrm{s}\sim\)\(10^{10}\mathrm{Gy(Si)} / \mathrm{s}\) (或根据要求)之间可调。对于宽脉冲应在 \(10^{3}\mathrm{Gy(Si)} / \mathrm{s}\sim 10^{9}\mathrm{Gy(Si)} / \mathrm{s}\) (或根据要求)之间可调。除另有规定外，被照射的被试器件总剂量达到总剂量失效级别的 \(20\%\) ，就应认为是破坏性试验，需从该批中剔除。
