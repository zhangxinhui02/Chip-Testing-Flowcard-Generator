# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5005.3 鉴定和质量一致性检验程序
### 3.6.3 B级B组检验的替代

---

根据承制方的意愿(仅B级)，在每个密封周中，可针对每种型号和引线涂覆外壳所组成的任一检验批进行B组检验。各分组可采用不同的检验批。这种方式的检验完成之后，在同一装配线上采用同一型号和引线镀涂外壳，并在同一周中密封的所有其他型号器件，可予以接收，不必重复进行B组检验。对于包含有特定型号及引线镀涂外壳的器件检验批，只有在对每个密封周中、采用该种型号和引线镀涂外壳封装的器件，成功地完成B组检验之后，才能认为该检验批通过B组检验。  
采用B组替代程序时，如在B组中出现一个失效，应从该失效期间封装的相同型号、引线镀涂的外壳及同一周中密封的连续三个检验批中抽取样本，针对所失效的分组进行试验。如果三个检验批都通过试验，则在同一封装线上用同一型号及引线镀涂外壳并在同一周中密封的全部器件，可认为通过B组检验。如果三个检验批中，有一批或一批以上失效，则在同一封装线上采用同一型号、引线镀涂外壳并在同一周中密封的器件，都不应认为通过B组检验，只有对每一检验批均通过发生失效分组的试验，才视为通过B组检验。
