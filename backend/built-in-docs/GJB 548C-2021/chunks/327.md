# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 3.9 辐射温度箱

---

在进行高温辐射试验时，使用的辐射温度箱在辐射过程中应能使被试器件维持在 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 范围内。辐射前，温度箱应能以较快的速度使试验样品从环境温度升到辐射温度，辐射后应能在 \(20\mathrm{min}\) 内使其从辐射温度冷却到环境温度。在升温和冷却过程中， 应始终保持偏置。被试器件的升温、恒温、冷却过程可通过使用充入热、冷液体的热沉来实现， 或是采用交替充入热、冷空气的方法， 或是其他可以得到正确结果的方法。对于低温辐射试验， 温度箱应能使被试器件在辐射期间维持在所需低温的 \(\pm 5^{\circ}\mathrm{C}\) 范围内(比如， 液氮或液氮)。温度箱应能使被试器件在辐射后电参数测试期间维持低温。
