# 中华人民共和国国家军用标准
## 方法1002 浸液
### 3 程序

---

本试验连续进行数次浸液循环，每次循环先浸在温度为 \(65^{\circ}\mathrm{C}\) 的热淡水(自来水)槽中，然后浸在冷水槽中。循环次数、每次浸渍时间(即冷、热水槽浸渍时间)、冷水槽的性质和温度都在表1中加以规定。样品应尽可能迅速地从一个槽转移到另一槽中。在任何情况下，转移时间都不应超过15s。完成最后一次循环后，应在淡水或蒸馏水中把样品迅速而彻底地清洗，并擦干或吹干所有表面，使其干净和干燥。除另有规定外，应在末次循环完成4h后、48h内对器件进行测试。当订购文件有规定时，在电测试和外部自检完成后，应按方法2013打开器件，检查内部元器件有无浸蚀现象和盐的晶粒。当本试验作为一个试验组或试验分组的一部分来进行时，在本项试验结束时，不必专门进行试验后的测试或检查，而可以在该组或分组试验结束时再进行。
