# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 4.1 试验实施

---

细检漏和粗检漏试验应按相应试验条件的要求和程序实施。除采用试验条件 \(\mathrm{A}_{5}\) 或光学检漏或累积式氨检漏外，试验顺序为先进行细检漏，后进行粗检漏。规定时(见第7章),应在检漏程序完成后进行测试。采用试验条件 \(\mathrm{A}_{4}\) 时，不必进行粗检漏。试验条件 \(\mathrm{A}_{4}\) 不能用于已封装的器件所要求的密封试验。  
当规定的加压压力超过器件封装的承受能力时，可以使用其他满足试验要求的加压压力、压力作用时间、候检时间的组合条件，但在任何情况下作用于器件上的压力不得小于 \(202\mathrm{kPa}\) (两个绝对大气压),或对于试验条件 \(\mathrm{A}_{4}\) ，任何情况下试验压力至少为 \(69\mathrm{kPa}\) 。  
若按试验条件A或试验条件B对多个器件同时进行试验时(即在检漏仪中每次放置一个以上器件),如该组器件出现漏率超标，可以对每个器件分别重新进行试验以决定接受与否，但必须在将器件从示踪气体加压室中取出后1h之内完成对该组所有器件的重新试验。仅对试验条件 \(\mathrm{B}_{1}\) ，可对多个器件同时进行试验或单独试验以决定接受与否，但必须在将器件从示踪气体加压室中取出后1.5h之内完成试验。  
对试验条件 \(\mathrm{C}_{4}\) 和试验条件 \(\mathrm{C}_{5}\) ，封装必须满足5.4.1中规定的要求。这包括敷形涂层器件，例如电路板组件。
