# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5013.1 GaAs工艺的晶圆制造控制和接收程序
### 3.3.2 工艺监测图形(PM)

---

工艺监测图形是一组为工艺控制和确定晶圆可接收性提供数据的测试结构。PM既可以插入于每个晶圆的特定位置，位于划片槽位置，或包含在每个芯片中，也可以是这些方法的组合。因此，适用时它们可以在最终的互连金属化和钝化完成后用探针进行测试。应在基线文件中给出PM结构，测试和可接收性要求。表1列出了常用的PM测试结构。
