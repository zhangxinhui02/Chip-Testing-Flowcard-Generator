# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.7.1.3 真空/压力室容积

---

5.7.1.3 真空/压力室容积检漏的真空/压力室容积应尽量小，因为真空/压力室固定容积 \((V_{\mathrm{C}})\) 与器件内空腔容积 \((V_{\mathrm{d}})\) 的比超过100将降低粗检灵敏度。对于单个器件检测或一组检测，需确定真空/压力室固定容积 \((V_{\mathrm{C}})\) 与器件内空腔容积 \((V_{\mathrm{d}})\) 的最大比值，以保证在信噪比至少为3时能够检测到粗检漏率的范围。该比值是净化气体中氨气含量、内空腔容积和被试器件数量的函数。
