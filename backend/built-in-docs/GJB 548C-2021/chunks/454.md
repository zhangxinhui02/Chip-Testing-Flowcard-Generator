# 中华人民共和国国家军用标准
## 方法1023.1 线性微电路的剂量率响应和翻转阈值
### 4.3.5 旁路电容器

---

4.3.5 旁路电容器高频电容器应置于被试器件的每个偏置电源端， 引线长度尽可能短。这些电容器容量应足够大(典型值在 \(4.7\mu \mathrm{F} \sim 10\mu \mathrm{F}\) 之间), 使在辐射脉冲作用时器件的电源电压下降小于 \(10\%\) 。采用一只低感电容器(如 \(0.1\mu \mathrm{F}\) )与此电容并联， 应尽可能靠近电源端， 引线尽可能的短。此外， 对于进入器件的每根电源线， 一个大于等于 \(100\mu \mathrm{F}\) 的大电容器可以置于距器件较近的位置， 并应对辐射屏蔽。
