# 中华人民共和国国家军用标准
## 方法1011.1 热冲击
### 4.1 通则

---

样品应放于试验槽中的合适位置，使液体在样品周围的流动不应受到阻碍。然后根据表2的规定，使负载经受试验条件B或表2中规定的其他试验条件（见第5章b））下的15次循环。在完成规定的试验循环总次数期间，为了进行器件批的加载或卸载，或由于电源或设备故障，允许中断试验。然而，对任何给定的试验，若中断次数超过规定循环总次数的10％时，试验必须重新从头开始。
