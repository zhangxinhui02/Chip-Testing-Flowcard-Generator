# 中华人民共和国国家军用标准
## 方法1022 MOS场效应晶体管阈值电压
### 3.4 电压源 \((\mathbf{V}_{\mathrm{S}1}\) 和 \(\mathbf{V}_{\mathrm{S}2}\)

---

3.4 电压源 \((\mathbf{V}_{\mathrm{S}1}\) 和 \(\mathbf{V}_{\mathrm{S}2}\) 电压源应在标定的 \(0\mathrm{V}\sim 20\mathrm{V}\) 范围内可调，输出电流至少应等于被试器件的最大额定漏极电流。其噪声和纹波电压应小于输出电压的 \(0.5\%\) 。
