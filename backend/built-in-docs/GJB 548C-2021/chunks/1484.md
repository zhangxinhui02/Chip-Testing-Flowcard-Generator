# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5010.3 复杂单片微电路检验程序
### 3.4.1 通则

---

应按3.4.1~3.4.13以及表4进行器件的筛选。这一规定并不排除特殊器件的附加筛选要求以及为优化筛选结果要求的附加测试或检验，但是所有的特殊测试检查均应遵从GJB 597B-2012中相关要求。进行规定的各种筛选试验时不得采用抽样检查的方式。仅当批记录中有明文规定时，才允许进行老炼，并且老炼中的失效应记入相应的PDA 计算中。对于筛选程序中要求进行终点测试或试验后测试的试验方法，如果这种试验后测试在随后的中间（老炼后）电测试或最终电测试时还要重复进行，则在筛选中无需进行这种测试，仅需在中间（老炼后）电测试或最终电测试时进行。  
通过较高可靠性等级筛选要求的器件应视为满足所有较低等级的筛选要求。任何情况下都不允许采用比规定要求低的等级进行筛选。在筛选程序中未能通过任何试验或检验判据的器件应在发现时或观察到失效的试验结束时立即从该批中剔除。一旦拒收并证明失效的器件，就不得再重复进行试验。除在详细规范中另有规定外，电参数拒收的器件可按3.5规定选择作为鉴定及质量一致性检验的样本构成。  
器件筛选应按表4顺序进行，本方法特别允许的顺序变化除外。
