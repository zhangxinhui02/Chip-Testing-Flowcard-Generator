# 中华人民共和国国家军用标准
## 方法1034.1 染料渗透
### 3.1.2.1 溶液混合

---

浸渗剂和黄色荧光粉按100:1进行混合，步骤如下：a) 将 \(2000\mathrm{mL}\) 浸渗剂倒入烧杯中。  
b) 将 \(20\mathrm{g}\) 黄色荧光粉倒入盛有 \(2000\mathrm{mL}\) 浸渗剂的烧杯中，搅拌 \(5\mathrm{min}\) 。然后将该溶液分装至2个1000mL 烧杯中以便于后续试验。在这 2 个 1000mL 烧杯中各放入一个磁力搅拌器， 把烧杯放置在搅拌板上， 搅拌混合物溶液各 \(2\mathrm{h} \sim 3\mathrm{h}\) 。
