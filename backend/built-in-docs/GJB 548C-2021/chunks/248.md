# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.7.2.1 概述

---

将完整的器件放入如5.1.2.2中试验条件 \(\mathrm{A}_1\) 和试验条件 \(\mathrm{A}_2\) 的真空/压力室内(如抽真空，应记录抽真空压力)。加压后压力释放(可用空气冲洗或其他技术减少5.7.2.2中表面吸附效应),将每个样品转移到另一试验箱或几个试验箱，这些试验箱与抽真空系统和质谱检漏仪连接。当试验箱抽真空后，原先加压进入样品的示踪气体就会泄漏出来，并在检漏仪上显示为测量漏率 \(R\) 。试验中采用的器件数应满足下述要求：对于内空腔容积大于 \(0.05\mathrm{cm}^3\) 的器件，从压力室中移出后，对于试验条件 \(\mathrm{CH}_1\) ，能够在 \(4\mathrm{h}\) (或固定方法表格中选择的候检时间)之内完成检测，对于试验条件 \(\mathrm{CH}_2\) ，能够在选择的候检时间 \(t_2\) 内完成检测。对于内空腔容积小于 \(0.05\mathrm{cm}^3\) 的器件，对于试验条件 \(\mathrm{CH}_1\) ，必须在 \(4\mathrm{h}\) (或固定方法表格中选择的候检时间)之内开始检测，对于试验条件 \(\mathrm{CH}_2\) ，必须在选择的候检时间内开始检测。  
除非在订购文件或合同上有特别规定，否则应使用灵活方法。
