# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 4.4.1 计算封装热阻

---

当按4.1和4.2所述步骤分别测量结温的峰值 \(T_{\mathrm{J}(\mathrm{峰值})}\) 、平均值 \(T_{\mathrm{J}(\mathrm{平均})}\) 或区域值 \(T_{\mathrm{J}(\mathrm{区域})}\) 之后，通过计算可得到器件的热阻。若要估计整个封装的热阻，应对被试器件施加额定功率 \(P_{\mathrm{D}(\mathrm{封装})}\) 。若用作为质量控制，可用单个被测结的功耗 \(P_{\mathrm{D}(\mathrm{单结})}\) 来计算热阻。  
记录了每种测量数据之后，应按公式(4)\\~公式(6)计算热阻:根据上述公式计算单结热阻时，应采用 \(P_{\mathrm{D}(\mathrm{单结})}\) 。值得注意的是，这些热阻值与封装的散热方式无关。这可能由于外壳或片式载体(参考)温度是在封装壳体自身的某个合适位置上(即在从芯片经封装到散热器的主要热流通道上给出典型温度的位置)测量的缘故。  
\[R_{\\mathrm{th}(\\mathrm{J} - \\mathrm{C})}(\\mathrm{峰值}) = \\frac{T_{\\mathrm{J}(\\mathrm{峰值})} - T_{\\mathrm{C}}}{P_{\\mathrm{D}(\\mathrm{封装})}} \\quad (4)\]  
\[R_{\\mathrm{th}(\\mathrm{J} - \\mathrm{C})}(\\mathrm{平均}) = \\frac{T_{\\mathrm{J}(\\mathrm{平均})} - T_{\\mathrm{C}}}{P_{\\mathrm{D}(\\mathrm{封装})}} \\quad (5)\]  
\[R_{\\mathrm{th}(\\mathrm{J} - \\mathrm{C})}(\\mathrm{区域}) = \\frac{T_{\\mathrm{J}(\\mathrm{区域})} - T_{\\mathrm{C}}}{P_{\\mathrm{D}(\\mathrm{封装})}} \\quad (6)\]
