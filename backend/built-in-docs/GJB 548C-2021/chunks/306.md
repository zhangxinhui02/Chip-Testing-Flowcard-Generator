# 中华人民共和国国家军用标准
## 方法1018.2 内部气体成分分析
### 2.2 真空室和传递通道

---

能放置器件的一个开口真空室和使器件与2.1中质谱仪相连接的真空传递通道。真空传递通道应能有效地将器件内的气体传送到质谱仪的离子源(防止明显的水汽吸附损失)。  
对于系统的初始确认或检测能力的扩展，当使用外部夹具固定器件时，系统上的器件温度可通过将热电偶置入一个与被试器件具有类似质量、内部体积、结构和尺寸的空腔内进行表征，以保证器件温度在 \(100^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}\) 保持至少 \(10\mathrm{min}\) 时间。它也适用于检测在外部烘箱中预烘烤的器件在转移过程中温度的下降情况。这些记录应由检测实验室保留。
