# 中华人民共和国国家军用标准
## 方法1005.1 稳态寿命
### 3.3.2.2 单片功率电路的试验温度

---

器件无论功率大小都应能以最高额定工作温度进行老炼或寿命试验。用环境温度 \(T_{A}\) 表示最高工作温度的器件，采用表1。用外壳温度 \(T_{C}\) 表示最高工作温度的器件， \(125^{\circ }C\) 的环境温度有可能使 \(T_{J}\) 超过 \(200^{\circ }C\) （对S级，超过 \(175^{\circ }C\) )，为此进行老炼和寿命试验时的环境温度可以从 \(125^{\circ }C\) 降低到某一个温度，但要求在该温度下， \(T_{j}\) 应控制在 \(175^{\circ}\mathrm{C} \sim 200^{\circ}\mathrm{C}\) 之间， \(T_{\mathrm{C}}\) 等于或大于 \(125^{\circ}\mathrm{C}\) ，这时不必改变试验时间。应准备好支持这种降温处理的数据，在有要求时提交给订购方和鉴定机构。
