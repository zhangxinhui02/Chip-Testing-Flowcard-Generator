# 附录A（方法5004）（资料性附录）晶圆制造过程的缺陷控制程序
## 方法5010.3 复杂单片微电路检验程序
### 3.5.3.6.2 在线验证测试

---

程序如下：a）对每一台测试设备及手工测试时的操作人员，应使用相关器件（比如校准件）进行校验，以确保满足本标准4.5.2的准确度要求。  
b）电测试中使用的所有设备已经过校准和确认。  
c) 完成测试后(不超过8h以及在更换工作人员时),应即时接受质量保证部门指派的人员按下述方式进行生产测试验证。  
1) 用目检方式检查所用夹具、设备、软件及程序是否正确。  
2) 用生产中所采用的夹具、设备、软件及程序对受控的已知良好器件进行实测。对受控器件应读取并记录要求的全部A组测试变量数据。这些数据应随批保存。  
3) 检验小组应在随工单上盖章或签字，以证明上述数据满足测试要求，同时所有上述各项程序已经执行完毕并且是满意的。  
4) 若验证试验不合格，则至少要对硬件、软件、工装及零部件进行仔细的工程审查。如工程审查确定了问题所在，应对准备交付的全部器件重新进行一次(仅允许一次)A组的 \(100\%\) 测试。如果工程审查没有确定问题所在，在对整批进行重新测试前，应对验证器件进行失效分析。  
(a) 如果失效分析能确定问题所在，可以对失效时供交收的所有器件重新进行一次(仅允许一次)适当分组的测试，并在测试前重新进行在线验证测试。  
(b) 如果失效分析并不能具体确定问题所在，可对A组要求的全部器件重新进行一次(仅允许一次)A组方法 \(100\%\) 测试，并在重新进行在线验证测试。
