# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.12.3.1 过辐射试验

---

按如下步骤进行过辐射试验:a) 按标准条件对每一个被试器件附加0.5倍规定总剂量的额外总剂量辐射。注意， 该过程不需进行电测试。  
b) 如果通过表征试验证明如下结果， 可不进行4.12.3.1a)的0.5倍总剂量额外辐射:器件传输延迟、输出驱动和最小工作电压等参数向辐射前恢复的量值均不大于按4.12.3.2加速退火试验的恢复量值， 并且辐射和加速退火试验所选择的偏置应使这些参数在加速退火过程中的响应达到最坏情况。  
c) 表征试验应在规定的辐射等级下进行。应至少包括静态和动态偏置情况下的辐射试验， 辐射后接着进行最坏情况静态偏置加速退火试验(4.12.3.2的条件)。
