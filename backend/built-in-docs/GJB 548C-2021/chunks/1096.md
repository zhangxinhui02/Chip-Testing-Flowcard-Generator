### 3.4 记录

当有规定时，记录分离时力的大小与失效类别。  
![](assets/0be9f6c501e318f4746361767e3fb1e773c60f6a821786fc29127149c10d849d.png)  
注： \(x\) 轴为对数刻度，未给出的点不能从图上外推，应根据公式(1)和公式(2)计算：示例：假设被试芯片尺寸为 \(0.8\mathrm{cm}\times 0.8\mathrm{cm}\)  
\[x = 0.8\\mathrm{cm}\\times 0.8\\mathrm{cm} = 0.64\\mathrm{cm}^2\]  
则 \(y = 14.8\times \log x + 47.2 = 14.8\times \log (0.64) + 47.2 = 14.8\times (- 0.194) + 47.2 = - 2.9 + 47.2 = 44.3\mathrm{N}\)  
因此1.0倍时 \(y = 44.3\mathrm{N}\) ，2.0倍时 \(y = 88.6\mathrm{N}\) 。
