# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.15.2 环境试验顺序

---

至少采用五个样品经受下面规定的环境条件试验。应采用能代表最终使用情况的背面材料、附着表面和尺寸的最大元件/基板/封装组合来制备样品。元件类型包括电阻器、电容器、集成电路芯片和半导体分立器件芯片。对每一类型应采用两个元件，利用该类元件推荐使用的聚合物材料(I型或II型)将元件粘接到基板上。样品应按下面给出的顺序依次经受环境条件作用:a) 热冲击(方法1011,试验条件C,15个循环);  
b) 温度循环(方法1010,试验条件C,100个循环);  
c) 机械冲击(方法2002,试验条件B,仅Y1方向);  
d) 扫频振动(方法2007,试验条件A,仅Y1方向);  
e) 恒定加速度(方法2001,试验条件B,仅Y1方向)。
