# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.11.1 概述

---

4.11.1 概述对某些低剂量率辐射环境(如空间应用的辐射剂量率),标准条件试验被认为过度保守。即使试验是在较高辐射剂量率下进行(比如 \(0.5\mathrm{Gy(Si)} / \mathrm{s} \sim 3\mathrm{Gy(Si)} / \mathrm{s}\) ),补充室温退火试验仍可估计在非常低剂量率环境下的器件特性。该程序包括：先按标准条件步骤进行辐射试验，辐射后将器件置于室温环境进行一定时间(见4.11.3c))的退火，该过程可以使那些漏电相关、超过辐射前规范值的参数返回到规范要求之内。
