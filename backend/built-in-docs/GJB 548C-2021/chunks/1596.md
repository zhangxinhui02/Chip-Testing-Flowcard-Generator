# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5011 聚合物材料的评价和验收程序
### 5.8.6.3 填充剂含量的热稳定性

---

对于采用填充剂来提高其电导率或热导率等特性的聚合物材料，应采用如下方法测量聚合物材料中填充剂含量的热稳定性：应在气流速率为 \((10\sim 20)\mathrm{mL / min}\) 的空气气氛中将样品加热到 \(600^{\circ}C\) ，升温速率为 \((10\sim 20)^{\circ}\mathrm{C / min}\) 。将温度维持在 \(600^{\circ}C\) ，直到重量保持恒定。允许在5.8.6.2试验基础上按上述要求将温度从 \(210^{\circ}C\) 提升到 \(600^{\circ}C\) 。测量结果应采用固化样品中填充剂含量的重量百分比表示。
