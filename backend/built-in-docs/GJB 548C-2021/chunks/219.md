# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.2.1 激活参数

---

激活压力和吸收时间应按公式(3)确定:式中:  
\[Q_{S} = \\frac{R}{S\\cdot K\\cdot T\\cdot P\\cdot t} \\quad (3)\]  
\(Q_{S}\) ——被试器件的最大允许漏率， \((\mathrm{Pa}\cdot \mathrm{cm}^{3}) / \mathrm{s}\) (氦)。  
\(R\) ——活化后，器件漏率等于 \(Q_{S}\) 时，每分钟超出环境背景的计数大小。如之前已经受放射性检漏试验，则为超过计数设备和器件背景的拒收计数。  
\(S\) ——激活系统中的氦- 85示踪气体放射性比活度， \(\mathrm{Bq} / (\mathrm{Pa}\cdot \mathrm{cm}^{3})\) 。  
\(K\) ——闪烁晶体对被试器件内部空腔中氦- 85的总计数效率，计数数目/ \(\mathrm{Bq}\cdot \mathrm{min}\) ;该因子大小取决于器件形状结构和闪烁晶体的尺寸。计数效率的确定见5.2.2。  
\(T\) ——吸收时间(即器件受激活的时间), \(\mathrm{h}\) 。  
\(P\) ——等于 \(P_{\mathrm{e}}^{2} - P_{\mathrm{i}}^{2}\) ，其中 \(P_{\mathrm{e}}\) 是器件的激活压力，可按规范确定或者在确定了合适的吸收时间 \(T\) 的情况下，按公式(3)确定，单位为绝对大气压。 \(P_{\mathrm{i}}\) 为器件内部原有的压力，单位为绝对大气压。  
\(t\) ——小时转换成秒的换算系数， \(1\mathrm{h} = 3600\mathrm{s}\) 。
