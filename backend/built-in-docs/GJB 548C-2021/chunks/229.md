# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.4.1 概述

---

光学检漏(OLT)适用于独立的器件和安装到印制电路板或更高等级组装的器件。光学检漏系统是基于其使盖板或封装产生弹性形变的能力。被试器件的盖板变形量应满足干涉仪所检测的最小灵敏度，即最少为 \(7.25\times 10^{- 7}\mu \mathrm{m / Pa}\) 。这些试验条件对由金属、陶瓷或其他加压时盖板有可检测变形的已封盖器件是有效的。一般地，用氮气作为加压媒质。所需的设备包括合适的压力室，内置一个光学检漏仪。光学检漏仪应经过预置和适当的校准，使其对漏孔的等效标准漏率检测灵敏度达到第6章表6的要求。
