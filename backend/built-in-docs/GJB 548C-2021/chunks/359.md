# 中华人民共和国国家军用标准
## 方法1019.3 电离辐射(总剂量)
### 4.13.4 双极(或 BiCMOS)线性或混合信号器件的低剂量率或高温辐射试验

---

具有 ELDRS 特性的器件应按如下任意一个条件进行试验：a) 商定的剂量率试验以 4.6.4 条件 C 进行辐射试验。  
b) 规定的低剂量率试验以 4.6.5 条件 D 进行辐射试验，并附加 1.5 倍过辐射因子，即只有通过 1.5 倍规范规定总剂量试验的器件，才可接收。  
c) 加速试验以 4.6.6 条件 E 进行加速条件的辐射试验，并附加参数 \(\Delta\) 值设计裕量和(或)过辐射因子(见 4.13.3)。
