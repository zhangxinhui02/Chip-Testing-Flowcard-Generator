# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2030.1 芯片粘接的超声检测
### 4.11 芯片粘接的检测和拒收判据

---

在器件检测中，下列情况应被看作是不可接收的芯片粘接。如果关注散热器，直接在“热”区域下方的粘接界面缺陷可能存在问题，应进一步用可靠性试验进行评估。当用超声波为器件成像时，某些种类的安装材料可能无法显示空洞的真正图像，检测此类器件时，应在检测报告中注明这类安装材料。呈现下列缺陷的器件应被拒收：a) 多个空洞总和大于预计接触区总面积的 \(50\%\) ；b) 单个空洞大于预计接触区总面积的 \(15\%\) (见表1)；  
c) 拐角区空洞大于预计接触区总面积的 \(10\%\) (见表1);  
d) 当用平分两对边方法把图像分成四个面积相等的象限时， 任一象限中的空洞大于该象限预计接触区面积的 \(70\%\) (见表1)。  
在有争议的情况下， 空洞的百分比应使用数字图像的空洞和粘接图像分析功能来实际测量确定， 至少使用两个阈值水平。
