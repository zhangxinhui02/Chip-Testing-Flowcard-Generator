# 附录A（方法2018）（资料性附录）SEM检查金属化层完整性的替代方法
## 方法2037 铅锡焊料成分分析- X 射线荧光检测
### 3.5.1 通则

---

对于含有铅锡成分样品的器件批，当每个样品检测结果的铅含量(重量百分比)大于 \(3\%\) 时，该批器件可接收，合同或订购文件另有规定时除外。应基于设备准确度计量的再现性和重复性对该最低铅含量值进行调整，例如，当设备已确认的准确度为 \(\pm 20\%\) 时，则要求的最低铅含量为 \(3.6\%\) (重量百分比)。发现一个超标样品应拒收该样品代表的整批器件。失效批应保持拒收或进行返工，或按3.5.2的要求对其进行评估。
