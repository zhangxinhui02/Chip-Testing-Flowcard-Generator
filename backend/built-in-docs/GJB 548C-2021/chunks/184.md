# 中华人民共和国国家军用标准
## 方法1012.1 热性能
### 4.3.2 测量结温与时间的关系以确定 \(t_{\mathrm{JR}}\)

---

在结上施加或去掉阶跃形式的加热功率，使结温随时间发生变化。应采用红外微辐射仪或电测试仪器来观测结温的变化。这些仪器的响应时间均应小于 \(100\mu \mathrm{s}\) ，而且电测试仪器还应具有足够的灵敏度，能够读取(已预校准的)结的温敏电参数测量值。测量过程中器件规定参考点的温度(规定值)应保持不变。施加或去掉阶跃形式的功率，并且从施加或去掉电源的时刻开始到结温达到稳定时为止，应记录结温和时间关系的曲线。  
应记录下列数据:a) 温敏电参数(见4.2.3);  
b) 红外显微镜的空间分辨率(见4.2.2);  
c) 随时间变化的结温(峰值、平均值或区域值，见4.2.2或4.2.3的说明);  
d) 外壳温度 \(T_{\mathrm{C}}\) 或安装表面温度 \(T_{\mathrm{M}}\) (通常是 \((60\pm 0.5)^{\circ}\mathrm{C}\) );  
e) 封装内的功耗 \(P_{\mathrm{D}(\mathrm{封装})}\) 或 \(P_{\mathrm{D}}\) (单结);  
f) 测量点的参考温度；  
g) 安装布局。
