# 中华人民共和国国家军用标准
## 方法1017.1 中子辐射
### 3.4.1 中子注量

---

通过测定与被试器件同时受辐射的快中子活化箔如 \(^{32}\mathrm{S}\) 、 \(^{54}\mathrm{Fe}\) 和 \(^{58}\mathrm{Ni}\) 中感生的放射总量可获得用于器件受辐射的中子注量。把活化箔中测得的放射量换算到中子注量的方法由有关标准给出。  
由箔的放射量换算到中子注量，需预先知道入射到箔上的中子能谱。若中子能谱未知，则应采用GJB3495或其他适用方法确定入射在箔上的中子谱。一旦确定了中子能谱，并计算出1MeV等效中子注量，就应在随后的辐射中采用适当的监测用箔(如 \(^{32}\mathrm{S}\) 、 \(^{54}\mathrm{Fe}\) 和 \(^{58}\mathrm{Ni}\) )确定中子注量。使用活化箔来测量1MeV等效中子注量的方法仅当中子能谱保持不变时有效。
