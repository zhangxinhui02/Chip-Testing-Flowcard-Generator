# 中华人民共和国国家军用标准
## 方法2012.2 X射线检查
### 3.11.2.2 多余物

---

多余物应包括但又不限于下述类型：a) 存在于芯片、元件、基板表面或封装内的尺寸大于 \(25\mu \mathrm{m}\) 的可动多余物或尺寸虽小但足以跨接器件中互不相连接的导电部分的任何可动多余物；b) 附着的多余物桥连了金属化通道、封装引线、引线与封装金属化区、电路功能元件、接头或任一组合；注：附着或可动多余物可以通过比较两次相同观察条件之间进行的轻微机械振动(如PIND试验)加以验证。c) 内引线尾丝与不应相连的金属相连，内引线尾丝在基板或外壳接线端的长度超过了直径的4倍，或在芯片、元件上的尾丝长度超过引线直径的2倍；d) 焊料、合金或导电粘结剂使得非设计要求的金属化区域(即引线、键合柱、有源金属化区或任一组合)发生桥连；e) 底座上、外引线键合区或外壳内任一处的金层剥离。
