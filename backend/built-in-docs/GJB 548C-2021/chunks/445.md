# 中华人民共和国国家军用标准
## 方法1023.1 线性微电路的剂量率响应和翻转阈值
### 3.1 概述

---

在进行剂量率试验之前，应准备一份试验方案，其中包括辐射源、剂量测定方法、测试设备、被试器件、试验条件和完善的试验设计(试验示例参见附录A和附录B)。应对每个被试器件准备详细的程序，并作为试验方案的一部分或放入单独的试验程序文件中。试验程序应包括偏置条件、试验顺序、试验装置和被测的特殊功能。在试验方案或程序中至少应包括下述条款：a) 器件类型，包括封装类型、承制方、日期代码和被试器件数量。  
b) 追溯要求，如：连续性编号，晶圆或批的可追溯性等要求。  
c) 数据报告和提交要求。  
d) 试验装置的方框图或原理表征。  
e) 试验中所用的设备清单，其校准符合性情况。  
f) 试验条件，如偏置电压、温度等。  
g) 用于监控的电参数和器件工作条件，包括辐射脉冲前后及试验中的功能测试要求。具有存储单元的器件所采用的测试图形或具有输入图形灵敏性的器件，也应加以规定。  
h) 剂量率试验前后A组电测试要求，适用时应包括测试极限值和失效判据。  
i) 辐射试验参数，如脉冲宽度、每个脉冲的辐射剂量和剂量率范围。  
j) 适合每个器件类型的总电离剂量极限值。  
k) 翻转和失效判据，如在模一数转换器(ADC)中、有效位数(ENOB)或丢码、 \(V_{\mathrm{OH}}\) 或 \(V_{\mathrm{ref}}\) 变化量、恢复时间、输出波形的形状或频率失真等。
