# 中华人民共和国国家军用标准
## 微电子器件试验方法和程序
### 4.5.9.2 测试温度 \(T_{\mathrm{C}}\) 、 \(T_{\mathrm{A}}\) 或 \(T_{\mathrm{J}}\) 高于 \(25^{\circ} \mathrm{C}\) 时的控制

---

除另有规定外，在不加电的条件下将器件(包括内部元器件；例如芯片、电容器、电阻器)温度稳定在 \(T_{\mathrm{C}}\) 、 \(T_{\mathrm{A}}\) 或 \(T_{\mathrm{J}}\) 的 \(\pm 3^{\circ} \mathrm{C}\) (对于混合集成电路为 \(5^{\circ} \mathrm{C}\) ) 之内。如果混合集成电路的结构指明且承制方能够保证该器件在试验下不会退化，则混合集成电路可以超过正偏差。当建立的温度特征曲线对器件试验是可用的，那么可使用该特征曲线代替温度测量，以决定合适的热浸条件从而达到本要求。当使用温度特征曲线时，试验设备监控应保证控制可以提供正确的试验环境。在稳定后，器件工作和 \(T_{\mathrm{C}}\) 、 \(T_{\mathrm{A}}\) 或 \(T_{\mathrm{J}}\) 进行控制不得降低规定温度的 \(3^{\circ} \mathrm{C}\) 以上。在测试过程中器件温度不应低于规定的 \(T_{\mathrm{C}}\) 、 \(T_{\mathrm{A}}\) 或 \(T_{\mathrm{J}}\)\(3^{\circ} \mathrm{C}\) 以上。如果承制方保证器件不会退化，测试期间温度可以比规定的 \(T_{\mathrm{C}}\) 、 \(T_{\mathrm{A}}\) 或 \(T_{\mathrm{J}}\) 高 \(3^{\circ} \mathrm{C}\) 以上。应采用低占空比脉冲，或有规定时采用稳定功率条件(见4.5.9.5)来测量电参数。
