# 中华人民共和国国家军用标准
## 方法1014.3 密封
### 5.1.2.4 试验条件 \(\mathbf{A}_2\) -灵活方法

---

选择适当的加压压力、压力作用时间和候检时间， 使有缺陷的被试器件测得的示踪气体测量漏率 \((R)\) 的读数大于质谱仪的最小检测灵敏度。器件应至少承受两个绝对大气压 \((202\mathrm{kPa})\) 的氮气作用。将上述选定的数值以及被试器件的内空腔容积和"等效标准漏率" \(L\) 的最大极限值(如第6章表6所示或在有关订购文件中规定)一起代入公式(1)计算"测量漏率" \(R_{1}\) 的极限值:式中:  
\[R_{1} = \\frac{L\\cdot P_{\\mathrm{E}}}{P_{\\mathrm{O}}}\\left(\\frac{M_{\\mathrm{A}}}{M}\\right)^{1 / 2}\\left\\{1 - \\exp \\left[-\\frac{L\\cdot t_{1}}{V\\cdot P_{\\mathrm{O}}}\\left(\\frac{M_{\\mathrm{A}}}{M}\\right)^{1 / 2}\\right]\\right\\} \\exp \\left[-\\frac{L\\cdot t_{2}}{V\\cdot P_{\\mathrm{O}}}\\left(\\frac{M_{\\mathrm{A}}}{M}\\right)^{1 / 2}\\right] \\quad (1)\]  
\(R_{1}\) ——计算得到的测量漏率的最大允许拒收极限值， \(\mathrm{(Pa\cdot cm^3)/s}\) (氮);  
\(L\) ——等效标准漏率的最大允许拒收极限值， \(\mathrm{(Pa\cdot cm^3)/s}\) (空气);  
\(P_{\mathrm{E}}\) ——绝对作用压力， \(\mathrm{Pa}\) ;  
\(P_{\mathrm{O}}\) ——绝对大气压力(为 \(1.01\times 10^{5}\) ), \(\mathrm{Pa}\) ;  
\(M_{\mathrm{A}}\) ——空气的分子量(为 \(28.96\) ), \(\mathrm{g}\) ;  
\(M\) ——示踪气体(氮)的分子量(为4), \(\mathrm{g}\) ;  
\(t_{1}\) ——受 \(P_{\mathrm{E}}\) 压力作用的时间， \(\mathrm{s}\) ;  
\(t_{2}\) ——去除压力后到检漏之间的候检时间， \(\mathrm{s}\) ;  
\(V\) ——被试器件封装的内空腔容积， \(\mathrm{cm}^3\) 。
