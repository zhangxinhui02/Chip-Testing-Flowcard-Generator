# 附录B (方法5010) (规范性附录) 晶圆批验收
## 方法5012 数字微电路的故障覆盖率测量
### 2.8 逻辑(组合、时序) logic (combinational and sequential)

---

逻辑包括组合逻辑和时序逻辑。  
组合逻辑指输出状态仅取决于输入端的逻辑信号，不考虑开关延迟。组合逻辑电路仅由不含存储器的部件组成，而且电路中没有反馈路径。  
时序逻辑指在任意给定时刻的输出状态取决于前一时刻输入端所加逻辑信号的顺序和时间关系。时序逻辑电路至少有一个部件含有存储器，或至少有一条反馈路径，或两者都有。例如，一个触发器就含有一个存储器，交叉耦合逻辑门引入了反馈路径。
